# 古堡法阵

## 题目大意

有 $N$ 个点，可以在任意两点之间建边。每条边的通过时间都是 $1$，其中给出的 $M$ 条边可以免费建，其余边每建一条需要花费 $Q$。

设建出的图中有 $S$ 条付费边，所有有序点对最短路长度之和为：

$$
P=\sum_{x=1}^{N}\sum_{y=1}^{N}T(x,y)
$$

要求最小化：

$$
P+Q\times S
$$

其中 $M\le 2$，答案对 $998244353$ 取模。

## 思路分析

首先考虑没有免费边的情况。

如果只想让图连通，边数至少是 $N-1$。在所有树中，星形图的距离和最小：中心到其他点距离为 $1$，两个非中心点之间距离为 $2$。

星形图的有序点对距离和为：

$$
2(N-1)+2(N-1)(N-2)=2(N-1)^2
$$

所以当 $Q$ 比较大时，我们倾向于少建边，使用星形图。

另一方面，如果 $Q\le 2$，那么建一条缺失的边一定不会更差。因为一条新边至少能让它两个端点之间的有序距离各减少 $1$，总共至少减少 $2$，而付出的代价不超过 $2$。所以这时直接建完全图最优。

完全图中所有不同点对距离都是 $1$，因此：

$$
P=N(N-1)
$$

付费边数为：

$$
S=\frac{N(N-1)}{2}-cnt
$$

其中 $cnt$ 表示去重后的免费边数量。

这样就解决了 $Q\le 2$ 的情况。

下面考虑 $Q>2$。此时如果图已经是星形图，再额外添加一条付费边，最多只会让一对无序点之间的距离从 $2$ 变成 $1$，也就是让有序距离和减少 $2$，但需要付出大于 $2$ 的代价，所以不值得。

由于免费边最多只有两条，我们按免费边的形态分类讨论。

### 没有免费边

直接建一棵星形树即可：

$$
P=2(N-1)^2
$$

$$
S=N-1
$$

### 只有一条免费边

把星形图的中心选在这条免费边的一个端点上，这样这条免费边可以作为星形图中的一条边。

距离和仍然是：

$$
P=2(N-1)^2
$$

付费边少一条：

$$
S=N-2
$$

### 两条免费边有公共端点

如果两条免费边共享一个端点，那么把星形图中心选在这个公共端点上，两条免费边都可以作为星形图中的边。

于是：

$$
P=2(N-1)^2
$$

$$
S=N-3
$$

### 两条免费边不相交

这是唯一需要再比较的情况。

方案一：仍然采用星形结构，并且额外保留另一条免费边。

此时星形图本身的距离和是 $2(N-1)^2$，额外的那条免费边会让它两个端点之间的距离从 $2$ 变成 $1$，有序距离和减少 $2$，所以：

$$
P=2(N-1)^2-2=2N^2-4N
$$

付费边数为：

$$
S=N-2
$$

方案二：把两条免费边都作为树边使用。可以理解成用一条付费边连接两条免费边所在的小连通块，其余点接到其中一个端点上。

相比方案一，这个方案少用一条付费边，但距离和会增加 $2N-4$，所以：

$$
P=2N^2-2N-4
$$

$$
S=N-3
$$

比较两个方案的总代价：

$$
(2N^2-2N-4)+Q(N-3)
$$

和：

$$
(2N^2-4N)+Q(N-2)
$$

方案二更优当且仅当：

$$
Q>2N-4
$$

否则选择方案一即可。

下面是正解：

```c++
#include <bits/stdc++.h>
using namespace std;

using ll = long long;
using pii = pair<ll, ll>;

const ll MOD = 998244353;
const ll INV2 = 499122177;

ll norm(ll x) {
    x %= MOD;
    if (x < 0) x += MOD;
    return x;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int T;
    cin >> T;
    while (T--) {
        ll n, q;
        int m;
        cin >> n >> q >> m;

        vector<pii> edges(m);
        for (auto &e : edges) {
            cin >> e.first >> e.second;
            if (e.first > e.second) swap(e.first, e.second);
        }

        sort(edges.begin(), edges.end());
        edges.erase(unique(edges.begin(), edges.end()), edges.end());

        int cnt = (int)edges.size();
        ll nm = n % MOD;
        ll qm = q % MOD;

        if (q <= 2) {
            ll total_pairs = nm * norm(n - 1) % MOD;
            ll paid_edges = norm(total_pairs * INV2 % MOD - cnt);
            cout << (total_pairs + qm * paid_edges) % MOD << '\n';
            continue;
        }

        ll P, S;

        if (cnt == 0) {
            P = 2 * norm(n - 1) % MOD * norm(n - 1) % MOD;
            S = norm(n - 1);
        } else if (cnt == 1) {
            P = 2 * norm(n - 1) % MOD * norm(n - 1) % MOD;
            S = norm(n - 2);
        } else {
            bool share = false;
            if (edges[0].first == edges[1].first) share = true;
            if (edges[0].first == edges[1].second) share = true;
            if (edges[0].second == edges[1].first) share = true;
            if (edges[0].second == edges[1].second) share = true;

            if (share) {
                P = 2 * norm(n - 1) % MOD * norm(n - 1) % MOD;
                S = norm(n - 3);
            } else {
                if (q > 2 * n - 4) {
                    P = norm(2 * nm % MOD * nm % MOD - 2 * nm % MOD - 4);
                    S = norm(n - 3);
                } else {
                    P = norm(2 * nm % MOD * nm % MOD - 4 * nm % MOD);
                    S = norm(n - 2);
                }
            }
        }

        cout << (P + qm * S) % MOD << '\n';
    }

    return 0;
}
```

由于 $M\le 2$，每个测试用例只需要常数次分类讨论，时间复杂度为 $O(1)$。空间复杂度也是 $O(1)$。
