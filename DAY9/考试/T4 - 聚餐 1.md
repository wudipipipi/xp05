# T112471.聚餐 1

## 题目描述

输入文件：`dinner.in`

输出文件：`dinner.out`

小 L 需要在一个 `n×m` 的餐厅中选择一个矩形区域用于聚餐。餐厅的每一个位置分为座位和餐桌，其中 `1` 代表座位，`0` 代表餐桌。小 L 想要希望选择的区域中不会出现拥挤，所以座位与座位不能相邻，同时他又希望坐更多的人，如果选择的矩形为 `a×b`，那么必须有大于 `ab/2` 个座位。小 L 想知道有多少种不同的选择聚餐区域的方案。

形式化地说，现在有一个 `n×m`（`n` 行，`m` 列）的网格，其中第 `i` 行，第 `j` 列的格子为 `a[i,j]`（`a[i,j]∈{0,1}`）。现在需要选出四个整数 `x1,y1,x2,y2` 满足 `1≤x1≤x2≤n`，`1≤y1≤y2≤m`，同时要满足以下条件：

1. 对于任意 `x3,x4∈[x1,x2]`，`y3,y4∈[y1,y2]` 若 `|x3−x4|+|y3−y4|=1`，则 `a[x3,y3]+a[x4,y4]≤1`；
2. `∑(i=x1..x2) ∑(j=y1..y2) a[i,j] > ((x2−x1+1)(y2−y1+1))/2`。

你需要求出有多少不同的 `x1,x2,y1,y2` 满足以上条件。

## 输入格式

第一行一个整数 `t`（`1≤t≤100`），表示有 `t` 组数据。

对于每组数据，第一行两个整数 `n,m`（`1≤n,m≤3000`）表示网格的大小。

接下来 `n` 行，每行包含 `m` 个字符 `0` 或 `1`，表示给出的 `a[i,j]`。

保证 `∑nm≤9×10^6`。

## 输出格式

对于每组数据输出一行一个整数表示满足条件的方案数。

## 输入输出样例

### 输入 #1

```text
3
3 3
101
010
101
5 5
10001
01010
10101
00100
10001
3 4
1001
1100
0110
```

### 输出 #1

```text
10
20
6
```

## 说明 / 提示

对于 `20%` 的数据保证 `1≤t≤5,1≤n,m≤20`；

对于 `40%` 的数据保证 `1≤t≤5,1≤n,m≤100`；

对于另外 `20%` 的数据保证 `a[i,j]=(i+j) mod 2`；

对于 `100%` 的数据保证 `1≤t≤100,1≤n,m≤3000,∑nm≤9×10^6`。
