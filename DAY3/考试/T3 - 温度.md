# T118321.温度

## 题目描述

吉东拥有一家烤肉店。这家店顾客盈门，所以很多人喜欢在去之前先预订。  
吉东努力想让顾客满意，甚至把所有顾客能接受的温度范围都记住了！他查看预订名单，想通过控制餐厅的温度来满足所有顾客。  
这家餐厅有一台奇怪的空调，有 3 种状态：关闭、制热和制冷。当空调关闭时，餐厅的温度保持不变。当空调制热时，温度每分钟升高 1 度。最后，当空调制冷时，温度每分钟降低 1 度。吉东可以在任意整数分钟时随意更改空调状态。空调初始处于关闭状态。

现在吉东告诉了你关于具体的情况：第 `i` 位顾客到店的时间 `ti`（以分钟计）；第 `i` 位顾客能接受的最低温度 `li`；第 `i` 位顾客能接受的最高温度 `ri`。

如果顾客到餐厅时温度处于其能接受的范围内，那么该顾客就是满意的。换句话说，第 `i` 位顾客在第 `ti` 分钟到店，顾客能接受的温度在 `li` 到 `ri` 之间。

已知初始温度、预约顾客的到店时间以及顾客能接受的温度范围，请你帮助吉东判断是否能满足所有顾客的要求。

## 输入格式

每个测试包含一个或多个测试用例。第一行包含测试用例的数量 `q`（`1≤q≤500`）。

每个测试用例的第一行包含两个整数 `n` 和 `m`（`1≤n≤100, -10^9≤m≤10^9`），其中 `n` 是预订顾客的数量，`m` 是餐厅的初始温度。

接下来有 `n` 行。其中第 `i` 行包含三个整数 `ti、li、hi`（`1≤ti≤10^9, -10^9≤li≤hi≤10^9`），其中 `ti` 表示第 `i` 位顾客到店的时间，`li` 表示第 `i` 位顾客能接受的最低温度，`hi` 表示第 `i` 位顾客能接受的最高温度。

顾客按照其到店时间的非递减顺序给出，当前时间为 `0`。

## 输出格式

对于每个测试用例，如果能够满足所有顾客的需求则输出 `YES`，否则输出 `NO`。

## 输入输出样例

### 输入 #1

```text
4
3 0
5 1 2
7 3 5
10 -1 0
2 12
5 7 10
10 16 20
3 -100
100 0 0
100 -50 50
200 100 100
1 100
99 -100 0
```

### 输出 #1

```text
YES
NO
YES
NO
```

## 说明 / 提示

在第一种情况下，吉东可以通过以下方式控制空调来满足所有顾客的需求：

1. 在第 `0` 分钟时，将状态更改为加热（此时温度为 `0`）。
2. 在第 `2` 分钟时，将状态更改为关闭（此时温度为 `2`）。
3. 在第 `5` 分钟时，将状态更改为加热（此时温度为 `2`，第 1 位顾客满意）。
4. 在第 `6` 分钟时，将状态更改为关闭（此时温度为 `3`）。
5. 在第 `7` 分钟时，将状态更改为冷却（此时温度为 `3`，第 2 位顾客满意）。
6. 在第 `10` 分钟时，温度将为 `0`，这满足了最后一位顾客的要求。

在第三种情况中，吉东可以在第 `0` 分钟时将状态更改为加热并保持不变。这样所有顾客都会满意。请注意，第 1 位顾客的访问时间等于第 2 位顾客的访问时间。

在第二种情况和第四种情况中，吉东至少会让一位顾客不满意。

## 测试点

1. `1∼1`：`n≤10, ti≤10`，`-10≤li, ri≤10` 且 `l(i-1)≤li, h(i-1)≤hi`
2. `2∼2`：`n≤10, ti≤10`，`-10≤li, ri≤10` 且 `li≤l(i-1), hi≤h(i-1)`
3. `3∼6`：`n≤5, ti≤5`，`0≤li, ri≤10`
4. `7∼10`：`n≤100, ti≤10^9`，`-10^9≤li, ri≤10^9`
