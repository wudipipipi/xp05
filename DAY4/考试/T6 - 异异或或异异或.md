# T118331.异异或或异异或

## 题目描述

给定一个数组 `a1, a2, ..., an`。计算满足以下条件的三元组 `(x, y, z)` 的数量：

`1≤x≤y≤z≤n`

`f(x, y) ⊕ f(y, z) > f(x, z)`

其中，`f(l, r) = al ⊕ a(l+1) ⊕ ... ⊕ ar`，`⊕` 表示按位异或运算。

## 输入格式

第一行包含一个整数 `t`（`1≤t≤10^4`）—— 测试用例的数量。

每个测试用例的第一行包含一个整数 `n`（`1≤n≤10^5`）。

每个测试用例的第二行包含 `n` 个整数 `a1, a2, ..., an`（`1≤ai≤10^9`）。

保证所有测试用例的 `n` 之和不超过 `10^5`。

## 输出格式

对于每个测试用例，输出一个整数——符合题目要求的三元组数量。

## 输入输出样例

### 输入 #1

```text
3
3
6 2 4
1
3
5
7 3 7 2 1
```

### 输出 #1

```text
4
0
16
```

## 说明 / 提示

### 样例说明

在第一个测试用例中，数组 `[6,2,4]` 共有 4 个满足条件的元组：

1. `(1,2,2)`：`(a1⊕a2)⊕(a2)=4⊕2>(a1⊕a2)=4`
2. `(1,1,3)`：`(a1)⊕(a1⊕a2⊕a3)=6⊕0>(a1⊕a2⊕a3)=0`
3. `(1,2,3)`：`(a1⊕a2)⊕(a2⊕a3)=4⊕6>(a1⊕a2⊕a3)=0`
4. `(1,3,3)`：`(a1⊕a2⊕a3)⊕(a3)=0⊕4>(a1⊕a2⊕a3)=0`

在第二个测试用例中，没有满足条件的元组。

## 测试点

1. `1∼4`：`n≤50`
2. `5∼10`：`n≤300`
3. `11∼20`：`n≤10^5`
