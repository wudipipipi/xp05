# 训猫

## 题目大意

有一棵 $N$ 个点的树，每个点有一个互不相同的高度 $P_i$，高度正好是 $1\sim N$ 的一个排列。猫一开始在高度为 $N$ 的点上。

每次可以选择一个没有障碍的点放障碍。如果放的不是猫所在的点，猫不动；如果放的是猫所在的点，那么猫会在当前还能到达的点中，选择高度最大的点移动过去，并且移动距离等于树上两点之间的距离。若没有可去的相邻点，游戏结束。

要求通过最优放障碍，使猫的总移动距离最大。

## 思路分析

首先观察猫的移动过程。因为猫每次会移动到当前可达点中高度最大的点，而当前所在点会被放上障碍，所以猫之后到达的点高度一定会越来越小。

也就是说，猫的移动路线可以看成一个高度严格递减的序列：

$$
N \to h_1 \to h_2 \to \cdots
$$

其中每一次移动都发生在树上两个点之间。

我们先考虑树是一条链的情况。

如果猫当前在某个位置，我们把这个位置放上障碍后，链会被分成左右两段。猫如果还能移动，那么它一定会移动到左边那段或者右边那段中高度最大的点。

反过来想会更自然。我们把点按高度从小到大加入。对于链来说，已经加入的点会形成若干个连续段，每个连续段中高度最大的点，就是这个连通块的代表点。

当加入一个新点 $u$ 时，它最多会连接左边和右边两个已经存在的连续段。设某个相邻连续段的代表点为 $v$，那么原问题中就可以从 $u$ 移动到 $v$；反过来做 DP，就是可以从 $v$ 转移到 $u$。

于是链上的转移就是：

$$
dp[u]=\max(dp[u],dp[v]+dist(u,v))
$$

其中 $v$ 是加入 $u$ 时，和 $u$ 相邻的某个已加入连续段的代表点。

这个思想可以直接推广到树上。

我们把每个点按照高度重新编号，也就是说高度为 $h$ 的点直接看作编号 $h$。这样猫的移动序列是从大编号往小编号走。为了方便 DP，我们反过来考虑，从小高度往大高度转移。

当处理高度 $u$ 时，所有高度小于 $u$ 的点都已经被加入。它们在树上形成若干个连通块。因为我们是按高度递增加入的，所以每个连通块里高度最大的点，就是最后被加入的那个点，也可以作为这个连通块的并查集代表。

如果 $u$ 和某个已经加入的连通块相邻，设这个连通块的代表为 $v$。那么在原问题里，当猫位于 $u$ 且进入这个连通块时，它一定会去这个连通块中高度最大的点，也就是 $v$。

所以原问题中存在一次移动：

$$
u\to v
$$

反过来做 DP，就得到转移：

$$
v\to u
$$

定义：

$$
dp[u]
$$

表示在反向过程中，最后走到高度 $u$ 时的最大距离和。处理 $u$ 时，枚举它相邻的已经处理过的连通块代表 $v$，转移：

$$
dp[u]=\max(dp[u],dp[v]+dist(u,v))
$$

转移完成后，把这些相邻连通块都合并到 $u$ 所在的连通块中。因为当前处理到的高度是 $u$，它一定是合并后连通块中高度最大的点，所以合并后的代表就是 $u$。

最后答案就是 $dp[N]$，也就是反向走到最高点的最大距离和。

为了快速计算 $dist(u,v)$，需要在按高度重编号后的树上预处理 LCA。

下面是正解：

```c++
#include <bits/stdc++.h>
using namespace std;

using ll = long long;

const int LOG = 20;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<int> p(n + 1);
    for (int i = 1; i <= n; i++) {
        cin >> p[i];
    }

    vector<vector<int>> g(n + 1);
    for (int i = 1; i < n; i++) {
        int u, v;
        cin >> u >> v;

        // 按高度重新编号
        u = p[u];
        v = p[v];
        g[u].push_back(v);
        g[v].push_back(u);
    }

    vector<array<int, LOG>> fa(n + 1);
    vector<int> dep(n + 1);

    function<void(int, int)> dfs = [&](int u, int pre) {
        dep[u] = dep[pre] + 1;
        fa[u][0] = pre;
        for (int i = 1; i < LOG; i++) {
            fa[u][i] = fa[fa[u][i - 1]][i - 1];
        }
        for (int v : g[u]) {
            if (v == pre) continue;
            dfs(v, u);
        }
    };

    dfs(1, 0);

    auto lca = [&](int x, int y) {
        if (dep[x] < dep[y]) swap(x, y);
        int d = dep[x] - dep[y];
        for (int i = 0; i < LOG; i++) {
            if (d >> i & 1) x = fa[x][i];
        }
        if (x == y) return x;
        for (int i = LOG - 1; i >= 0; i--) {
            if (fa[x][i] != fa[y][i]) {
                x = fa[x][i];
                y = fa[y][i];
            }
        }
        return fa[x][0];
    };

    auto dist = [&](int x, int y) {
        int z = lca(x, y);
        return dep[x] + dep[y] - 2 * dep[z];
    };

    vector<int> dsu(n + 1);
    iota(dsu.begin(), dsu.end(), 0);

    function<int(int)> find = [&](int x) {
        if (dsu[x] == x) return x;
        return dsu[x] = find(dsu[x]);
    };

    vector<ll> dp(n + 1, 0);

    for (int u = 1; u <= n; u++) {
        for (int v : g[u]) {
            int rt = find(v);

            // 只考虑已经加入的低高度连通块
            if (rt < u) {
                dp[u] = max(dp[u], dp[rt] + dist(u, rt));
            }
        }

        for (int v : g[u]) {
            int rt = find(v);
            if (rt < u) {
                dsu[rt] = u;
            }
        }
    }

    cout << dp[n] << '\n';

    return 0;
}
```

每条边只会在并查集合并过程中被处理常数次，LCA 预处理为 $O(N\log N)$，每次距离查询为 $O(\log N)$，所以总时间复杂度为 $O(N\log N)$。空间复杂度为 $O(N\log N)$。
