# 众最大数

## 题目大意

给定一个已经从小到大排序的可重集合 $S$，其中相同数值但下标不同的元素也被认为是不同元素。

现在枚举 $S$ 的所有非空子集，把每个子集的最大值放入新集合 $S'$ 中，要求输出 $S'$ 中出现次数最多的数。如果有多个数出现次数相同，输出其中最大的。

## 思路分析

首先考虑暴力做法。对于 $n\le 10$，可以直接枚举所有非空子集，统计每个子集最大值出现次数，这样可以拿到前 $30\%$ 的数据。但是 $n$ 最大有 $10^5$，显然不能枚举 $2^n$ 个子集。

我们来直接计算每个不同的数作为最大值会出现多少次。

假设某个值 $x$ 在排序后的数组中出现的位置是：

$$
l,l+1,\dots,r
$$

如果一个子集的最大值是 $x$，那么：

- 不能选择位置 $r+1$ 到 $n$ 的任何元素，因为它们都比 $x$ 大；
- 可以从前 $r$ 个元素中任选；
- 但必须至少选择一个位置在 $[l,r]$ 中的元素，否则最大值就小于 $x$。

所以最大值为 $x$ 的子集数量是：

$$
2^r-2^{l-1}
$$

下面看相邻两个不同值。假设当前值出现到位置 $r$，下一个更大的值至少出现在位置 $r+1$。当前值的出现次数小于：

$$
2^r
$$

而下一个值的出现次数至少为：

$$
2^{r+1}-2^r=2^r
$$

因此更大的那个值作为最大值的出现次数一定严格更多。

也就是说，随着数值增大，它在 $S'$ 中的出现次数一定增大。于是众数一定是原集合中的最大值，也就是最后一个元素 $s_n$。

下面是正解：

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    long long x, ans = 0;
    for (int i = 1; i <= n; i++) {
        cin >> x;
        ans = x;
    }

    cout << ans << '\n';

    return 0;
}
```

只需要读入所有数并输出最后一个，时间复杂度为 $O(n)$，空间复杂度为 $O(1)$。
