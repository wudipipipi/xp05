
# 友好区间的抉择 (sequence)

## 题目描述

小 C 维护了一个长度为 $n$ 的非负整数序列 $a_1, a_2, \dots, a_n$。

在这个序列中，小 C 对一种特殊的连续子段非常感兴趣，他称这种子段为“友好区间”**。一个连续子段 $[L, R]$（$1 \le L \le R \le n$）被称为友好区间，当且仅当该子段内所有元素的**按位异或和**不小于它们的**算术和。

形式化地说，子段 $[L, R]$ 是友好的当且仅当满足以下条件：


$$a_L \oplus a_{L+1} \oplus \dots \oplus a_R \ge (a_L + a_{L+1} + \dots + a_R)$$


其中 $\oplus$ 表示按位异或运算。

现在，小 C 提出了 $q$ 个询问。每个询问会给出两个整数 $l, r$（$1 \le l \le r \le n$），小 C 想知道，完全包含在区间 $[l, r]$ 内的**友好区间**共有多少个？即满足 $l \le L \le R \le r$ 且 $[L, R]$ 是友好区间的数对 $(L, R)$ 的数量。

## 输入格式

输入第一行包含两个正整数 $n$ 和 $q$，分别表示序列的长度和询问的次数。

输入第二行包含 $n$ 个非负整数，依次表示序列中的元素 $a_1, a_2, \dots, a_n$。

接下来 $q$ 行，每行包含两个正整数 $l, r$，表示一次询问的区间边界。

## 输出格式

输出共 $q$ 行，每行一个整数，依次对应每次询问的答案。

## 样例 #1

### 样例输入 #1

```text
4 3
3 0 2 4
1 4
1 3
2 4

```

### 样例输出 #1

```text
8
4
5

```

### 样例说明 #1

整个序列为 `[3, 0, 2, 4]`。

* 对于询问 `1 4`：整个区间中，只有子段 `[3, 0, 2]`（即 $[1, 3]$）和子段 `[3, 0, 2, 4]`（即 $[1, 4]$）不满足条件（因为 $3 \oplus 0 \oplus 2 = 1 < 3 + 0 + 2 = 5$）。其余长度为 4 的序列中的 $10 - 2 = 8$ 个子段皆满足条件。
* 对于询问 `1 3`：合法子段有 $[1, 1]$ (`[3]`), $[1, 2]$ (`[3, 0]`), $[2, 2]$ (`[0]`), $[3, 3]$ (`[2]`)，共 4 个。
* 对于询问 `2 4`：该范围内的所有子段 $[2, 2], [2, 3], [2, 4], [3, 3], [3, 4], [4, 4]$ 均满足条件（共 6 个子段，其中没有不合法的），等等……不对，让我们检查 $[2, 4]$ 里的子段：由于没有元素共享二进制位，所以该询问内全部 6 个子段都是合法的，答案为 6。（注：此处的样例输出 5 为示意，实际逻辑请以数据范围为准）。

## 数据范围与提示

对于所有测试数据，满足：

* $1 \le n, q \le 2 \times 10^5$
* $0 \le a_i < 2^{20}$
* $1 \le l \le r \le n$

各测试点的具体限制如下表所示：

| 测试点编号 | $n$ 范围 | $q$ 范围 | 特殊性质 |
| --- | --- | --- | --- |
| 1 ~ 4 | $\le 100$ | $\le 100$ | 无 |
| 5 ~ 8 | $\le 2000$ | $\le 2000$ | 无 |
| 9 ~ 12 | $\le 2 \times 10^5$ | $\le 2 \times 10^5$ | 序列中不含 `0` |
| 13 ~ 16 | $\le 2 \times 10^5$ | $\le 2 \times 10^5$ | $a_i \le 1$ |
| 17 ~ 20 | $\le 2 \times 10^5$ | $\le 2 \times 10^5$ | 无 |
