# 擦黑板

## 题目描述

黑板上有若干个数字 $1$、$2$、$3$。

一次操作中，可以进行如下步骤：

选择两个不同的数字，将它们从黑板上擦去，并写上一个与这两个数字都不同的数字。

例如，若黑板上有 $1,1,1,2,2,3$，选择数字 $1$ 和 $3$，擦去后写上一个 $2$，那么黑板上的数字就会变成 $1,1,2,2,2$。

经过任意次操作之后，黑板上能否只剩下一种数字。如果可以，请判断分别是否可能只剩下数字 $1$、$2$、$3$。

## 输入格式

每个测试包含多个测试用例。

第一行包含一个整数 $t$，表示测试用例的数量。

接下来 $t$ 行，每行包含三个整数 $a,b,c$，分别表示数字 $1$、$2$、$3$ 的个数。

## 输出格式

对于每个测试用例，输出一行，包含三个整数。

如果经过若干次操作后，黑板上可能只剩下数字 $1$，则第一个数输出 $1$，否则输出 $0$。

同样地，如果可能只剩下数字 $2$，则第二个数输出 $1$，否则输出 $0$。

同样地，如果可能只剩下数字 $3$，则第三个数输出 $1$，否则输出 $0$。

## 样例

### 样例 1 输入

```text
3
1 1 1
2 3 2
82 47 59
```

### 样例 1 输出

```text
1 1 1
0 1 0
1 0 0
```

## 提示

### 【样例 1 解释】

在第一个测试用例中，可以擦去数字 $2$ 和 $3$，并写上一个 $1$。之后，黑板上将只剩下 $1$ 的若干个副本。类似地，也可以通过相应的操作使黑板上最后只剩下数字 $2$ 或数字 $3$。

在第二个测试用例中，可以擦去数字 $1$ 和 $3$，并写上一个 $2$。经过 $2$ 次操作后，黑板上将只剩下数字 $2$。可以证明，不可能只剩下数字 $1$ 或只剩下数字 $3$。

在第三个测试用例中，存在一种操作序列，最终黑板上只剩下数字 $1$。可以证明，不可能只剩下数字 $2$ 或只剩下数字 $3$。

### 【数据范围】

对于所有测试数据，保证：

- $1 \leq t \leq 10^5$；
- $0 \leq a,b,c \leq 100$。

| 测试点编号 | $t$ | $a,b,c$ | 特殊性质 |
| :--: | :--: | :--: | :--: |
| $1 \sim 4$ | $1 \leq t \leq 10^5$ | $1 \leq a,b,c \leq 100$ | $a,b,c$ 均为奇数 |
| $5 \sim 8$ | $1 \leq t \leq 10^5$ | $1 \leq a,b,c \leq 100$ | $a,b,c$ 均为偶数 |
| $9 \sim 12$ | $1 \leq t \leq 10^5$ | $0 \leq a,b,c \leq 100$ | 至少有一个数为 $0$ |
| $13 \sim 20$ | $1 \leq t \leq 10^5$ | $0 \leq a,b,c \leq 100$ | 无 |
