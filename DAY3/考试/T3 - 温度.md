# 温度

## 题目描述

吉东经营着一家烤肉店。由于顾客很多，许多人会在到店前先进行预订。吉东想尽量满足所有顾客的需求，于是记录下了每位顾客能够接受的温度范围，并试图通过控制店内温度来让所有顾客满意。

这家餐厅有一台奇怪的空调，具有三种状态：关闭、制热和制冷。

- 当空调关闭时，餐厅温度保持不变；
- 当空调制热时，餐厅温度每分钟上升 $1$ 度；
- 当空调制冷时，餐厅温度每分钟下降 $1$ 度。

吉东可以在任意整数分钟改变空调状态，且空调初始处于关闭状态。

现在，吉东告诉你以下信息：

- 第 $i$ 位顾客到店的时间为 $t_i$；
- 第 $i$ 位顾客能接受的最低温度为 $l_i$；
- 第 $i$ 位顾客能接受的最高温度为 $r_i$。

如果顾客到店时，餐厅温度处于该顾客可接受的范围内，则该顾客满意。

已知初始温度、所有预约顾客的到店时间以及他们能接受的温度范围，请你判断是否可以让所有顾客都满意。

## 输入格式

每个测试包含多个测试用例。

第一行包含一个整数 $q$，表示测试用例的数量。

接下来是每个测试用例的描述。

每个测试用例的第一行包含两个整数 $n,m$，分别表示预约顾客的数量和餐厅的初始温度。

接下来 $n$ 行，每行包含三个整数 $t_i,l_i,r_i$，分别表示第 $i$ 位顾客到店的时间、可接受的最低温度和可接受的最高温度。

保证顾客按照到店时间的非递减顺序给出，当前时间为 $0$。

## 输出格式

对于每个测试用例，如果能够满足所有顾客的需求，则输出 `YES`，否则输出 `NO`。

## 样例

### 样例 1 输入

```text
4
3 0
5 1 2
7 3 5
10 -1 0
2 12
5 7 10
10 16 20
3 -100
100 0 0
100 -50 50
200 100 100
1 100
99 -100 0
```

### 样例 1 输出

```text
YES
NO
YES
NO
```

## 提示

### 【样例 1 解释】

在第一种情况下，吉东可以通过以下方式控制空调来满足所有顾客的需求：

- 在第 $0$ 分钟时，将状态更改为制热，此时温度为 $0$；
- 在第 $2$ 分钟时，将状态更改为关闭，此时温度为 $2$；
- 在第 $5$ 分钟时，将状态更改为制热，此时温度为 $2$，第 $1$ 位顾客满意；
- 在第 $6$ 分钟时，将状态更改为关闭，此时温度为 $3$；
- 在第 $7$ 分钟时，将状态更改为制冷，此时温度为 $3$，第 $2$ 位顾客满意；
- 在第 $10$ 分钟时，温度将为 $0$，这满足了最后一位顾客的要求。

在第三种情况下，吉东可以在第 $0$ 分钟时将状态更改为制热并保持不变，这样所有顾客都会满意。注意，第 $1$ 位顾客和第 $2$ 位顾客的到店时间相同。

在第二种情况和第四种情况中，吉东至少会让一位顾客不满意。

### 【数据范围】

对于所有测试数据，保证：

- $1 \leq q \leq 500$；
- $1 \leq n \leq 100$；
- $-10^9 \leq m \leq 10^9$；
- $1 \leq t_i \leq 10^9$；
- $-10^9 \leq l_i \leq r_i \leq 10^9$。

| 测试点编号 | $n \leq$ | $t_i \leq$ | $l_i,r_i$ | 说明 |
| :--: | :--: | :--: | :--: | :--: |
| $1$ | $10$ | $10$ | $[-10,10]$ | A |
| $2$ | $10$ | $10$ | $[-10,10]$ | B |
| $3 \sim 6$ | $5$ | $5$ | $[0,10]$ | 无 |
| $7 \sim 10$ | $100$ | $10^9$ | $[-10^9,10^9]$ | 无 |

特殊性质 A：对于所有 $2 \leq i \leq n$，均有 $l_{i-1} \leq l_i$ 且 $r_{i-1} \leq r_i$。

特殊性质 B：对于所有 $2 \leq i \leq n$，均有 $l_{i-1} \geq l_i$ 且 $r_{i-1} \geq r_i$。
