# Disappearing Permutation

## 题目描述

Dmitry有一个字符串$s$，由小写拉丁字母组成。

Dmitry决定从字符串$s$中删除两个连续的字符，你想知道在这样的操作后可以获得多少个不同的字符串。

例如，Dmitry有一个字符串"aaabcc"。您可以得到以下不同的字符串："abcc"（通过删除前两个或第二个和第三个字符），"aacc"（删除第三个和第四个字符）、"aaac"（删除第一个和第五个字符）和"aaab"（删除最后两个字符）。

## 输入格式

第一行输入数据包含一个整数t(1 \leq t \leq 10^{4})$------测试用例的数量。

测试用例的描述如下。

每个测试用例描述的第一行包含一个整数$n(3 \leq n \leq 2 \times 10^{5})$。

每个测试用例描述的第二行包含一个长度为$n$的字符串$s$
由小写拉丁字母组成。

保证所有测试用例的$n$之和不超过$2 \times 10^{5}。

## 输出格式

对于每个测试用例，输出一个整数------通过删除两个连续字母可以获得的不同字符串的数量。

## 输入样例

```
7
6
aaabcc
10
aaaaaaaaaa
6
abcdef
7
abacaba
6
cccfff
4
abba
5
ababa
```

## 输出样例

```
4
1
5
3
3
3
1
```

## 说明/提示

### 【样例1解释】

题目中解释了第一个例子。

在第三个示例中，获得了以下字符串："cdef"、"adef"、"abef"、"abcf"、"abcd"。

在第七个示例中，任何删除都将导致字符串"aba"。

### 数据范围

对于所有测试数据，都满足$1 \leq t \leq 10^{4}$，$ 3 \leq n \leq 2 \times 10^{5}$。

所有测试用例中 $n$ 的总和不超过$2 \times 10^{5}$。

|   测试点   |               $n$                |      $s$       |
| :--------: | :------------------------------: | :------------: |
| $1\sim 2$  |        $3 \leq n \leq 20$        | 只包含'a'，'b' |
| $3\sim 3$  | $ 3 \leq n \leq 2 \times 10^{3}$ | 只包含一种字符 |
| $4\sim 5$  | $ 3 \leq n \leq 2 \times 10^{4}$ | 相邻字符不相同 |
| $6\sim 10$ |     $ 3 \leq n \leq 10^{5}$      |                |

## 题目大意

从字符串$s$中删除两个连续的字符，操作后可以获得多少个不同的字符串。



## 思路分析

1.从测试点$1-2$考虑，删除的两个字符必须是相邻的，尝试暴力枚举，将每次得到的新字符串使用$set$存储，没出现一个新的未存在的字符串就累加答案。

2.接下来考虑测试点$3$，可以尝试特判$s$中所有的字符是否相同，如果相同，不论如何删除答案。

3.思考一下删除的是位置$i$和$i + 1$，以及位置$i + 1$和$i + 2$，对于新组成的字符串，前$i - 1$个位置与原字符串一致，只有第$i$个位置不同，对于第一种情况，留下的是$s[i + 2]$，对于第二种情况，留下的是$s[i]$，因此只需要考虑$s[i]$是否与$s[i + 2]$相等，若相等贡献减$1$。通过样例可以得出，删除两个字符串最多能得到$n - 1$个不同的字符串。时间复杂夫$O(n)$，可以通过所有的测试点。

```
#include <iostream>
#include <vector>
#include <queue>
#include <map>
#include <set>

using namespace std;

void solve() {
    int n;
    cin >> n;
    string s;
    cin >> s;
    int res = n - 1;
    for (int i = 1; i + 1 < n; ++i) {
        if (s[i - 1] == s[i + 1]) {
            res--;
        }
    }
    cout << res << '\n';
}

int main(int argc, char* argv[]) {
    int t;
    cin >> t;
    for (int i = 0; i < t; ++i) {
        solve();
    }
}
```
