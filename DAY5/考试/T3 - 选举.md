# 选举

输入文件：election.in

输出文件：election.out

## 题目描述

JOI 共和国由 $n$ 个州组成，现在正在进行选举。理惠想要赢得选举，于是决定通过发表演讲来提高支持率：

1. 在某州演讲时间达到 $a_i$，可获得一张选票；
2. 在某州演讲时间达到 $b_i$，可获得一名协作者。协作者可以与她同时在任何州演讲，多人在同一州演讲时，时间可以叠加；
3. 若 $b_i=-1$，表示无法获得协作者，且保证当 $b_i \neq -1$ 时，有 $b_i \geq a_i$。

请你求出，理惠最少需要演讲多少时间，才能获得至少 $k$ 张选票。

## 输入格式

输入的第一行包含一个整数 $n$。

输入的第二行包含一个整数 $k$。

接下来 $n$ 行，每行包含两个整数 $a_i$ 和 $b_i$，分别表示第 $i$ 个州的两个阈值。

## 输出格式

输出一个实数，表示所需的最小演讲时间。答案与标准答案的绝对误差不超过 $0.01$。

## 提示

### 【样例 1 解释】

按照如下方案进行演讲，理惠将在 $5.5$ 小时内赢得 $3$ 张选票。

- 在第 $2$ 个州演讲 $2$ 个小时，赢得一张选票；
- 在第 $2$ 个州再演讲 $1$ 个小时，获得一个协作者；
- 在第 $3$ 个州与协作者一起演讲 $2$ 个小时，赢得一张选票；
- 在第 $1$ 个州与协作者一起演讲 $0.5$ 个小时，赢得一张选票。

这个样例满足子任务 $3,4,5,6,7$ 的性质。

### 【样例 2 解释】

按照如下方案进行演讲，理惠将在 $32$ 小时内赢得 $4$ 张选票。

- 在第 $1$ 个州演讲 $4$ 个小时，赢得一张选票；
- 在第 $2$ 个州演讲 $11$ 个小时，赢得一张选票；
- 在第 $3$ 个州演讲 $6$ 个小时，赢得一张选票；
- 在第 $6$ 个州演讲 $11$ 个小时，赢得一张选票。

这个样例满足子任务 $1,2,3,4,5,7$ 的性质。

### 【样例 3 解释】

按照如下方案进行演讲，理惠将在 $11.5$ 小时内赢得 $3$ 张选票。

- 在第 $4$ 个州演讲 $7$ 个小时，赢得一张选票，并获得一个协作者；
- 在第 $1$ 个州演讲 $4$ 个小时，赢得一张选票。与此同时，协作者在第 $2$ 个州演讲 $4$ 个小时；
- 在第 $2$ 个州与协作者一起演讲 $0.5$ 个小时，赢得一张选票。

这个样例满足子任务 $2,3,4,5,7$ 的性质。

### 【样例 4 解释】

这个样例满足子任务 $3,4,5,7$ 的性质。

### 【样例 5 解释】

这个样例满足子任务 $4,5,7$ 的性质。

### 【数据范围】

本题采用测试点捆绑，同一行中的测试点作为一个测试点组计分。

对于所有测试数据，保证：

- $1 \leq n \leq 500$；
- $1 \leq k \leq n$；
- $1 \leq a_i \leq 10^3$；
- $a_i \leq b_i \leq 10^3$ 或者 $b_i=-1$。

| 子任务 | 分值 | $n$ | $k$ | 特殊性质 |
| :--: | :--: | :--: | :--: | :--: |
| $1 \sim 2$ | $4$ | $500$ | 无 | A |
| $3 \sim 4$ | $4$ | $500$ | 无 | B |
| $5 \sim 10$ | $12$ | $7$ | 无 | 无 |
| $11 \sim 16$ | $12$ | $20$ | 无 | 无 |
| $17 \sim 33$ | $34$ | $100$ | 无 | 无 |
| $34 \sim 38$ | $10$ | $500$ | $n$ | 无 |
| $39 \sim 50$ | $24$ | $500$ | 无 | 无 |

特殊性质 A：对于所有 $1 \leq i \leq n$，均有 $b_i=-1$。

特殊性质 B：对于所有 $1 \leq i \leq n$，均有 $b_i=-1$ 或 $b_i=a_i$。
