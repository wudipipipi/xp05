# 树影染色

## 题目描述

有一棵 $n$ 个点的树，根节点为 $1$。每个点都有一个颜色，颜色为 $0$ 或 $1$。

一开始，点 $i$ 的颜色为 $c_i$。之后，小 L 可以进行若干次操作，也可以一次都不进行。

每次操作时，小 L 会选择一个当前颜色为 $0$ 的节点 $u$，并将 $u$ 子树内所有节点的颜色取反。也就是说，对于 $u$ 子树内的每个点 $v$，执行

$$
c_v\leftarrow 1-c_v.
$$

所有操作结束后，点 $i$ 的颜色变为了 $c'_i$。

现在给定这棵树的结构以及最终颜色 $c'_1,c'_2,\ldots,c'_n$。请你求出有多少种不同的初始颜色序列 $c_1,c_2,\ldots,c_n$，可以经过若干次合法操作后得到给定的最终颜色。

答案对 $998244353$ 取模。

两种初始状态不同，当且仅当存在某个点在两种初始状态中的颜色不同。

## 输入格式

第一行包含一个整数 $n$，表示树的节点数。

接下来 $n-1$ 行，每行包含两个整数 $u,v$，表示树上有一条连接 $u$ 和 $v$ 的边。

最后一行包含 $n$ 个整数 $c'_1,c'_2,\ldots,c'_n$，表示每个点的最终颜色。

## 输出格式

输出一行一个整数，表示可能的初始状态数量对 $998244353$ 取模后的结果。

## 样例

### 样例 1 输入

```text
5
1 2
1 3
2 4
2 5
1 0 1 1 0
```

### 样例 1 输出

```text
20
```

## 数据范围与约定

对于所有测试数据，保证：

- $1 \le n \le 2 \times 10^5$；
- $0 \le c'_i \le 1$；
- 输入给出的是一棵树。

本题采用测试点捆绑计分，同一子任务内的所有测试点均通过才能获得该子任务分数。

| 子任务 | 测试点编号 | 分值 | $n$ | 特殊性质 |
| :--: | :--: | :--: | :--: | :--: |
| $1$ | $1 \sim 2$ | $8$ | $\le 15$ | 无 |
| $2$ | $3 \sim 5$ | $12$ | $\le 2 \times 10^5$ | A |
| $3$ | $6 \sim 7$ | $8$ | $\le 2 \times 10^5$ | B |
| $4$ | $8 \sim 10$ | $12$ | $\le 2 \times 10^5$ | C |
| $5$ | $11 \sim 13$ | $12$ | $\le 2 \times 10^5$ | D |
| $6$ | $14 \sim 17$ | $16$ | $\le 2 \times 10^5$ | E |
| $7$ | $18 \sim 30$ | $32$ | $\le 2 \times 10^5$ | 无 |

特殊性质 A：树是一条链，且对于 $1\le i<n$，节点 $i$ 与节点 $i+1$ 之间有一条边。

特殊性质 B：对于所有 $1\le i\le n$，均有 $c'_i=0$。

特殊性质 C：所有节点到根节点 $1$ 的距离不超过 $1$。

特殊性质 D：对于所有 $1\le i\le n$，均有 $c'_i=1$。

特殊性质 E：所有节点到根节点 $1$ 的距离不超过 $2$。
