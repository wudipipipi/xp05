## 题目大意

给定 $n$ 个正比例函数 $y=kx$，再给出 $m$ 个下凸二次函数 $y=ax^2+bx+c\ (a > 0)$，对于每一个二次函数，试图从给定的一次函数中找出一个函数，使得两个函数图像没有交点。

## 思路分析

两个函数图像有交点，明显等价于联立方程之后有解，所以我们实际上要找的是一条 $y=kx$ 满足 $kx = ax^2+bx+c$ 有解，移项可以得到 $ax^2+(b-k)x+c$。根据数学知识，我们可以通过 $\Delta = b^2+4ac$ 来判断这个方程是否有解，由于移项后 $x$ 的系数是 $(b-k)$，所以 $\Delta = (b-k)^2-4ac$。其中 $4ac$ 对于每一个二次函数都是固定的，所以我们只需要遍历所有的一次函数，找到一个函数满足 $(b-k)^2-4ac < 0$， 这个可以通过 $O(nm)$ 的复杂度暴力枚举，可以拿到 $30$ 分。

```c++
typedef long long ll;
void solve() {
    int n, m;
    cin >> n >> m;
    vector<ll> k(n);
    for (int i = 0; i < n; ++i) {
        cin >> k[i];
    }
    for (int i = 0; i < m; ++i) {
        ll a, b, c;
        cin >> a >> b >> c;
        ll tmp = 4 * a * c;
        bool flag = false;
        for (int j = 0; j < n; ++j) {
            if ((b - k[j]) * (b - k[j]) - tmp < 0) {
                cout << "YES" << endl;
                cout << k[j] << endl;
                flag = true;
                break;
            }
        }
        if(!flag) {
            cout << "NO" << endl;
        }
    }
    cout << endl;
}
```
这个题只需要我们找到一个存在的解就可以了，对于解没有更加具体的要求，所以因为我们的目标是让 $\Delta$ 尽可能小，实际上就是最小化 $(b-k)^2$，因此我们在对所有的 $k$ 排序之后，在里面 $\text{lower\_bound}$ 查找最接近 $b$ 的两个数字，判断是否存在无解即可，排序需要 $O(n\log n)$，每一组查询是 $O(\log n)$ 的，一共 $m$组，所以复杂度为 $O((n+m)\log n)$。
```c++
typedef long long ll;
void NeverSayNever() {
    int n, m;
    cin >> n >> m;
    vector<ll> k(n);
    for (int i = 0; i < n; ++i) {
        cin >> k[i];
    }
    sort(k.begin(), k.end());
    for (int i = 0; i < m; ++i) {
        ll a, b, c;
        cin >> a >> b >> c;
        int low = lower_bound(k.begin(), k.end(), b) - k.begin();
        int tmp = 4 * a * c;
        if (low != n && (b - k[low]) * (b - k[low]) - tmp < 0){
            cout << "YES" << endl;
            cout << k[low] << endl;
            continue;
        }
        if(low != 0 && (b - k[low-1]) * (b - k[low-1]) - tmp < 0){
            cout << "YES" << endl;
            cout << k[low-1] << endl;
            continue;
        }
        cout << "NO" << endl;
    }
    cout << endl;
}
```
