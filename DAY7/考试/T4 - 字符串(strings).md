# strings

输入文件：strings.in

输出文件：strings.out

## 题目描述

小 $\mathcal{l}$ 有一个长度为 $n$ 的字符串 $s$，字符集大小为 $m$。

定义一个长度为 $n$ 的字符串是可非回文划分的，当且仅当存在至少一个序列 $p$，满足其长度为 $k$，$2 \leq k \leq n$，且

$$
1=p_1<p_2<\cdots<p_{k-1}<p_k=n+1，
$$

并且对于所有 $1 \leq i < k$，子串 $s_{p_i}s_{p_i+1}\dots s_{p_{i+1}-1}$ 均不是回文串。

简单来说，一个字符串是可非回文划分的，当且仅当存在至少一种将其划分为若干非回文子串的划分方法。如果原串本身就不是回文串，那么它也满足这一条件。

例如字符串 $\texttt{abbba}$，存在 $k=3$，$p_1=1$，$p_2=3$，$p_3=6$，得到的字符串 $\texttt{ab}$、$\texttt{bba}$ 均不是回文串，所以字符串 $\texttt{abbba}$ 是可非回文划分的；字符串 $\texttt{aaaaa}$ 不存在这样的序列 $p$，故 $\texttt{aaaaa}$ 不是可非回文划分的。

现在小 $\mathcal{l}$ 的字符串中的一些字符看不清了，他想知道自己的字符串有多少种可能是可非回文划分的。如果一种情况都没有，答案为 $0$。你只需要输出答案对 $998244353$ 取模后的结果。

## 输入格式

输入的第一行包含一个整数 $t$，表示测试数据组数。

对于每组数据，第一行包含两个整数 $n,m$，分别表示字符串 $s$ 的长度和字符集大小。

接下来一行包含 $n$ 个整数，依次表示字符串 $s$。若 $s_i \in [1,m]$，则表示一个确定的字符；若 $s_i=-1$，则表示这个字符看不清了。

## 输出格式

对于每组数据，输出一行一个整数，表示可非回文划分的字符串数量对 $998244353$ 取模后的结果。

## 样例

### 样例 1 输入

```text
5
5 2
1 2 2 2 1
5 2
1 1 1 1 1
3 3
-1 -1 -1
4 5
1 -1 -1 2
6 99999
1 -1 1 -1 1 -1
```

### 样例 1 输出

```text
1
0
18
25
681058014
```

## 提示

### 【样例 1 解释】

第一组数据中，字符串为 $\texttt{abbba}$。可以将其划分为 $\texttt{ab}$ 与 $\texttt{bba}$，这两个子串都不是回文串，因此它是可非回文划分的，答案为 $1$。

第二组数据中，字符串为 $\texttt{aaaaa}$。无论如何划分，都会出现回文子串，因此它不是可非回文划分的，答案为 $0$。

第三组数据中，长度为 $3$，字符集大小为 $3$。共有 $27$ 种可能的字符串，其中有 $18$ 种是可非回文划分的。

第四组数据中，无论两个看不清的字符分别是什么，原串的首尾字符都不同，因此原串本身不是回文串，所有 $25$ 种情况均满足条件。

### 【数据范围】

本题采用测试点捆绑，同一行中的测试点作为一个测试点组计分。

对于所有测试数据，保证：

- $1 \leq t \leq 100$；
- $1 \leq n,m \leq 10^5$；
- $\sum n \leq 5 \times 10^5$；
- $s_i \in [1,m]$ 或 $s_i=-1$。

| 子任务 | 分值 | $n$ | $m$ | $\sum n$ | 特殊性质 |
| :--: | :--: | :--: | :--: | :--: | :--: |
| $1$ | $10$ | $7$ | $7$ | $700$ | 无 |
| $2$ | $30$ | $1000$ | $1000$ | $5000$ | 无 |
| $3$ | $10$ | $10^5$ | $1$ | $5 \times 10^5$ | 无 |
| $4$ | $10$ | $10^5$ | $10^5$ | $5 \times 10^5$ | A |
| $5$ | $20$ | $10^5$ | $2$ | $5 \times 10^5$ | 无 |
| $6$ | $20$ | $10^5$ | $10^5$ | $5 \times 10^5$ | 无 |

特殊性质 A：对于所有 $1 \leq i \leq n$，均有 $s_i \neq -1$。
