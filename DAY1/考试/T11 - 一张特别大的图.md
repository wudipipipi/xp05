# T4- 一张特别大的图

## 题目描述

小码君画了一棵有 $n$ 个节点的树，节点编号为 $1 \sim n$。他觉得这张图还不够大，于是准备按照“距离越远越要连起来”的奇怪规则，重新生成很多张图。

给定这棵 $n$ 个节点的无根树。

对于每个整数 $k$，定义一张无向图 $G_k$。$G_k$ 的点集为 $\{1, 2, \ldots, n\}$，对于任意两个不同节点 $u, v$，若它们在原树上的距离大于等于 $k$，则在 $G_k$ 中连接一条无向边 $(u, v)$。

请你分别求出当 $k = 1, 2, \ldots, n$ 时，图 $G_k$ 的连通块个数。

## 输入格式

第一行包含一个整数 $n$，表示树的节点数。

接下来 $n - 1$ 行，每行包含两个整数 $u, v$，表示树上的一条无向边。

## 输出格式

输出一行 $n$ 个整数，第 $k$ 个整数表示图 $G_k$ 的连通块个数。

## 样例

### 样例 1 输入

```text
6
1 2
1 3
2 4
2 5
3 6
```

### 样例 1 输出

```text
1 1 2 4 6 6
```

### 样例 2 输入

```text
5
1 2
2 3
3 4
3 5
```

### 样例 2 输出

```text
1 1 3 5 5
```

## 提示

### 【样例 1 解释】

当 $k = 1$ 时，$G_k$ 中所有节点都连通。

当 $k = 4$ 时，$G_k$ 中只有 $(4, 6)$ 和 $(5, 6)$ 两条边，因此连通块个数为 $4$。

### 【样例 2 解释】

当 $k = 1$ 时，$G_k$ 中所有节点都连通。

当 $k = 2$ 时，所有点都与树上距离大于等于 $2$ 的点连边，形成的图依然连通。

当 $k = 3$ 时，节点 $1, 4, 5$ 构成一个连通块，其余两个节点各自构成一个连通块，因此连通块个数为 $3$。

当 $k = 4$ 或 $k = 5$ 时，树上没有任意一对点之间的距离大于等于 $4$，因此图中没有边，连通块个数为 $n$。

### 【数据范围】

对于所有测试数据，保证：

- $2 \leq n \leq 10^5$；
- $1 \leq u, v \leq n$；
- 输入的 $n - 1$ 条边构成一棵树。

| 测试点编号 | $n \leq$ | 特殊性质 |
| :--: | :--: | :--: |
| $1 \sim 4$ | $1000$ | 无 |
| $5 \sim 6$ | $10^5$ | A |
| $7 \sim 8$ | $10^5$ | B |
| $9 \sim 10$ | $10^5$ | C |
| $11 \sim 20$ | $10^5$ | 无 |

特殊性质 A：保证所有节点的度数均不超过 $2$。

特殊性质 B：保证数据随机生成。

特殊性质 C：保证存在一个度数为 $n - 1$ 的节点。
