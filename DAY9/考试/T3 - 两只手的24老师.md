# 两只手的24老师

输入文件：card.in

输出文件：card.out

## 题目描述

有 $2n$ 张卡牌从左到右依次放在桌子上，编号为 $1,2,\dots,2n$。第 $i$ 张卡牌上写有整数 $a_i$。对于每个 $x=1,2,\dots,n$，恰好有两张卡牌上写的整数为 $x$。

24 老师决定用这些卡牌玩一个游戏。游戏流程如下：

依次考虑卡牌 $i=1,2,\dots,2n$。对于当前卡牌，24 老师可以选择拿起它，也可以选择跳过它。

如果他选择跳过该卡牌，则什么也不会发生。

如果他选择拿起该卡牌，则依次执行以下步骤：

1. 如果 24 老师的手中已经持有一张写有 $a_i$ 的卡牌，那么该卡牌和卡牌 $i$ 同时消失，他获得 $v_{a_i}$ 分；
2. 如果 24 老师的左手中有一张卡牌，那么他将其丢弃；
3. 如果 24 老师的右手中有一张卡牌，那么他将其转移到左手；
4. 如果卡牌 $i$ 没有在第 $1$ 步中消失，那么他将其放在右手中。

整个游戏过程中获得的分数之和即为 24 老师的最终得分。

请你求出 24 老师能获得的最大得分。

## 输入格式

输入的第一行包含一个整数 $n$。

输入的第二行包含 $2n$ 个整数 $a_1,a_2,\dots,a_{2n}$。

输入的第三行包含 $n$ 个整数 $v_1,v_2,\dots,v_n$。

## 输出格式

输出一行一个整数，表示最大得分。

## 样例

### 样例 1 输出

## 提示

### 【样例 1 解释】

24 老师可以通过以下流程获得 $13$ 分：

1. 拿起卡牌 $1$，该卡牌上写着 $1$。由于 24 老师没有其他写着 $1$ 的卡牌，所以他将其放在右手中；
2. 跳过卡牌 $2$；
3. 拿起卡牌 $3$，该卡牌上写着 $3$。由于 24 老师没有其他写着 $3$ 的卡牌，所以卡牌 $1$ 从右手转移到左手，他将卡牌 $3$ 放在右手中；
4. 拿起卡牌 $4$，该卡牌上写着 $1$。由于卡牌 $1$ 上也写着 $1$，所以两张牌都消失了，他获得 $v_1=5$ 分，卡牌 $3$ 从右手转移到左手；
5. 跳过卡牌 $5$；
6. 拿起卡牌 $6$，该卡牌上写着 $3$。由于卡牌 $3$ 上也写着 $3$，所以两张牌都消失了，他获得 $v_3=8$ 分，此时他两只手上都没有任何卡牌。

可以证明这是最优的。

### 【样例 2 解释】

24 老师可以通过拿起卡牌 $1,2,3,4,5,6,8$ 来获得 $v_1+v_2+v_3=156$ 分。可以证明这是最优的。

### 【数据范围】

本题采用测试点捆绑，同一行中的测试点作为一个测试点组计分。

对于所有测试数据，保证：

- $1 \leq n \leq 4 \times 10^5$；
- 对于每个 $x=1,2,\dots,n$，$x$ 在 $a_1,a_2,\dots,a_{2n}$ 中恰好出现两次；
- $1 \leq v_k \leq 10^9$。

| 子任务 | 分值 | $n$ | 特殊性质 |
| :--: | :--: | :--: | :--: |
| $1 \sim 4$ | $8$ | $5000$ | A |
| $5 \sim 10$ | $12$ | $4 \times 10^5$ | A |
| $11 \sim 13$ | $6$ | $9$ | 无 |
| $14 \sim 18$ | $10$ | $18$ | 无 |
| $19 \sim 26$ | $16$ | $300$ | 无 |
| $27 \sim 35$ | $18$ | $3000$ | 无 |
| $36 \sim 44$ | $18$ | $1.5 \times 10^5$ | B |
| $45 \sim 48$ | $8$ | $1.5 \times 10^5$ | 无 |
| $49 \sim 50$ | $4$ | $4 \times 10^5$ | 无 |

特殊性质 A：$(a_1,a_2,\dots,a_n)=(1,2,\dots,n)$。

特殊性质 B：对于所有 $1 \leq k \leq n$，均有 $v_k=1$。
