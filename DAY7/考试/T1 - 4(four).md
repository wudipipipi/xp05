# four

输入文件：four.in

输出文件：four.out

## 题目描述

小 $\mathcal{l}$ 有一个整数 $n$。

小 $\mathcal{l}$ 喜欢 $4$ 的倍数，他可以对这个整数施加魔法。每次魔法可以交换这个整数在十进制表示下任意两位的值，交换后的整数可以包含前导零。例如，$123 \to 132$，$9876 \to 7896$，$200 \to 002$，$3507 \to 0537$。

注意，原数字不包含前导零。例如，不能将 $2$ 视为 $02$，再通过交换得到 $20$。

现在小 $\mathcal{l}$ 想要知道，最少施加几次魔法才能让整数 $n$ 变为 $4$ 的倍数。

## 输入格式

输入的第一行包含一个整数 $t$，表示测试数据组数。

接下来 $t$ 行，每行包含一个整数 $n$，表示一次询问。

## 输出格式

对于每组测试数据，输出一行一个整数，表示最少需要施加魔法的次数。如果无论施加多少次魔法都不能变为 $4$ 的倍数，则输出 $-1$。

## 样例

### 样例 1 输入

```text
5
1001
4013
1111
114514
142857
```

### 样例 1 输出

```text
1
2
-1
1
1
```

## 提示

### 【样例 1 解释】

对于 $1001$，交换第 $2$ 位和第 $4$ 位可得到 $1100$，此时 $1100$ 是 $4$ 的倍数。原数 $1001$ 不是 $4$ 的倍数，因此答案为 $1$。

对于 $4013$，可以先交换第 $1$ 位和第 $4$ 位，得到 $3014$；再交换第 $2$ 位和第 $3$ 位，得到 $3104$。此时 $3104$ 是 $4$ 的倍数，可以证明无法通过 $1$ 次交换达到目标，因此答案为 $2$。

对于 $1111$，任意交换后得到的数仍然是 $1111$，不是 $4$ 的倍数，因此答案为 $-1$。

对于 $114514$，交换第 $3$ 位和第 $5$ 位，得到 $111544$，它是 $4$ 的倍数，因此答案为 $1$。

对于 $142857$，交换第 $3$ 位和第 $6$ 位，得到 $147852$，它是 $4$ 的倍数，因此答案为 $1$。

### 【数据范围】

本题采用测试点捆绑，同一行中的测试点作为一个测试点组计分。

对于所有测试数据，保证：

- $1 \leq t \leq 10^5$；
- $1 \leq n \leq 10^9$。
