# 一张特别大的图

## 题目描述

给定一棵 $n$（$2\leq n\leq 10^5$）个节点的无根树，对于每个 $k$（$1\leq k \leq n$），定义一个无向图 $G_k$，其中由边连接的两个节点 $u$ 和 $v$ 的距离至少为 $k$。请你计算 $G_k$ 的连通块个数。

## 输入格式

第一行包含整数 $n$。

接下来 $n-1$ 行，每行包含两个整数 $u$ 和 $v$，表示树上的一条无向边。

## 输出格式

输出共 $n$ 行，每行一个整数，表示 $G_k$ 的连通块个数。


### 输入样例1

```
6
1 2
1 3
2 4
2 5
3 6
```

### 输出样例1

```
1 1 2 4 6 6
```

### 输入样例2

```
5
1 2
2 3
3 4
3 5
```

### 输出样例3

```
1 1 3 5 5
```
## 说明/提示

#### 【样例 1 解释】

当 $k=1$ 时，所有的点都连通；

当 $k=4$ 时，只有$(4,6)$和$(5,6)$两条边连通，所以连通块个数为4；

#### 【样例 2 解释】

当 $k=1$ 时，所有的点都连通；

当 $k=2$ 时，所有的点都与其距离大于等于 $2$ 的所有点连边，形成的图依然连通；

当 $k=3$ 时，节点 $\{1,4,5\}$ 构成一个连通块，其余两个节点分别为一个连通块；

当 $k=4/5$ 时，树上没有任何一对点之间的距离 $\geq 4$，因此图没有任何边，此时联通块个数等于点数。



对于所有测试点都满足 $2\leq n \leq 10^5$。

|   测试点    |          n          |        特殊性质         |
| :---------: | :-----------------: | :---------------------: |
|  $1\sim 4$  | $2\leq n \leq 1000$ |           无            |
|  $5\sim 6$  | $2\leq n \leq 10^5$ |  所有点的度数 $\leq 2$  |
|  $7\sim 8$  | $2\leq n \leq 10^5$ |      数据随机生成       |
| $9\sim 10$  | $2\leq n \leq 10^5$ | 存在度数为 $n−1$ 的结点 |
| $11\sim 10$ | $2\leq n \leq 10^5$ |           无            |


## 题目大意

给定一棵 $n$（$2\leq n\leq 10^5$）个节点的无根树，对于每个 $k$（$1\leq k \leq n$），定义一个无向图 $G_k$，其中由边连接的两个节点 $u$ 和 $v$ 的距离至少为 $k$。请你计算 $G_k$ 的连通块个数。

## 思路分析

如果一个点到它距离最远的点的距离小于 $k$ 的话，这个点一定是独立成为连通块的。而对于一个点来说，其最远的点一定是树直径的端点之一，如果一个点里直径端点的距离大于等于 $k$，那么这个点一定会和端点成为连通块。所以每一个点只有两种可能，要么和端点连边成为一个连通块，要么自身独立成为一个连通块。

因此做法是先 DFS 找出两个直径端点，之后对于每一个端点处理出到达其他点的距离。

这时候可以选择枚举 $k$ 同时检查所有点到两个端点的距离是否有一个 $\geq k$，如果有的话这个点之后就会独自成为一个连通块，否则就一定会在两个端点所形成的连通块里，这样的做法是 $O(n^2)$ 的，可以通过四个测试点。

对于菊花图而言，$k=1$ 时答案一定是 $1$，当 $k=2$ 时，中心点会自己成为一个连通块，其他的点会成为一个连通块，当 $k\geq 3$ 的时候，因为菊花图内一定不存在距离 $\geq 3$ 的点，所以答案和点数一致。因此只需要构造 $\{1,2,n,\cdots,n\}$ 的长度为 $n$ 的答案就可以通过数据为菊花图的测试点。

那么正解其实是我们维护一个 $ans$ 数组，显然一个点会独立当且仅当 $k$ 超过了他距离的最远端点的距离都超过了 $k$，因此我们对于每个点 $i$，让 $ans[\max(dis1[i],dis2[i])]$ 加一。之后对 $ans$ 数组做前缀和就是最后的答案。

```c++
#include <bits/stdc++.h>
using namespace std;
const int N = 5e5 + 5;
int x,y,dis,dis1[N],dis2[N];
vector<vector<int>> adj(N);
bool vis[N];
void dfs1(int cur,int fa,int cnt) {
    if (cnt > dis) {
        x = cur;
        dis = cnt;
    }
    for (int nxt: adj[cur]) {
        if (nxt == fa) continue;
        dfs1(nxt, cur, cnt + 1);
    }
}
void dfs2(int cur,int fa,int cnt) {
    if (cnt > dis) {
        y = cur;
        dis = cnt;
    }
    for (int nxt: adj[cur]) {
        if (nxt == fa) continue;
        dis1[nxt] = cnt + 1;
        dfs2(nxt, cur, cnt + 1);
    }
}
void dfs3(int cur ,int fa, int cnt) {
    for (int nxt: adj[cur]) {
        if (nxt == fa) continue;
        dis2[nxt] = cnt + 1;
        dfs3(nxt, cur, cnt + 1);
    }
}
int main() {
    int n;
    cin >> n;
    for (int i = 1; i < n; i++) {
        int u, v;
        cin >> u >> v;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
    dfs1(1, 1, 0);
    dis = 0;
    dfs2(x, x, 0);
    dfs3(y, y, 0);
    vector<int> ans(n + 1);
    for (int i = 1; i <= n; i++) {
        if (i != y) {
            ans[max(dis1[i], dis2[i]) + 1]++;
        }
    }
    ans[0] = 1;
    for (int i = 1; i <= n; i++) {
        ans[i] += ans[i - 1];
        cout << ans[i] << ' ';
    }
    return 0;
}
```
