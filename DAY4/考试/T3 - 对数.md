# 对数

## 题目描述

现在有一个数组 $a$，包含 $n$ 个整数。请找出所有满足如下条件的数对 $(i,j)$：

$l \leq a_i+a_j \leq r$，其中 $1 \leq i < j \leq n$。

也就是说，数对的和 $a_i+a_j$ 需要大于等于 $l$，且小于等于 $r$。

例如，当 $n=3$，$a=[5,1,2]$，$l=4$，$r=7$ 时，有以下两个符合条件的数对：

1. $i=1$，$j=2$，因为 $4 \leq 5+1 \leq 7$；
2. $i=1$，$j=3$，因为 $4 \leq 5+2 \leq 7$。

## 输入格式

每个测试包含多个测试用例。

输入的第一行包含一个整数 $t$，表示测试用例的数量。

对于每个测试用例，输入的第一行包含三个整数 $n,l,r$，分别表示数组的长度以及数对和的上下限。

输入的第二行包含 $n$ 个整数 $a_1,a_2,\dots,a_n$，表示数组 $a$。

## 输出格式

对于每个测试用例，输出一行一个整数，表示满足条件 $l \leq a_i+a_j \leq r$ 且 $i<j$ 的数对 $(i,j)$ 的数量。

## 样例

### 样例 1 输入

```text
4
3 4 7
5 1 2
5 5 8
5 1 2 4 3
4 100 1000
1 1 1 1
5 9 13
2 5 5 1 1
```

### 样例 1 输出

```text
2
7
0
1
```

## 提示

### 【样例 1 解释】

对于第一个测试用例，符合条件的数对有 $(1,2)$ 和 $(1,3)$，因此答案为 $2$。

对于第二个测试用例，满足条件的数对共有 $7$ 个，因此答案为 $7$。

对于第三个测试用例，任意两个数的和都小于 $100$，因此答案为 $0$。

对于第四个测试用例，只有数对 $(1,2)$ 满足条件，因此答案为 $1$。

### 【数据范围】

对于所有测试数据，保证：

- $1 \leq t \leq 10^4$；
- $1 \leq n \leq 2 \times 10^5$；
- $1 \leq l \leq r \leq 10^9$；
- $1 \leq a_i \leq 10^9$；
- 所有测试用例的 $n$ 之和不超过 $2 \times 10^5$。

| 测试点编号 | $n \leq$ |
| :--: | :--: |
| $1 \sim 6$ | $1000$ |
| $7 \sim 20$ | $2 \times 10^5$ |
