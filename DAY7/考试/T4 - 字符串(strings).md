# T112105.字符串(strings)

## 题目描述

小 L 有一个长度为 `n` 的字符串 `S` 是 Non-Palindrome，字符集的大小为 `m`。

定义一个长度为 `n` 的字符串为 Non-Palindrome，当且仅当存在至少一个序列 `p`。

满足其长度为 `2≤k≤n`，且

`1=p1<p2<⋯<p(k−1)<pk=n+1`

且字符串的 `S[p_i]S[p_i+1]...S[p(i+1)−1]`，`i∈[1,k)` 均不是回文串。

简单来说，一个字符串为 Non-Palindrome 当且仅当存在至少一种将其划分为若干非回文子串的划分方法。（如果原串本身就不是回文串，那么也为 Non-Palindrome）

例如字符串 `abbba`，存在 `k=3, p1=1, p2=3, p3=6`，得到的字符串 `ab, bba` 均不是回文串，所以字符串 `abbba` 是 Non-Palindrome；字符串 `aaaaa` 不存在这样的序列 `p`，故 `aaaaa` 不是 Non-Palindrome。

现在小 L 的字符串中的一些字符看不清了，他想知道自己的字符串有多少种可能（一种情况都没有则为 `0`），你只需要计算出可能的情况数除以 `998244353` 后的余数即可。

## 输入格式

第一行一个整数 `t`，表示有 `t` 组数据。

对于每组数据，第一行两个整数 `n,m`，表示字符串 `S` 的长度和字符集的大小。

接下来一行包含 `n` 个整数，依次表示字符串 `S`。若 `Si∈[1,m]` 则表示一个确定的字符，若 `Si=-1` 则表示这个字符看不清了。

## 输出格式

对于每组数据输出这个整数，表示满足是 Non-Palindrome 的字符串的个数除以 `998244353` 后的余数。

## 输入输出样例

### 输入 #1

```text
5
5 2
1 2 2 2 1
5 2
1 1 1 1 1
3 3
-1 -1 -1
4 5
1 -1 -1 2
6 99999
1 -1 1 -1 1 -1
```

### 输出 #1

```text
1
0
18
25
681058014
```

## 说明 / 提示

对于 `10%` 的数据保证 `1≤n,m≤7`；

对于 `30%` 的数据保证 `1≤n,m≤1000，∑n≤5000`；

对于另外 `10%` 的数据保证 `m=1`；

对于另外 `10%` 的数据保证 `Si≠−1`；

对于另外 `20%` 的数据保证 `m≤2`；

对于 `100%` 的数据保证 `1≤n,m≤10^5`，`1≤t≤100`，`∑n≤5×10^5`，`Si∈[1,m]` 或 `Si=-1`。

题目采用文件读写：

输入文件：`strings.in`

输出文件：`strings.out`
