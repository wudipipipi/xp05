# 4

## 题目大意

给定一个整数 $n$，每次操作可以交换十进制表示中的任意两位，交换后允许出现前导零。问最少操作多少次，可以让这个数变成 $4$ 的倍数；如果无论如何都不行，输出 $-1$。

## 思路分析

首先利用一个基本性质：一个十进制整数能否被 $4$ 整除，只和它的最后两位有关。

所以我们不需要关心整个数字，只需要考虑能否通过交换，把某两位数字放到最后两位，使得这两位组成的两位数是 $4$ 的倍数。

对于长度为 $1$ 的数，只能判断它本身是不是 $4$ 的倍数。因为题目中 $n\ge 1$，所以只有 `4` 和 `8` 可以，其他都不行。

对于长度至少为 $2$ 的数，答案最多是 $2$。因为只要存在两个不同位置的数字 $x,y$，使得组成的两位数 $10x+y$ 是 $4$ 的倍数，我们最多只需要两次交换，就可以把 $x$ 放到十位、$y$ 放到个位。

于是可以按照答案从小到大检查。

### 0 次操作

如果原来的最后两位组成的数已经是 $4$ 的倍数，直接输出 $0$。

### 1 次操作

一次交换只会影响两个位置。想让最后两位变合法，有几种情况：

1. 直接交换最后两位；
2. 把前面的某一位换到个位；
3. 把前面的某一位换到十位。

只要其中一种能让最后两位成为 $4$ 的倍数，答案就是 $1$。

### 2 次操作

如果 $0$ 次和 $1$ 次都不行，那么继续判断是否存在任意两个不同位置的数字，可以组成一个 $4$ 的倍数。

如果存在，那么最多两次交换即可：

- 第一次把十位需要的数字换到倒数第二位；
- 第二次把个位需要的数字换到最后一位。

如果不存在，就说明无论怎么交换都凑不出合法末两位，答案为 $-1$。

下面是正解：

```c++
#include <bits/stdc++.h>
using namespace std;

void solve() {
    string s;
    cin >> s;

    int n = (int)s.size();

    if (n == 1) {
        if (s[0] == '4' || s[0] == '8') {
            cout << 0 << '\n';
        } else {
            cout << -1 << '\n';
        }
        return;
    }

    auto ok = [](char x, char y) {
        int val = (x - '0') * 10 + (y - '0');
        return val % 4 == 0;
    };

    // 0 次操作
    if (ok(s[n - 2], s[n - 1])) {
        cout << 0 << '\n';
        return;
    }

    // 1 次操作：交换最后两位
    if (ok(s[n - 1], s[n - 2])) {
        cout << 1 << '\n';
        return;
    }

    // 1 次操作：用前面的某一位替换十位或个位
    for (int i = 0; i < n - 2; i++) {
        if (ok(s[n - 2], s[i])) {
            cout << 1 << '\n';
            return;
        }
        if (ok(s[i], s[n - 1])) {
            cout << 1 << '\n';
            return;
        }
    }

    // 2 次操作：任意两位能否组成 4 的倍数
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            if (i == j) continue;
            if (ok(s[i], s[j])) {
                cout << 2 << '\n';
                return;
            }
        }
    }

    cout << -1 << '\n';
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int T;
    cin >> T;
    while (T--) {
        solve();
    }

    return 0;
}
```

由于 $n\le 10^9$，字符串长度最多为 $10$，所以即使最后枚举两位也是常数复杂度。单组数据时间复杂度为 $O(L^2)$，其中 $L$ 是数字位数，在本题中可以看作 $O(1)$。
