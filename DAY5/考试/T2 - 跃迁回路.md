# 跃迁回路

输入文件：loop.in

输出文件：loop.out

## 题目描述

某公司正在开发一种跳跃机器人。为了测试机器人，他们在一个多边形平台上设置了一个由 $n$ 个特殊平台组成的环形路线，平台从 $1$ 到 $n$ 编号。第 $i$ 个平台与第 $i+1$ 个平台之间的距离为 $d_i$，最后一个平台与第一个平台之间的距离为 $d_n$，其中长度分别为 $d_1,d_2,\dots,d_n$ 的边可以组成一个 $n$ 边形。

机器人配备了人工智能，在测试过程中会不断学习跳得更远。在任何时刻，机器人都有一个整数 $a$ 表示它的灵敏度。如果 $a \geq d_i$，机器人可以从平台 $i$ 跳到平台 $i+1$；同样地，如果 $a \geq d_n$，机器人可以从最后一个平台跳到第一个平台。每次跳跃后，机器人的灵敏度增加 $1$。

机器人的开发人员会选择一个平台作为起始平台。如果机器人可以完成 $n$ 次跳跃并回到原来的平台，则认为实验成功。开发人员需要确定机器人的最小起始灵敏度，并选择哪个平台作为起始平台。

## 输入格式

输入的第一行包含一个整数 $n$。

输入的第二行包含一个整数 $f$。

- 如果 $f=1$，则输入的第三行包含 $n$ 个整数 $d_1,d_2,\dots,d_n$。
- 如果 $f=2$，则输入的第三行包含一个整数 $m$，以及三个整数 $x,y,z$。输入的第四行包含 $m$ 个整数 $c_1,c_2,\dots,c_m$。此时距离值 $d_i$ 根据以下公式计算：
  - 如果 $1 \leq i \leq m$，则 $d_i=c_i$；
  - 如果 $m+1 \leq i \leq n$，则 $d_i = ((x \times d_{i-2} + y \times d_{i-1} + z) \bmod 10^9) + 1$。

## 输出格式

输出两个整数，分别表示最小允许的起始灵敏度 $a$ 和可用于放置机器人的起始平台编号。

如果有多个最小起始灵敏度对应的起始平台，输出编号最小的起始平台。

## 提示

### 【样例解释】

在第二个示例中，距离数组为 $[1,2,3,4,5,18,45,112,273,662]$。

根据公式计算 $d_6$ 到 $d_{10}$ 的值：

- $d_6 = ((1 \cdot d_4 + 2 \cdot d_5 + 3) \bmod 10^9) + 1 = 18$；
- $d_7 = ((1 \cdot d_5 + 2 \cdot d_6 + 3) \bmod 10^9) + 1 = 45$；
- $d_8 = ((1 \cdot d_6 + 2 \cdot d_7 + 3) \bmod 10^9) + 1 = 112$；
- $d_9 = ((1 \cdot d_7 + 2 \cdot d_8 + 3) \bmod 10^9) + 1 = 273$；
- $d_{10} = ((1 \cdot d_8 + 2 \cdot d_9 + 3) \bmod 10^9) + 1 = 662$。

### 【数据范围】

本题采用测试点捆绑，同一行中的测试点作为一个测试点组计分。

对于所有测试数据，保证：

- $3 \leq n \leq 10^7$；
- $1 \leq f \leq 2$；
- 当 $f=1$ 时，$1 \leq d_i \leq 10^9$；
- 当 $f=2$ 时，$2 \leq m \leq \min(n,10^5)$，$0 \leq x,y,z \leq 10^9$，$1 \leq c_i \leq 10^9$。

| 子任务 | 分值 | $n$ | $f$ | 特殊性质 |
| :--: | :--: | :--: | :--: | :--: |
| $1 \sim 3$ | $12$ | $300$ | $1$ | A |
| $4 \sim 7$ | $16$ | $5000$ | $1$ | 无 |
| $8 \sim 9$ | $8$ | $10^5$ | $1$ | B |
| $10 \sim 14$ | $20$ | $10^5$ | $1$ | 无 |
| $15$ | $4$ | $10^7$ | $2$ | C |
| $16 \sim 25$ | $40$ | $10^7$ | $2$ | 无 |

特殊性质 A：$d_i \leq 300$。

特殊性质 B：保证从第一个平台开始跳是最佳选择。

特殊性质 C：保证从第一个平台开始跳是最佳选择。
