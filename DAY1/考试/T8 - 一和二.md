# T8 - 一和二

## 题目描述

你有一个序列 `a1, a2, ..., an`，每个元素是 `1` 或 `2`。

判断是否存在一个整数 `k` 满足以下条件：

`1 ≤ k ≤ n-1`，

`a1 · a2 · ... · ak = a(k+1) · a(k+2) · ... · an`。

如果存在多个满足条件的 `k`，输出最小的那个。若不存在，输出 `-1`。

## 输入格式

每个测试包含多个测试用例。第一行包含测试用例的数量 `t`（`1 ≤ t ≤ 100`）。测试用例说明如下：

每个测试用例的第一行包含一个整数 `n` (`2 ≤ n ≤ 1000`)。

每个测试用例的第二行包含 `n` 个整数 `a1, a2, ..., an` (`1 ≤ ai ≤ 2`)。

## 输出格式

对于每个测试用例，如果没有 `k`，则打印 `-1`。

否则，打印尽可能小的 `k`。

## 输入输出样例

### 输入 #1

```text
3
6
2 2 1 2 1 2
3
1 2 1
4
1 1 1 1
```

### 输出 #1

```text
2
-1
1
```

## 说明 / 提示

对于第一个测试用例，`k=2` 满足条件，因为 `a1 · a2 = a3 · a4 · a5 · a6 = 4`。`k=3` 也满足给定条件，但应打印最小值。

对于第二个测试用例，没有 `k` 满足 `a1 · a2 · ... · ak = a(k+1) · a(k+2) · ... · an` 的条件。

对于第三个测试用例，`k=1`、`2` 和 `3` 满足给定条件，因此答案为 `1`。

对于所有测试点都满足 `1 ≤ t ≤ 100`，`2 ≤ n ≤ 1000`，`1 ≤ ai ≤ 2`。

测试点：

1. `1∼2`：`n≤30`，无特殊性质
2. `3∼4`：`n≤100`，无特殊性质
3. `5∼6`：`n≤1000`，`ai` 必定为 `1`
4. `7∼8`：`n≤1000`，`ai` 必定为 `2`
5. `9∼20`：`n≤1000`，无特殊性质
