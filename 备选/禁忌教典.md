# 交错手稿

## 题目描述

星塔图书馆中保存着一卷特殊手稿。手稿由 $n$ 个符号组成，第 $i$ 个符号的权值为 $a_i$。

这些符号并不是简单相加的。第一个符号产生正向影响，第二个符号产生反向影响，第三个符号再次产生正向影响，第四个符号再次产生反向影响，之后依次交替。

形式化地，对于一个序列 $b_1,b_2,\ldots,b_m$，定义它的交错值为

$$
E(b)=\sum_{i=1}^{m}(-1)^{i+1}b_i.
$$

空序列的交错值定义为 $0$。

现在需要从原序列中恰好删除一段非空连续区间。也就是说，需要选择一个区间 $[l,r]$，删除 $a_l,a_{l+1},\ldots,a_r$。删除后，剩余元素会按照原来的相对顺序连接成一个新的序列

$$
a_1,\ldots,a_{l-1},a_{r+1},\ldots,a_n.
$$

请你计算，有多少种不同的删除区间 $[l,r]$，能使删除后得到的新序列交错值恰好等于 $k$。

## 输入格式

第一行包含两个整数 $n,k$，分别表示序列长度和目标交错值。

第二行包含 $n$ 个正整数 $a_1,a_2,\ldots,a_n$，表示每个符号的权值。

## 输出格式

输出一行一个整数，表示满足要求的删除区间数量。

## 样例

### 样例 1 输入

```text
6 3
4 1 3 2 5 1
```

### 样例 1 输出

```text
2
```

### 样例 1 解释

满足要求的删除区间共有 $2$ 个，分别为 $[2,5]$、$[3,6]$。

## 数据范围与约定

对于所有测试数据，保证：

- $1 \le n \le 2 \times 10^5$；
- $1 \le a_i \le 10^{12}$；
- $|k| \le 10^{18}$。

本题采用测试点捆绑计分，同一子任务内的所有测试点均通过才能获得该子任务分数。

| 子任务 | 测试点编号 | 分值 | $n$ | $a_i$ | 特殊性质 |
| :--: | :--: | :--: | :--: | :--: | :--: |
| $1$ | $1 \sim 2$ | $6$ | $\le 20$ | $\le 10^3$ | 无 |
| $2$ | $3 \sim 6$ | $14$ | $\le 3000$ | $\le 10^{12}$ | 无 |
| $3$ | $7 \sim 8$ | $8$ | $\le 2 \times 10^5$ | $=1$ | 无 |
| $4$ | $9 \sim 12$ | $12$ | $\le 2 \times 10^5$ | $\le 10$ | 无 |
| $5$ | $13 \sim 17$ | $14$ | $\le 2 \times 10^5$ | $\le 10^{12}$ | A |
| $6$ | $18 \sim 21$ | $12$ | $\le 2 \times 10^5$ | $\le 10^{12}$ | B |
| $7$ | $22 \sim 30$ | $34$ | $\le 2 \times 10^5$ | $\le 10^{12}$ | 无 |

记原序列交错值为 $S=E(a_1,a_2,\ldots,a_n)$，记 $M=\sum_{i=1}^{n}a_i$。

特殊性质 A：$|S+k|>2M$ 或 $|S-k|>2M$。

特殊性质 B：在 $P_0,P_1,\ldots,P_n$ 中，不同数值的个数不超过 $30$，其中 $P_i=\sum_{j=1}^{i}(-1)^{j+1}a_j$。
