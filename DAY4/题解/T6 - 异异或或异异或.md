# 异异或或异异或

## 题目描述

- 给定一个数组 $a_1, a_2, \ldots, a_n$。计算满足以下条件的三元组 $(x, y, z)$ 的数量：

  - $1 \leq x \leq y \leq z \leq n$，且
  - $f(x, y) \oplus f(y, z) > f(x, z)$。

  其中，$f(l, r) = a_l \oplus a_{l + 1} \oplus \ldots \oplus a_r$，$\oplus$ 表示按位异或运算

## 输入格式

第一行包含一个整数 $t$（$1 \leq t \leq 10^4$）—— 测试用例的数量。  

每个测试用例的第一行包含一个整数 $n$（$1 \leq n \leq 10^5$）。  

每个测试用例的第二行包含 $n$ 个整数 $a_1, a_2, \ldots, a_n$（$1 \leq a_i \leq 10^9$）。  

保证所有测试用例的 $n$ 之和不超过 $10^5$

## 输出格式

对于每个测试用例，在新行中输出一个整数--描述的三元组数。


### 输入样例1

```
3
3
6 2 4
1
3
5
7 3 7 2 1
```

### 输出样例1

```
4
0
16
```

## 说明/提示

#### 样例说明

在第一个测试用例中，数组 $[6, 2, 4]$ 共有 4 个满足条件的元组：  

- ($1$, $2$, $2$): $(a_1 \oplus a_2) \oplus (a_2) = 4 \oplus 2 > (a_1 \oplus a_2) = 4$  
- ($1$, $1$, $3$): $(a_1) \oplus (a_1 \oplus a_2 \oplus a_3) = 6 \oplus 0 > (a_1 \oplus a_2 \oplus a_3) = 0$  
- ($1$, $2$, $3$): $(a_1 \oplus a_2) \oplus (a_2 \oplus a_3) = 4 \oplus 6 > (a_1 \oplus a_2 \oplus a_3) = 0$  
- ($1$, $3$, $3$): $(a_1 \oplus a_2 \oplus a_3) \oplus (a_3) = 0 \oplus 4 > (a_1 \oplus a_2 \oplus a_3) = 0$  

在第二个测试用例中，没有满足条件的元组。

#### 数据范围

对于所有测试数据，都满足 $1 \le t \le 10^4$，$1 \le n \leq 10^5$，$1 \leq a_i \leq 10^9$，$\displaystyle\sum n \leq 10^5$

|   测试点    |      $n$      |
| :---------: | :-----------: |
|  $1\sim 4$  |  $n \leq 50$  |
| $5\sim 10$  | $n \leq 300$  |
| $11\sim 20$ | $n \leq 10^5$ |

## 题目大意

给定一个数组 $a_1, a_2, \ldots, a_n$。计算满足以下条件的三元组 $(x, y, z)$ 的数量：

- $1 \leq x \leq y \leq z \leq n$，且
- $f(x, y) \oplus f(y, z) > f(x, z)$。

其中，$f(l, r) = a_l \oplus a_{l + 1} \oplus \ldots \oplus a_r$，$\oplus$ 表示按位异或运算

## 思路分析

先给一份模拟暴力代码，枚举 $x,y,z$，之后对于每一个区间暴力计算异或和，这样的时间复杂度是 $O(n^4)$。

```c++
void solve() {
    int n;
    cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; ++i) cin >> a[i];
    long long ans = 0;
    for (int x = 0; x < n; ++x) {
        for (int y = x; y < n; ++y) {
            for (int z = y; z < n; ++z) {
                long long fxy = 0,fyz = 0,fxz = 0;
                for (int i = x; i <= y; ++i) fxy ^= a[i];
                for (int i = y; i <= z; ++i) fyz ^= a[i];
                for (int i = x; i <= z; ++i) fxz ^= a[i];
                if ((fxy ^ fyz) > fxz) {
                    ans++;
                }
            }
        }
    }
    cout << ans << endl;
}
```



如果在暴力的基础上加上异或前缀和优化，只需要枚举 $x,y,z$，复杂度可以达到 $O(n^3)$。

```c++
void solve() {
    int n;
    cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; ++i) {
        cin >> a[i];
    }
    vector<long long> pre(n + 1, 0);
    for (int i = 1; i <= n; ++i) {
        pre[i] = pre[i - 1] ^ a[i - 1];
    }
    long long ans = 0;
    for (int x = 1; x <= n; ++x) {
        for (int y = x; y <= n; ++y) {
            for (int z = y; z <= n; ++z) {
                long long fxy = pre[y] ^ pre[x - 1];
                long long fyz = pre[z] ^ pre[y - 1];
                long long fxz = pre[z] ^ pre[x - 1];
                if ((fxy ^ fyz) > fxz) {
                    ans++;
                }
            }
        }
    }
    cout << ans << endl;
}
```

首先可以想到的第一件事是分析这个式子 $f(x, y) \oplus f(y, z) > f(x, z)$，尝试分析一下式子的左边，其实他这一部分就是把 $[x,y]$ 区间里的所有数字异或和再异或上一个 $y$，所以左边这一部分等价于 $f(x,z) \oplus y$ 那么我们再看这个式子 $f(x,z) \oplus y > f(x,z)$，问题变成了一个数字异或上另一个数字之后是否会变得更大。

接下来思考一个数字异或上另一个数字之后是否会变得更大这个问题，我们考虑在什么样的情况下 $a \oplus b$ 会让 $a$ 变得更大，这里的结论是我们只关心最高位就可以了，因为如果在 $b$ 的最高位的 $1$，其对应的 $a$ 的位如果为 $0$ 那么就一定会在这一位上做出贡献，这个时候低位的情况我们就可以不考虑了，因为就算后面的低几位 $a$ 和 $b$ 全部一样，减少的数值也不会比这一位增加的多，因为 $\displaystyle \sum_{i=0}^{k-1}  2^i < 2^k$ 的成立是显然的。

那么我们遍历所有的数字，我们只需要关注每一个数字最高位 $k$ 就可以了，然后我们需要找到有多少种 $[x,y)$ 和 $[y,z]$ 可以使得异或和的第 $k$ 位是 $0$，那么如果要想然异或的答案是 $0$，要么两边这一位都是 $1$，要么都是 $0$。于是我们统计一个 $cnt_{i,j}$ 代表前 $i$ 个数字的异或和的第 $j$ 位出现了多少个 $1$，这个东西配合上一个异或前缀和是好实现的。

我们在统计答案的时候希望最高位这一位是 $0$，所以我们要找左边右边同时出现相同数字的情况数量，可以根据维护的  $cnt$ 快速求出，并且因为是二进制，总数减去 $1$ 的数量就是 $0$ 的数量是显然的。

```c++
void solve() {  
    int n;
    cin >> n;
    vector<int> vec(n+1);
    for (int i = 1; i <= n ; ++i) {
        cin >> vec[i];
    }
    vector<int> pre(n+1);
    for (int i = 1; i <= n ; ++i) {
        pre[i] = pre[i-1] ^ vec[i];
    }
    vector<vector<int>> cnt(n+1,vector<int>(32));
    for (int i = 1; i <= n ; ++i) {
        for (int j = 0; j <= 32; j++) {
            cnt[i][j] = cnt[i-1][j] + ((pre[i] >> j) & 1);
        }
    }
    long long ans = 0;
    for (int i = 1; i <= n ; ++i) {
        int high = __lg(vec[i]);
        ans += 1LL * cnt[i - 1][high] * (cnt[n][high] - cnt[i - 1][high]);
        ans += 1LL * (i - cnt[i - 1][high]) * (n - i + 1 - (cnt[n][high] - cnt[i - 1][high]));
    }
    cout << ans << endl;
}
```
