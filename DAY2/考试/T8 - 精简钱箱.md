# T2- 精简钱箱

## 题目描述

旧集市的钱箱里放着若干种面额的货币。第 $i$ 种货币的面额为 $a_i$，并且每种货币都有无限多张。

为了描述一个货币系统，我们记它为 $(n, a)$，其中 $n$ 表示货币种数，$a_1, a_2, \ldots, a_n$ 表示这些货币的面额。

若一个非负整数金额 $x$ 可以写成

$$
x = \sum_{i=1}^{n} a_i t_i,
$$

其中 $t_1, t_2, \ldots, t_n$ 都是非负整数，则称金额 $x$ 可以被货币系统 $(n, a)$ 表示。

两个货币系统 $(n, a)$ 与 $(m, b)$ 等价，当且仅当对于任意非负整数金额 $x$，它在两个系统中的可表示性完全相同：要么都能表示，要么都不能表示。

仓库管理员小码君想删去一些“多余”的面额，使新的货币系统仍然与原系统等价。请你求出，在所有等价的货币系统中，最少需要保留多少种货币。

## 输入格式

第一行包含一个整数 $T$，表示测试数据组数。

对于每组测试数据：

第一行包含一个整数 $n$，表示原货币系统中的货币种数。

第二行包含 $n$ 个整数 $a_1, a_2, \ldots, a_n$，表示每种货币的面额。

## 输出格式

对于每组测试数据，输出一行一个整数，表示与原货币系统等价时所需的最少货币种数。

## 样例

### 样例 1 输入

```text
2
4
3 19 10 6
5
11 29 13 19 17
```

### 样例 1 输出

```text
2
5
```

## 提示

### 【样例 1 解释】

对于第一组测试数据，只保留面额 $3$ 和 $10$ 就可以得到一个与原系统等价的货币系统，因此答案为 $2$。

对于第二组测试数据，没有任何一种面额可以删去，因此答案为 $5$。

### 【数据范围】

对于所有测试数据，保证：

- $1 \leq T \leq 20$；
- $1 \leq n \leq 100$；
- $1 \leq a_i \leq 25000$；
- 同一组测试数据中，$a_i$ 两两不同。

| 测试点编号 | $n \leq$ | $a_i \leq$ | 特殊性质 |
| :--: | :--: | :--: | :--: |
| $1 \sim 2$ | $15$ | $100$ | 无 |
| $3 \sim 4$ | $15$ | $300$ | 无 |
| $5 \sim 7$ | $50$ | $300$ | 无 |
| $8 \sim 10$ | $50$ | $25000$ | 无 |
| $11 \sim 20$ | $100$ | $25000$ | 无 |
