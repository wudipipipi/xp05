# T118304.一张特别大的图

## 题目描述

给定一棵 `n`（`2≤n≤10^5`）个节点的无根树，对于每个 `k`（`1≤k≤n`），定义一个无向图 `Gk`，当点对 `(u, v)` 在树中距离大于等于 `k` 的时候，`Gk` 中的 `u, v` 之间有一条无向边，请问当 `k=1, 2, 3, ..., n` 的时候，`Gk` 的连通块个数。

## 输入格式

第一行包含整数 `n`。

接下来 `n-1` 行，每行包含两个整数 `u` 和 `v`，表示树上的一条无向边。

## 输出格式

输出共 `n` 行，每行一个整数，表示 `Gk` 的连通块个数。

## 输入输出样例

### 输入 #1

```text
6
1 2
1 3
2 4
2 5
3 6
```

### 输出 #1

```text
1 1 2 4 6 6
```

### 输入 #2

```text
5
1 2
2 3
3 4
3 5
```

### 输出 #2

```text
1 1 3 5 5
```

## 说明 / 提示

### 样例 1 解释

当 `k=1` 时，所有的点都连通；

当 `k=4` 时，只有 `(4,6)` 和 `(5,6)` 两条边连通，所以连通块个数为 `4`；

### 样例 2 解释

当 `k=1` 时，所有的点都连通；

当 `k=2` 时，所有的点都与其距离大于等于 `2` 的所有点连边，形成的图依然连通；

当 `k=3` 时，节点 `{1,4,5}` 构成一个连通块，其余两个节点分别为一个连通块；

当 `k=4/5` 时，树上没有任何一对点之间的距离 `≥4`，因此图没有任何边，此时联通块个数等于点数。

### 数据范围

对于所有测试点都满足 `2≤n≤10^5`。

测试点：

1. `1∼4`：`2≤n≤1000`
2. `5∼6`：`2≤n≤10^5`，所有点的度数 `≤2`
3. `7∼8`：`2≤n≤10^5`，数据随机生成
4. `9∼10`：`2≤n≤10^5`，存在度数为 `n−1` 的结点
5. `11∼20`：`2≤n≤10^5`
