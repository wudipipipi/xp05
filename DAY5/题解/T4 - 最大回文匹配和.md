# 回文数组

## 题目大意

给定两个数组 $A$ 和 $B$。我们可以分别从两个数组中选择一个长度相同的连续子数组 $A'$ 和 $B'$，然后把它们对应位置相加，得到数组 $C$。

现在要求最大的长度 $k$，使得存在长度为 $k$ 的 $A'$ 和 $B'$，让得到的 $C$ 是回文数组。

## 思路分析

首先考虑暴力做法。我们可以枚举长度 $k$，再枚举 $A$ 中的起点和 $B$ 中的起点，直接构造出 $C$ 并判断是否回文。这样复杂度大约是 $O(nm\min(n,m))$ 级别，只能处理很小的数据，可以拿到子任务 $1$ 的 $13$ 分。

如果 $B$ 中所有数都相等，那么给 $A'$ 每个位置加上相同的数不会改变它是否回文，所以问题就变成在 $A$ 中找一个长度不超过 $m$ 的最长回文子数组，可以通过哈希或 Manacher 一类方法解决，可以拿到子任务 $2$ 的 $33$ 分。

下面考虑一般情况。

假设我们选择了长度为 $len$ 的两个子数组：

$$
A'=[a_1,a_2,\dots,a_{len}]
$$

$$
B'=[b_1,b_2,\dots,b_{len}]
$$

它们相加之后得到：

$$
C=[a_1+b_1,a_2+b_2,\dots,a_{len}+b_{len}]
$$

要求 $C$ 是回文，也就是对于每个位置 $i$ 都有：

$$
a_i+b_i=a_{len-i+1}+b_{len-i+1}
$$

移项可以得到：

$$
a_i-a_{len-i+1}=b_{len-i+1}-b_i
$$

也就是说：

$$
A'-reverse(A')=reverse(B')-B'
$$

这就是本题最关键的转化。对于固定长度 $len$，我们只需要判断是否存在一个 $A$ 的子数组和一个 $B$ 的子数组，使得上面两个“差值数组”相等。

这个判断可以用哈希完成：

- 对 $A$ 的所有长度为 $len$ 的子数组，计算 `正向哈希 - 反向哈希`，放入集合；
- 对 $B$ 的所有长度为 $len$ 的子数组，计算 `反向哈希 - 正向哈希`，如果集合中出现过，就说明存在合法方案。

接下来还需要找最大长度。注意如果存在一个长度为 $len$ 的回文数组，那么去掉首尾之后，长度为 $len-2$ 的部分仍然是回文。所以对于奇偶性相同的长度，合法性具有单调性。

因此我们分别对奇数长度和偶数长度二分答案即可。

下面是正解：

```c++
#include <bits/stdc++.h>
using namespace std;

const int MOD1 = 1000000007;
const int MOD2 = 1000000009;
const int BASE = 911382323;

struct Hash {
    int x, y;

    bool operator==(const Hash &other) const {
        return x == other.x && y == other.y;
    }
};

struct PairHash {
    size_t operator()(const Hash &h) const {
        return (1ULL * h.x << 32) ^ h.y;
    }
};

Hash add(Hash a, Hash b) {
    int x = a.x + b.x;
    int y = a.y + b.y;
    if (x >= MOD1) x -= MOD1;
    if (y >= MOD2) y -= MOD2;
    return {x, y};
}

Hash sub(Hash a, Hash b) {
    int x = a.x - b.x;
    int y = a.y - b.y;
    if (x < 0) x += MOD1;
    if (y < 0) y += MOD2;
    return {x, y};
}

Hash mul(Hash a, Hash b) {
    return {
        int(1LL * a.x * b.x % MOD1),
        int(1LL * a.y * b.y % MOD2)
    };
}

struct Hasher {
    int n;
    vector<Hash> pre, suf;

    Hasher() {}

    Hasher(const vector<int> &a, const vector<Hash> &pw) {
        n = (int)a.size() - 1;
        pre.assign(n + 2, {0, 0});
        suf.assign(n + 3, {0, 0});

        for (int i = 1; i <= n; i++) {
            pre[i] = add(mul(pre[i - 1], {BASE, BASE}), {a[i], a[i]});
        }
        for (int i = n; i >= 1; i--) {
            suf[i] = add(mul(suf[i + 1], {BASE, BASE}), {a[i], a[i]});
        }
    }

    Hash get_forward(int l, int r, const vector<Hash> &pw) const {
        return sub(pre[r], mul(pre[l - 1], pw[r - l + 1]));
    }

    Hash get_reverse(int l, int r, const vector<Hash> &pw) const {
        return sub(suf[l], mul(suf[r + 1], pw[r - l + 1]));
    }
};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;
    cin >> n >> m;

    vector<int> a(n + 1), b(m + 1);
    for (int i = 1; i <= n; i++) cin >> a[i];
    for (int i = 1; i <= m; i++) cin >> b[i];

    int lim = min(n, m);
    vector<Hash> pw(lim + 1);
    pw[0] = {1, 1};
    for (int i = 1; i <= lim; i++) {
        pw[i] = mul(pw[i - 1], {BASE, BASE});
    }

    Hasher ha(a, pw), hb(b, pw);

    auto exists = [&](int len) {
        unordered_set<Hash, PairHash> s;
        s.reserve(n + m);

        for (int l = 1; l + len - 1 <= n; l++) {
            int r = l + len - 1;
            Hash forward = ha.get_forward(l, r, pw);
            Hash reverse = ha.get_reverse(l, r, pw);
            s.insert(sub(forward, reverse));
        }

        for (int l = 1; l + len - 1 <= m; l++) {
            int r = l + len - 1;
            Hash forward = hb.get_forward(l, r, pw);
            Hash reverse = hb.get_reverse(l, r, pw);

            if (s.count(sub(reverse, forward))) {
                return true;
            }
        }

        return false;
    };

    int ans = 0;

    for (int parity = 0; parity <= 1; parity++) {
        int l = 0, r = (lim - parity) / 2;
        int best = -1;

        while (l <= r) {
            int mid = (l + r) / 2;
            int len = mid * 2 + parity;

            if (len == 0 || exists(len)) {
                best = mid;
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }

        if (best != -1) {
            ans = max(ans, best * 2 + parity);
        }
    }

    cout << ans << '\n';

    return 0;
}
```

对于每个固定长度，判断一次需要枚举两个数组的所有子数组，复杂度为 $O(n+m)$。由于奇偶长度分别二分，总时间复杂度为：

$$
O((n+m)\log \min(n,m))
$$

哈希数组和集合的空间复杂度为 $O(n+m)$。使用双哈希可以让冲突概率非常低。
