# 射箭比赛

## 题目描述

诺丁汉警长组织了一场射箭比赛，罗宾汉将与警长进行最终对决。

有 $n$ 个目标排成一排，编号为 $1 \sim n$。当一名玩家射击目标 $i$ 时，该玩家的得分增加 $a_i$，且目标 $i$ 会被摧毁。游戏由若干回合组成，玩家轮流进行，罗宾汉总是先手，随后是警长，依此类推。游戏会一直进行到所有目标都被摧毁为止。两名玩家初始得分均为 $0$。

游戏结束时，得分更高的玩家获胜；如果双方得分相同，则为平局。

在每一回合中，玩家都可以射击任意一个尚未被射击过的目标。双方都会采用最优策略，使自己的最终得分尽可能高。

警长担心自己会输掉比赛。为了帮助他，你需要回答 $q$ 个询问。每个询问给出两个整数 $l,r$，表示本次比赛只使用目标 $l,l+1,\dots,r$，其余目标会在比赛开始前被移除。

对于每个询问，判断在只考虑目标 $l,l+1,\dots,r$ 时，警长是否不会输掉比赛。

## 输入格式

输入的第一行包含一个整数 $t$，表示测试用例的数量。

对于每个测试用例，输入的第一行包含两个整数 $n,q$，分别表示目标数量和询问数量。

接下来一行包含 $n$ 个整数 $a_1,a_2,\dots,a_n$，表示每个目标的得分。

接下来 $q$ 行，每行包含两个整数 $l,r$，表示一个询问。

## 输出格式

对于每个询问，如果仅考虑目标 $l,l+1,\dots,r$ 时警长不会输掉比赛，则输出 `YES`，否则输出 `NO`。

## 样例

### 样例 1 输入

```text
2
3 3
1 2 2
1 2
1 3
2 3
5 3
2 1 2 1 1
1 2
1 3
4 5
```

### 样例 1 输出

```text
NO
NO
YES
NO
NO
YES
```

## 提示

### 【样例 1 解释】

在第一个测试用例中：

- 对于询问 $[1,2]$，罗宾汉先手，必然可以拿到得分为 $2$ 的目标，警长只能拿到得分为 $1$ 的目标，因此警长会输，输出 `NO`；
- 对于询问 $[1,3]$，罗宾汉先手后，警长最多只能追平一部分收益，但罗宾汉最终仍能取得更高总分，因此警长会输，输出 `NO`；
- 对于询问 $[2,3]$，两名玩家分别取得一个得分为 $2$ 的目标，最终平局，警长不会输，输出 `YES`。

在第二个测试用例中：

- 对于询问 $[1,2]$，罗宾汉先手必然拿到得分为 $2$ 的目标，警长只能拿到得分为 $1$ 的目标，因此警长会输，输出 `NO`；
- 对于询问 $[1,3]$，罗宾汉和警长各会优先选择得分较高的目标，但罗宾汉先手仍能获得更高总分，因此警长会输，输出 `NO`；
- 对于询问 $[4,5]$，两名玩家各得到一个得分为 $1$ 的目标，最终平局，警长不会输，输出 `YES`。

### 【数据范围】

对于所有测试数据，保证：

- $1 \leq t \leq 10^4$；
- $1 \leq n,q \leq 2 \times 10^5$；
- $1 \leq a_i \leq 10^6$；
- $1 \leq l \leq r \leq n$；
- 所有测试用例中 $n$ 和 $q$ 的总和不超过 $2 \times 10^5$。

| 测试点编号 | $n$ | $a_i$ | 特殊性质 |
| :--: | :--: | :--: | :--: |
| $1 \sim 2$ | $\sum n \leq 3000$ | $1 \leq a_i \leq 2$ | 无 |
| $3 \sim 6$ | $1 \leq n \leq 2 \times 10^5$ | $a_i$ 为 $2$ 的幂 | 无 |
| $7 \sim 10$ | $1 \leq n \leq 2 \times 10^5$ | $1 \leq a_i \leq 10^6$ | 无 |
