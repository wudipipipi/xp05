# 擦黑板

## 题目描述

现在在黑板上有数字 $1、2、3$ 若干。在一次运算中，可以进行如下操作：

选择两个不同的数字，将其从黑板上擦除，并且写上一个与另外两个数字不同的数字。例如一开始的数字有 $1、1、1、2、2、3$，选择数字 $1$ 和 $3$，并且擦掉，再写上一个 $2$，黑板上的数字就会变成 $1、1、2、3、2$。

问题是经过任意次操作之后，黑板上能否只剩下一种数字，如果可以的话，可以是哪种数字？

## 输入格式

**输入**

每个测试包含多个测试用例。第一行包含测试用例的数量 $t$ ( $1 \le t \le 10^5$ )。测试用例说明如下。

每个测试用例的第一行也是唯一一行，包含三个整数 $a$ , $b$ , $c$ ( $1 \le a, b, c \le 100$ ) - 分别是 1 的个数、2 的个数和 3 的个数。

## 输出格式

**输出**

针对每个测试用例，输出一行，需要包含三个数字。

如果经过某些操作后，黑板上可能只剩下数字 $1$ ，则第一行应为 $1$ ，否则为 $0$ 。

同样，如果经过某些运算后，黑板上可能只剩下数字 $2$ ，则第二个数字应为 $1$ ，否则应为 $0$ 。

同样，如果经过某些运算后，黑板上可能只剩下数字 $3$ ，则第三个数字应为 $1$ ，否则应为 $0$ 。


### 输入样例1

```
3
1 1 1
2 3 2
82 47 59
```

### 输出样例1

```
1 1 1
0 1 0
1 0 0
```

## 说明/提示

#### 【样例解释】

在第一个测试案例中，可以删除数字 $2$ 和 $3$ 并写入数字 $1$ 。之后，黑板上将有 $2$ 个数字 $1$ 。她可以通过类似的操作使棋盘只剩下数字 $2$ 或 $3$ 。

在第二个测试案例中，可以删除数字 $1$ 和 $3$ 并写入数字 $2$ 。在进行了 $2$ 次操作后，黑板上将只剩下数字 $2$ 。可以证明，不可能只剩下数字 $1$ 或只剩下数字 $3$ 。

在第三个测试案例中，有一个运算序列使得黑板上只剩下数字 $1$ 。可以证明不可能只剩下数字 $2$ 或数字 $3$ 。

#### 数据范围

对于所有测试数据，都满足$1 \leq t \leq 10^5$，$0 \leq a,b,c \leq  100$

|   测试点    |         $t$          |                       $a,b,c$                        |
| :---------: | :------------------: | :--------------------------------------------------: |
|  $1\sim 4$  | $1 \leq t \leq 10^5$ |     $1\leq a,b,c \leq  100$，$a_i$ 一定全是奇数      |
|  $5\sim 8$  | $1 \leq t \leq 10^5$ |     $1 \leq a,b,c \leq  100$，$a_i$ 一定全是偶数     |
| $9\sim 12$  | $1 \leq t \leq 10^5$ | $0 \leq a,b,c \leq  100$，$a_i$ 一定至少有一个为 $0$ |
| $13\sim 20$ | $1 \leq t \leq 10^5$ |               $1 \leq a,b,c \leq  100$               |

## 题目大意

现在在黑板上有数字 $1、2、3$ 若干。在一次运算中，可以进行如下操作：

选择两个不同的数字，将其从黑板上擦除，并且写上一个与另外两个数字不同的数字。

问题是经过任意次操作之后，黑板上能否只剩下一种数字，如果可以的话，可以是哪种数字？

## 思路分析

这个题前两个特殊性质是在引导学生思考奇偶性的问题，第三个特殊性质引导学生思考较为简单的一类情况并且尝试推广。实际上并没有什么可行的暴力解。

对于前八个测试点，只要输出三个 $1$ 就可以全部通过。

下面是正解：

首先我们进行一次操作之后，其中两个数字的数量减一，另一个数字的数量加一，对于数字的个数来说，其每一次操作都会改变数字个数的奇偶性。

正因为奇偶性每次操作都会改变，因此如果两个数字数量的奇偶性一开始就不相同，那么不可能通过任何操作使得其奇偶性一致。

因为我们最后要做的是让两种数字个数为 $0$，也就是两种数字奇偶性相同。如果两个数字个数的奇偶性不同，我们一定无法构造其同时为 $1$ 的情况。

```c++
#include <bits/stdc++.h>
using namespace std;

int main(){
    int T, a, b, c;
    cin >> T;
    while (T--){
        cin >> a >> b >> c;
        cout << (b % 2 == c % 2) << ' ' << (a % 2 == c % 2) << ' ' << (a % 2 == b % 2) << endl;
    }
    return 0;
}
```
