# 涂树

## 题目描述

Welcome24ever 有一个 $n$ 个结点的树，最初所有结点都是白色的。

现在在树上有两个棋子，分别叫做 $P_A$ 和 $P_B$，一开始分别位于点 $a$ 和 点 $b$ 上。对于每一步来说，Welcome24ever 都需要按照顺序执行下列操作：

- 将 $P_A$ 移动到相邻一个相邻点，如果移动后的点是白色，那么就将其涂成红色。
- 将 $P_B$ 移动到相邻一个相邻点，如果移动后的点是红色，那么就将其涂成蓝色。

最开始，点 $a$ 会被涂成红色，如果 $a=b$  那么点 $a$ 就会变成蓝色。注意，每一步都必须移动两个棋子，并且两个棋子可以位于同一个顶点。

Welcome24ever 很想知道把所有顶点涂成蓝色所需要的操作轮数（$P_A$ 和 $P_B$ 都移动一次算一轮操作），请你帮帮他！

## 输入格式

第一行包含一个整数 $t$（$1 \le t \le 10^4$），表示测试用例的数量。接下来是 $t$ 个测试用例。

每个测试用例的第一行包含一个整数 $n$（$1\leq n\leq 2 \cdot 10^5$）。

每个测试用例的第二行包含两个整数 $a$ 和 $b$（$1 \le a,b \le n$）。

接下来的 $n-1$ 行，每一行包含两个整数 $x_i,y_i$（1\leq x_i,y_i \leq n），表示点 $x_i$ 和 $y_i$ 之间存在一条边，保证数据中这些边构成一条链。

保证所有测试用例的 $n$ 之和不超过 $2 \cdot 10^5$。

## 输出格式

对于每个测试用例，输出将所有顶点涂成蓝色所需的最少步骤数。


### 输入样例1

```
3
2
1 2
1 2
5
1 2
1 2
1 3
1 4
1 5
8
5 4
7 1
1 5
1 8
8 3
7 2
8 6
3 4
```

### 输出样例1

```
2
8
13
```

## 说明/提示

#### 数据范围

在第一个测试用例中，Welcome24ever 能够把按照下列操作顺序把所有点都 涂成蓝色：

- 最初，$P_A$ 位于点 $1$，并且 $P_B$ 位于点 $2$，点 $1$ 被涂成红色，点 $2$ 是白色。
- Welcome24ever 把 $P_A$ 移动到点 $2$ 并且将其涂为红色。之后 Welcome24ever 把 $P_B$ 移动到点 $1$ 并将其涂成蓝色。
- Welcome24ever 把 $P_A$ 移动到点 $1$。之后 Welcome24ever 把 $P_B$ 移动到点 $2$ 并将其涂成蓝色。

#### 数据范围

## 题目大意

Welcome24ever 有一个 $n$ 个结点的树，最初所有结点都是白色的。

现在在树上有两个棋子，分别叫做 $P_A$ 和 $P_B$，一开始分别位于点 $a$ 和 点 $b$ 上。对于每一步来说，Welcome24ever 都需要按照顺序执行下列操作：

- 将 $P_B$ 移动到相邻一个相邻点，如果移动后的点是红色，那么就将其涂成蓝色。

## 思路分析

对于一条链来说，这个只是引导学生来思考一条链上是如何移动的，以此推到正解的第一部分，如果学生能够分讨出正确情况，结合 $a=b$ 特殊性质的提示分析完就应该能够写出正解。

特殊性质 $a=b$，我们放到最后说。

根据题意，$B$ 在外面自己走肯定是对答案没有贡献的，所以我们考虑先让 $A$ 与 $B$ 移动到一起，之后 $B$ 跟随 $A$ 进行移动，那么第一步就是要想到让 $A$ 和 $B$ 在一起需要的操作数，这个东西是好做的，我们只要以 $A$ 为根搜索到 $B$，记录其深度为 $dep$，那么在深度为 $dep/2$ 的时候二者就移动到一起了。

移动到一起之后显然 $B$ 只需要跟着 $A$ 移动就可以了。问题在这个时候转化为了，从一个点出发，路径覆盖所有点需要的最小步。这个东西也是平凡的，不妨先考虑更简单的问题，如果要求我们路径的起点终点重合怎么办，这个时候最优的方案对于所有边都经过两次，因此答案显然为 $2\times (n-1)$。那如果我们不要求回到起点，我们可以把距离最远的那个点最后抵达。

因此结论是答案分为两部分，第一部分是 $dep/2$，另一部分则是 $A$ 自己走遍所有点的贡献，显然是 $2\times (n-1)$ 再减去以 $A$ 所在位置为根最大节点深度。

```c++
int n,a,b,t,root,deep;
vector<int> adj[N];
void find(int cur, int fa, int dep) {
    if (cur == b) {
        t = dep;
        return;
    }
    for (auto &nxt: adj[cur]) {
        if (nxt == fa) continue;
        find(nxt, cur, dep + 1);
        if (t != 0) {
            if (dep == t / 2) {
                root = cur;
            }
            return;
        }
    }
}
void dfs(int cur, int fa, int dep) {
    for (auto &nxt: adj[cur]) {
        if (nxt == fa) continue;
        dfs( nxt, cur, dep + 1);
    }
    deep = max(deep, dep);
}
void solve() {
    cin >> n >> a >> b;
    t = root = deep = 0;
    for (int i = 1; i <= n; ++i) {
        adj[i].clear();
    }
    for (int i = 1; i < n; ++i) {
        int u, v;
        cin >> u >> v;
        adj[u].push_back(v), adj[v].push_back(u);
    }
    root = a, t = 0;
    find(a, 0, 0);
    dfs(root, 0, 0);
    cout << (t + 1) / 2 + (n - 1) * 2 - deep << endl;
}
```

发现所谓的特殊性质 $a=b$ 实际上是我们求解正常问题的第二部分情况，因此只需要计算 $2\times (n-1)$ 再减去以 $A$ 所在位置为根最大节点深度就可以了。

```c++
int n,a,b,root,deep;
vector<int> adj[N];
void dfs(int cur, int fa, int dep) {
    for (auto &nxt: adj[cur]) {
        if (nxt == fa) continue;
        dfs( nxt, cur, dep + 1);
    }
    deep = max(deep, dep);
}
void solve() {
    cin >> n >> a >> b;
    t = root = deep = 0;
    for (int i = 1; i <= n; ++i) {
        adj[i].clear();
    }
    for (int i = 1; i < n; ++i) {
        int u, v;
        cin >> u >> v;
        adj[u].push_back(v), adj[v].push_back(u);
    }
    dfs(a, 0, 0);
    cout << (n - 1) * 2 - deep << endl;
}
```
