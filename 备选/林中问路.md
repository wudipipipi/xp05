# 林中问路

## 题目描述

有一棵 $n$ 个点的树，点从 $1$ 到 $n$ 编号。第 $i$ 个点上写着一个非负整数 $a_i$。

现在有 $q$ 次询问。每次询问给出两个不同的点 $x,y$，请考虑从 $x$ 到 $y$ 的简单路径。设这条路径上的点依次为

$$
v_1,v_2,\ldots,v_k,
$$

其中 $v_1=x,\ v_k=y$。

对于路径上的一个非空连续段

$$
v_l,v_{l+1},\ldots,v_r,
$$

如果它满足

$$
a_{v_l}\oplus a_{v_{l+1}}\oplus\cdots\oplus a_{v_r}
\ge
a_{v_l}+a_{v_{l+1}}+\cdots+a_{v_r},
$$

则称它是一段“不进位的路”。其中 $\oplus$ 表示按位异或。

请你对每次询问，输出从 $x$ 到 $y$ 的路径中有多少个非空连续段是不进位的。

## 输入格式

本题有多组测试数据。

第一行包含一个整数 $T$，表示测试数据组数。

对于每组测试数据：

第一行包含两个整数 $n,q$，分别表示树的点数和询问次数。

第二行包含 $n$ 个整数 $a_1,a_2,\ldots,a_n$。

接下来 $n-1$ 行，每行包含两个整数 $u,v$，表示树上的一条边。

接下来 $q$ 行，每行包含两个整数 $x,y$，表示一次询问。

## 输出格式

对于每次询问，输出一行一个整数，表示对应路径中“不进位的路”的数量。

## 样例

### 样例输入

```text
2
5 4
0 1 2 3 4
1 2
1 3
3 4
3 5
2 4
4 5
1 5
2 5
4 2
0 0 0 0
1 2
2 3
3 4
1 4
2 3
```

### 样例输出

```text
7
4
6
10
10
3
```

## 数据范围与约定

对于所有测试数据，保证：

- $1\le T\le 10^4$；
- $2\le n\le 10^5$；
- $1\le q\le 10^5$；
- $0\le a_i<2^{20}$；
- $1\le x,y,u,v\le n$；
- 每次询问满足 $x\ne y$；
- 输入的边构成一棵树；
- 所有测试数据的 $n$ 之和不超过 $10^5$，$q$ 之和不超过 $10^5$。

本题采用测试点捆绑计分，同一子任务内的所有测试点均通过才能获得该子任务分数。

| 子任务 | 测试点编号 | 分值 | $n,q$ | 特殊性质 |
| :--: | :--: | :--: | :--: | :--: |
| $1$ | $1 \sim 2$ | $6$ | $\le 200$ | 无 |
| $2$ | $3 \sim 4$ | $8$ | $\le 10^5$ | A |
| $3$ | $5 \sim 8$ | $14$ | $\le 10^5$ | B |
| $4$ | $9 \sim 13$ | $16$ | $n\le 5000$ | 无 |
| $5$ | $14 \sim 20$ | $20$ | $\le 10^5$ | C |
| $6$ | $21 \sim 30$ | $36$ | $\le 10^5$ | 无 |

特殊性质 A：所有 $a_i=0$。

特殊性质 B：每次询问对应的路径上点数不超过 $500$。

特殊性质 C：树是一条链，即存在一种编号顺序使得边集为 $(1,2),(2,3),\ldots,(n-1,n)$。
