# 训猫

输入文件：cat.in

输出文件：cat.out

## 题目描述

有 $n$ 个猫爬架，编号为 $1 \sim n$。第 $i$ 个猫爬架的高度为 $p_i$。所有猫爬架的高度均为 $1$ 到 $n$ 之间的整数，且两两不同。

共有 $n-1$ 对猫爬架相邻，猫猫可以从一个猫爬架飞跃到另一个与它相邻的猫爬架。保证从任意一个猫爬架出发，经过若干次飞跃，都可以到达任意一个猫爬架。

最初，猫猫位于高度为 $n$ 的猫爬架上。

接下来，你需要不断进行如下操作：

1. 选择一个没有障碍物的猫爬架，并在上面放置一个障碍物；
2. 如果猫猫不在被选中的猫爬架上，则无事发生；
3. 如果猫猫在被选中的猫爬架上：
   - 若所有相邻的猫爬架都有障碍物，则猫猫无法继续移动，训练结束；
   - 否则，猫猫会移动到所有与当前猫爬架可达的猫爬架中，除当前猫爬架外高度最大的猫爬架，并沿一条最短路径飞跃过去。

这里，在当前障碍物状态下，若猫猫可以从猫爬架 $x$ 出发，在不经过有障碍物的猫爬架的前提下，通过若干次飞跃到达猫爬架 $y$，则称猫爬架 $x$ 与猫爬架 $y$ 可达。

请你计算，在最优放置障碍物的情况下，整个训练过程中猫猫飞跃次数之和的最大值。

## 输入格式

输入的第一行包含一个整数 $n$。

输入的第二行包含 $n$ 个整数 $p_1,p_2,\dots,p_n$。

接下来 $n-1$ 行，每行包含两个整数 $u_j$ 和 $v_j$，表示一对相邻的猫爬架。

## 输出格式

输出一行一个整数，表示整个训练过程中猫猫飞跃次数之和的最大值。

## 样例

### 样例 1 输入

```text
4
3 4 1 2
1 2
2 3
3 4
```

### 样例 1 输出

```text
3
```

### 样例 2 输入

```text
7
3 2 7 1 5 4 6
1 2
1 3
2 4
2 5
3 6
3 7
```

### 样例 2 输出

```text
7
```

## 提示

### 【样例 1 解释】

由于猫爬架 $2$ 的高度为 $4$，因此猫猫一开始位于猫爬架 $2$。

一种最优的操作方式如下：

- 在猫爬架 $1$ 上放置障碍物，猫猫不移动；
- 在猫爬架 $2$ 上放置障碍物，此时与当前猫爬架可达且高度最高的猫爬架为 $4$，猫猫沿路径 $2 \to 3 \to 4$ 飞跃，共飞跃 $2$ 次；
- 在猫爬架 $4$ 上放置障碍物，猫猫沿路径 $4 \to 3$ 飞跃，共飞跃 $1$ 次；
- 在猫爬架 $3$ 上放置障碍物，猫猫无法继续移动，训练结束。

猫猫总共飞跃 $3$ 次。可以证明，不存在使飞跃次数之和大于 $3$ 的操作方式。

此样例满足子任务 $1,2,3,4,5,7$ 的限制。

### 【样例 2 解释】

此样例满足子任务 $4,6,7$ 的限制。

### 【数据范围】

本题采用测试点捆绑，同一行中的测试点作为一个测试点组计分。

对于所有测试数据，保证：

- $2 \leq n \leq 2 \times 10^5$；
- $1 \leq p_i \leq n$；
- 对于所有 $1 \leq i < j \leq n$，均有 $p_i \neq p_j$；
- $1 \leq u_j < v_j \leq n$；
- 给定的相邻关系构成一棵树；
- 输入的所有数均为整数。

| 子任务 | 分值 | $n$ | 特殊性质 |
| :--: | :--: | :--: | :--: |
| $1 \sim 4$ | $8$ | $16$ | A |
| $5 \sim 8$ | $8$ | $300$ | A |
| $9 \sim 12$ | $8$ | $5000$ | A |
| $13 \sim 17$ | $10$ | $5000$ | 无 |
| $18 \sim 27$ | $20$ | $2 \times 10^5$ | A |
| $28 \sim 37$ | $20$ | $2 \times 10^5$ | B |
| $38 \sim 50$ | $26$ | $2 \times 10^5$ | 无 |

特殊性质 A：对于所有 $1 \leq j \leq n-1$，均有 $u_j=j$ 且 $v_j=j+1$。

特殊性质 B：对于所有 $1 \leq j \leq n-1$，均有 $u_j=\left\lfloor\dfrac{j+1}{2}\right\rfloor$ 且 $v_j=j+1$。
