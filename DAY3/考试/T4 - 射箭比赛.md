# T118322.射箭比赛

## 题目描述

在这样的日子里，射箭总是当天的主要运动，因为诺丁汉郡的自耕农是全英格兰射长弓最厉害的人，但今年警长却犹豫了……

诺丁汉警长组织了一场射箭比赛。这是决赛，罗宾汉要和警长对决！

有 `n` 个目标排成一排，从 `1` 到 `n` 编号。当一名玩家射击目标 `i` 时，其得分增加 `ai` 并且目标 `i` 被摧毁。游戏由回合组成，玩家轮流进行。罗宾汉总是先开始游戏，然后是警长，依此类推。游戏一直持续到所有目标都被摧毁为止。两名玩家初始得分均为 `0`。

游戏结束时，得分最高的玩家获胜，另一方则失败。若双方得分相同，则为平局，无人获胜或失败。每一轮，玩家都可以射击之前未被射击过的任何目标。双方都以最佳策略行事，以争取尽可能高的得分。

诺丁汉警长怀疑自己可能会输掉比赛！这绝不能发生，你必须帮助警长。警长会提出 `q` 个问题，每个问题都指定 `l` 和 `r`。这意味着比赛只会用到目标 `l, l+1, ..., r`，因为警长会在比赛开始前移除其他目标。

对于每个查询 `l, r`，确定仅考虑目标 `l, l+1, ..., r` 时，警长是否不会输掉游戏。

## 输入格式

输入的第一行包含一个整数 `t` (`1≤t≤10^4`)——测试用例的数量。

每个测试用例的第一行包含两个整数 `n` 和 `q` (`1≤n,q≤2⋅10^5`) ，分别表示目标的数量以及警长将提出的询问数量。

每个测试用例的第二行包含 `n` 个整数 `a1, a2, ..., an` (`1≤ai≤10^6`)——击中每个目标的得分。

接下来有 `q` 行，每行包含两个整数 `l` 和 `r` (`1≤l≤r≤n`)——表示每个查询所考虑的目标范围。

可以保证所有测试用例中 `n` 和 `q` 的总和不超过 `2⋅10^5`。

## 输出格式

对于每个查询，如果仅考虑目标 `l, l+1, ..., r` 时警长不会输掉游戏，则输出 `"YES"`，否则输出 `"NO"`。

## 输入输出样例

### 输入 #1

```text
2
3 3
1 2 2
1 2
1 3
2 3
5 3
2 1 2 1 1
1 2
1 3
4 5
```

### 输出 #1

```text
NO
NO
YES
NO
NO
YES
```

## 说明 / 提示

测试点：

1. `1∼2`：`∑n≤3000`，`1≤ai≤2`
2. `3∼6`：`1≤n≤200000`，`ai` 是 `2` 的幂
3. `7∼10`：`1≤n≤200000`，`1≤ai≤10^6`
