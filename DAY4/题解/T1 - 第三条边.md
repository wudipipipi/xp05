# 第三条边

## 题目描述

粉红士兵给了你一个由 $n$ 个正整数组成的序列 $a$。

你必须重复执行以下操作**直到序列中只剩下 $1$ 个元素**：

- 选择两个**不同的**下标 $i$ 和 $j$。
- 然后选择一个正整数 $x$，使得存在一个以 $a_i$、$a_j$ 和 $x$ 为边长的**非退化三角形**$^{\text{∗}}$。
- 最后，删除 $a_i$ 和 $a_j$，并将 $x$ 添加到序列 $a$ 的末尾。

请找出序列 $a$ 中最后剩下的唯一元素的最大可能值。

$^{\text{∗}}$当边长 $a$、$b$、$c$ 满足 $a+b > c$、$a+c > b$ 且 $b+c > a$ 时，构成的三角形称为非退化三角形。

## 输入格式

每个测试包含多个测试用例。第一行包含测试用例的数量 $t$（$1 \le t \le 10^4$）。接下来是每个测试用例的描述。

每个测试用例的第一行包含一个整数 $n$（$1 \le n \le 2 \cdot 10^5$）。

每个测试用例的第二行包含 $n$ 个整数 $a_1, a_2, \ldots, a_n$（$1 \le a_i \le 1000$）——序列 $a$ 的元素。

保证所有测试用例的 $n$ 之和不超过 $2 \cdot 10^5$。

## 输出格式

对于每个测试用例，另起一行输出最后一个元素的最大可能值。


### 输入样例1

```
4
1
10
3
998 244 353
5
1 2 3 4 5
9
9 9 8 2 4 4 3 5 3
```

### 输出样例1

```
10
1593
11
39
```

## 说明/提示

#### 【样例解释】

在第一个测试用例中，序列初始就只有一个元素。最后一个元素的值就是 $10$。

在第二个测试用例中，初始序列为 $[998,244,353]$。可以执行以下操作序列：

1. 删除 $a_2=244$ 和 $a_3=353$，并在序列末尾添加 $596$。此时序列变为 $[998,596]$。
2. 删除 $a_1=998$ 和 $a_2=596$，并在序列末尾添加 $1593$。此时序列变为 $[1593]$。

可以证明最后一个元素的值不可能超过 $1593$。因此答案为 $1593$。

#### 数据范围

对于所有测试数据，都满足 $1 \le t \le 10^4$，$1 \le n \le 2 \cdot 10^5$，$1 \le a_i \le 10^5$，$\displaystyle\sum n \leq 2 \cdot 10^5$

|   测试点   |          $n$          |
| :--------: | :-------------------: |
| $1\sim 2$  |      $n \leq 2$       |
| $3\sim 8$  |     $n \leq 1000$     |
| $9\sim 20$ | $n \leq 2 \cdot 10^5$ |

## 题目大意

给一个长度为 $n$ 的数组，每次选择两个数字，再选择一个能和这两个数字组成一个非退化三角形的三条边的数字 $x$，把选出的两个数字删除，把 $x$ 加入到数组中。不断执行操作直到数组里只剩下一个数字，这个数字最大是多少？

## 思路分析

首先对于前两个测试点，如果 $n=1$ 直接输出这个数字，如果 $n=2$ 的话，很显然 $x$ 最大可以取到两个数字之和减一，这可以让学生想到后面的所有操作都是选择两个数字并且将其加和减一放入到数组里。

至于第二部分，个人认为并没有什么暴力可以写，因为无论怎么写暴力，其正确性都建立在第三条边最大是两个数字之和减一，因此只要能想到这个就能写出线性模拟的解，哪怕是使用 $\text{multiset}$ 这种带 $\log$ 容器，$O(n\log n)$ 也足够通过此题。

那么对于这个问题而言，一开始数字的总值是确定的，每次操作数组中就会少一个数字，每次会让总数减一，一共 $n$ 个数字操作 $n-1$  次。因此最后的答案就是所有的数字求和之后减去 $n-1$。

```c++
void solve() {
    int n;
    cin >> n;
    vector<int> vec(n);
    for (int i = 0; i < n; i++) cin >> vec[i];
    long long sum = 0;
    for (int i = 0; i < n; i++) sum += vec[i];
    cout << sum - (n - 1) << endl;
}
```
