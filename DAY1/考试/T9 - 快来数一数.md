# T2- 快来数一数

## 题目描述

Dmitry 正在玩一条字母纸带。纸带上从左到右写着一个由小写英文字母组成的字符串 $s$。

他会从纸带中剪掉两个连续的字符，并把剩下的部分重新接在一起。请你求出经过一次这样的操作后，可以得到多少个不同的字符串。

例如，当 $s = \texttt{aaabcc}$ 时，可以得到以下不同的字符串：

- $\texttt{abcc}$：删除第 $1, 2$ 个字符，或删除第 $2, 3$ 个字符；
- $\texttt{aacc}$：删除第 $3, 4$ 个字符；
- $\texttt{aaac}$：删除第 $4, 5$ 个字符；
- $\texttt{aaab}$：删除第 $5, 6$ 个字符。

因此一共可以得到 $4$ 个不同的字符串。

## 输入格式

每个测试文件包含多组测试数据。

第一行包含一个整数 $t$，表示测试数据组数。

对于每组测试数据：

第一行包含一个整数 $n$，表示字符串 $s$ 的长度。

第二行包含一个长度为 $n$ 的字符串 $s$，字符串仅由小写英文字母组成。

保证所有测试数据中 $n$ 的总和不超过 $2 \times 10^5$。

## 输出格式

对于每组测试数据，输出一行一个整数，表示删除两个连续字符后可以得到的不同字符串数量。

## 样例

### 样例 1 输入

```text
7
6
aaabcc
10
aaaaaaaaaa
6
abcdef
7
abacaba
6
cccfff
4
abba
5
ababa
```

### 样例 1 输出

```text
4
1
5
3
3
3
1
```

## 提示

### 【样例 1 解释】

第一组测试数据已在题目描述中解释。

对于第三组测试数据，可以得到以下 $5$ 个字符串：$\texttt{cdef}$、$\texttt{adef}$、$\texttt{abef}$、$\texttt{abcf}$、$\texttt{abcd}$。

对于第七组测试数据，无论删除哪两个连续字符，得到的字符串均为 $\texttt{aba}$，因此答案为 $1$。

### 【数据范围】

对于所有测试数据，保证：

- $1 \leq t \leq 10^4$；
- $3 \leq n \leq 2 \times 10^5$；
- 字符串 $s$ 仅由小写英文字母组成；
- 所有测试数据中 $n$ 的总和不超过 $2 \times 10^5$。

| 测试点编号 | $n \leq$ | 特殊性质 |
| :--: | :--: | :--: |
| $1 \sim 2$ | $20$ | A |
| $3$ | $2 \times 10^3$ | B |
| $4 \sim 5$ | $2 \times 10^4$ | C |
| $6 \sim 10$ | $2 \times 10^5$ | 无 |

特殊性质 A：保证字符串 $s$ 中至多包含两种不同字符。

特殊性质 B：保证字符串 $s$ 中所有字符均相同。

特殊性质 C：保证字符串 $s$ 中任意两个相邻字符均不相同。
