# T118324.修复二进制串

## 题目描述

给定一个长度为 `n` 的二进制字符串 `s`，由 `0` 和 `1` 组成。你可以执行以下操作恰好一次：

1. 选择一个整数 `p`（`1≤p≤n`）；
2. 反转子串 `s1 s2 ... sp`。执行此步骤后，字符串 `s1 s2 ... sn` 将变为 `sp sp-1 ... s1 s(p+1) s(p+2) ... sn`；
3. 然后，将字符串 `s` 循环左移 `p` 次。执行此步骤后，字符串 `s1 s2 ... sn` 将变为 `s(p+1) s(p+2) ... sn sp sp-1 ... s1`。

例如，对字符串 `110001100110` 执行 `p=3` 的操作：

第二步后变为 `011001100110`；

第三步后变为 `001100110011`。

称字符串 `s` 为 `k-proper`，当且仅当满足以下两个条件：

1. `s1 = s2 = ... = sk`；
2. 对所有 `i`（`1≤i≤n-k`），有 `s(i+k) ≠ si`。

当 `k=3` 时：

`000`、`111000111` 和 `111000` 是 `k-proper` 的；

`000000`、`001100` 和 `1110000` 不是 `k-proper` 的。

给定整数 `k`（保证是 `n` 的约数），找出一个满足 `1≤p≤n` 的整数 `p`，使得执行操作后字符串 `s` 变为 `k-proper`。如果不可能实现，请判定为无解。

注意：即使初始字符串已经是 `k-proper`，仍需执行恰好一次操作。

## 输入格式

每个测试包含多个测试用例。第一行输入一个整数 `t`（`1≤t≤10^4`）——表示测试用例的数量。接下来是每个测试用例的描述：

第一行包含两个整数 `n` 和 `k`（`1≤k≤n≤10^5`）——分别表示字符串 `s` 的长度和参数 `k` 的值。保证 `k` 是 `n` 的约数。

第二行包含一个长度为 `n` 的二进制字符串 `s`，由字符 `0` 和 `1` 组成。

保证所有测试用例的 `n` 之和不超过 `2⋅10^5`。

## 输出格式

对于每个测试用例，输出一个整数：

若能通过操作使字符串变为 `k-proper`，输出满足条件的 `p` 值（`1≤p≤n`）；

若不可能实现，输出 `-1`。

若存在多个可行的 `p` 值，输出其中任意一个即可。

## 说明 / 提示

在第一个测试用例中，若选择 `p=3` 执行操作：

第二步操作后字符串变为 `11100001`；

第三步操作后字符串变为 `00001111`；

最终字符串满足 `4-proper` 条件。

在第二个测试用例中，可以证明不存在任何操作能使字符串变为 `2-proper`。

在第三个测试用例中，若选择 `p=7` 执行操作：

第二步操作后字符串变为 `100011100011`；

第三步操作后字符串变为 `000111000111`；

最终字符串满足 `3-proper` 条件。

在第四个测试用例中，任意选择 `p` 执行操作后，字符串都满足 `5-proper` 条件。

## 测试点

1. `1∼6`：`n≤1000`
2. `7∼10`：`2≤n≤10^5`，`k=1`
3. `11∼20`：`2≤n≤10^5`
