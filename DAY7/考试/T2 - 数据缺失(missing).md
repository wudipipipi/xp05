# missing

输入文件：missing.in

输出文件：missing.out

## 题目描述

现在有一个长度为 $n$ 的序列 $a$，定义

$$
f_{i,j}=\max(a_i,a_j) \quad (1 \leq i < j \leq n)。
$$

小 d 收集了所有 $\frac{n(n-1)}{2}$ 个 $f_{i,j}$，并将它们放在一个无序的容器中。但是今天他发现，有一个 $f_{i,j}$ 消失了，容器中只剩下 $\frac{n(n-1)}{2}-1$ 个数字。

现在给出这些数据，你需要找出缺失的数字是什么。

如果合法的缺失数字只有一种可能，则输出这个数字；如果有无穷多种可能，则输出 Infinite。

可以证明，合法的缺失数字只有一种或无穷多种。

注意：给出的数的范围为 $[1,10^9]$，但是序列 $a$ 中可以包含不在这个范围内的数。

## 输入格式

输入的第一行包含一个整数 $t$，表示测试数据组数。

对于每组数据，第一行包含一个整数 $n$，表示原始序列的长度。

接下来一行包含 $\frac{n(n-1)}{2}-1$ 个整数，表示缺失一个数据后的所有 $f_{i,j}$。

数据保证至少存在一个合法的缺失数字。

## 输出格式

对于每组测试数据，输出一行。

如果合法的缺失数字只有一种可能，则输出这个值；否则输出 Infinite。

## 样例

### 样例 1 输入

```text
8
3
2 3
3
3 3
4
3 4 3 4 4
4
2 4 3 4 4
4
2 3 3 4 4
4
2 3 4 4 4
4
2 3 4 3 4
4
2 3 4 3 4
```

### 样例 1 输出

```text
3
Infinite
Infinite
3
4
3
4
4
```

### 样例 2 输入

```text
2
3
5 8
3
7 7
```

### 样例 2 输出

```text
8
Infinite
```

## 提示

### 【样例 2 解释】

样例 $2$ 包含两组测试数据，第一组的缺失数字唯一，第二组的缺失数字有无穷多种可能，因此对应输出如上。

### 【数据范围】

本题采用测试点捆绑，同一行中的测试点作为一个测试点组计分。

对于所有测试数据，保证：

- $1 \leq t \leq 100$；
- $3 \leq n \leq 1000$；
- 给出的数均在 $[1,10^9]$ 内；
- $\sum n \leq 1000$。

| 子任务 | 分值 | $t$ | $n$ | 特殊性质 |
| :--: | :--: | :--: | :--: | :--: |
| $1$ | $30$ | $5$ | $5$ | 无 |
| $2$ | $20$ | $100$ | $1000$ | A |
| $3$ | $20$ | $100$ | $1000$ | B |
| $4$ | $30$ | $100$ | $1000$ | 无 |

特殊性质 A：给出的数均在集合 $\{1,2\}$ 中。

特殊性质 B：给出的数中有至少 $n-1$ 个不同的整数。
