# 众最大数

输入文件：max.in

输出文件：max.out

## 题目描述

小 $\mathcal{l}$ 有一个大小为 $n$ 的可重集合 $s$。将其中元素从小到大排序后，第 $i$ 个元素为 $s_i$。

对于这个大小为 $n$ 的集合，存在 $2^n-1$ 个非空子集 $t \subseteq s$。对于相同值的元素，若它们在原集合中的位置不同，则仍视为不同的元素。

现在将所有非空子集的最大值放入一个新的可重集合 $s'$ 中，即

$$
s'=\{\max(t)\mid t \subseteq s,\ t \neq \varnothing\}。
$$

小 $\mathcal{l}$ 想知道 $s'$ 的众数是多少。如果存在多个数出现次数同为最多，则输出其中最大的数。

## 输入格式

输入的第一行包含一个整数 $n$，表示可重集合 $s$ 的大小。

输入的第二行包含 $n$ 个正整数 $s_1,s_2,\dots,s_n$。保证对于所有 $2 \leq i \leq n$，均有 $s_{i-1} \leq s_i$。

## 输出格式

输出一行一个整数，表示集合 $s'$ 的最大众数。

## 样例

### 样例 1 输入

```text
4
1 2 2 3
```

### 样例 1 输出

```text
3
```

## 提示

### 【样例 1 解释】

当 $s=\{1,2,2,3\}$ 时，可以得到

$$
s'=\{1,2,2,3,2,2,3,2,3,3,2,3,3,3,3\}。
$$

在一共 $15$ 个数中，$1$ 出现了 $1$ 次，$2$ 出现了 $6$ 次，$3$ 出现了 $8$ 次，所以众数为 $3$。

### 【数据范围】

本题采用测试点捆绑，同一行中的测试点作为一个测试点组计分。

对于所有测试数据，保证：

- $1 \leq n \leq 10^5$；
- $1 \leq s_i \leq 10^9$；
- 对于所有 $2 \leq i \leq n$，均有 $s_{i-1} \leq s_i$。

| 子任务 | 分值 | $n$ |
| :--: | :--: | :--: |
| $1$ | $30$ | $10$ |
| $2$ | $30$ | $1000$ |
| $3$ | $40$ | $10^5$ |
