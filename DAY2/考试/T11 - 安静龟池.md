# T5- 安静龟池

## 题目描述

小码君在池塘里搭了一片 $N \times M$ 的网箱养殖区。每个格子都是一个独立的小网箱，有些格子是水域，有些格子已经安装了养殖设备。

小乌龟可以养在水域格子中，不能养在有设备的格子中。虽然小乌龟看起来很安静，但它们非常在意自己的领地：如果两只小乌龟被放在上下左右相邻的两个格子里，它们就会互相打扰。

因此，一个饲养方案需要满足：

- 至少放入一只小乌龟；
- 只能把小乌龟放在水域格子中；
- 任意两只小乌龟所在格子不能上下左右相邻。

请你计算一共有多少种可行的饲养方案。答案可能很大，请对 $10^9 + 7$ 取模。

## 输入格式

第一行包含两个整数 $N, M$，表示养殖区有 $N$ 行 $M$ 列。

接下来 $N$ 行，每行包含 $M$ 个整数。其中 $1$ 表示水域格子，$0$ 表示安装了养殖设备的格子。

## 输出格式

输出一行一个整数，表示可行饲养方案数对 $10^9 + 7$ 取模后的结果。

## 样例

### 样例 1 输入

```text
2 2
1 1
0 1
```

### 样例 1 输出

```text
4
```

### 样例 2 输入

```text
2 3
1 1 0
0 1 1
```

### 样例 2 输出

```text
7
```

## 提示

### 【样例 1 解释】

样例 $1$ 中共有 $3$ 个水域格子。

只养一只小乌龟时，可以任选一个水域格子，共 $3$ 种方案。

养两只小乌龟时，只有把它们放在两个对角位置的水域格子中才可行，共 $1$ 种方案。

因此一共有 $3 + 1 = 4$ 种可行方案。

### 【数据范围】

对于所有测试数据，保证：

- $1 \leq N \leq 500$；
- $1 \leq M \leq 10$；
- 每个格子的状态为 $0$ 或 $1$。

| 测试点编号 | 特殊性质 |
| :--: | :--: |
| $1 \sim 5$ | $1 \leq N \leq 10$ |
| $6 \sim 10$ | $80 \leq N \leq 100$ |
| $11 \sim 25$ | 无 |
