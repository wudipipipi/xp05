# 长廊信标

## 题目描述

长廊上从左到右排着 $n$ 座信标塔，编号为 $1$ 到 $n$。第 $i$ 座塔上写着一个整数 $a_i$。

若 $a_i=0$，这座塔是沉默塔，不需要被点亮，也不会向外传递信号。

若 $a_i>0$，这座塔是信标塔。信标塔被点亮后，会在之后的每一秒向右传递信号：如果第 $i$ 座塔在某一秒开始时已经被点亮，那么在这一秒内，它会尝试点亮编号从

$$
i+1
$$

到

$$
\min(n,i+a_i)
$$

之间所有尚未点亮的信标塔。沉默塔不会被点亮，也不需要被点亮。

在第 $0$ 秒，你可以手动点亮至多 $k$ 座信标塔。之后从第 $1$ 秒开始，所有已经点亮的信标塔同步传递信号。注意：某座塔如果是在当前这一秒内刚刚被点亮，那么它要从下一秒开始才能继续传递信号。

请你求出一个最优的初始点亮方案，使得所有信标塔都被点亮所需的时间最短。若无论如何都无法点亮所有信标塔，输出 $-1$。

如果一开始就已经没有需要点亮的信标塔，答案为 $0$。

## 输入格式

本题有多组测试数据。

第一行包含一个整数 $T$，表示测试数据组数。

对于每组测试数据：

第一行包含两个整数 $n,k$，分别表示信标塔总数，以及第 $0$ 秒最多可以手动点亮的信标塔数量。

第二行包含 $n$ 个整数 $a_1,a_2,\ldots,a_n$。

## 输出格式

对于每组测试数据，输出一行一个整数：

- 若可以点亮所有信标塔，输出最短时间；
- 否则输出 $-1$。

## 样例

### 样例 1 输入

```text
4
7 2
2 0 1 1 0 1 0
5 3
0 2 0 1 5
6 1
2 0 1 0 1 1
5 1
2 1 1 1 1
```

### 样例 1 输出

```text
2
0
-1
3
```

## 数据范围与约定

对于所有测试数据，保证：

- $1\le T\le 10^5$；
- $1\le k\le n\le 5\times 10^5$；
- $0\le a_i\le n$；
- 单个测试文件内所有测试数据的 $n$ 之和不超过 $5\times 10^5$。

本题采用测试点捆绑计分，同一子任务内的所有测试点均通过才能获得该子任务分数。

| 子任务 | 测试点编号 | 分值 | $n$ | $a_i$ | 特殊性质 |
| :--: | :--: | :--: | :--: | :--: | :--: |
| $1$ | $1 \sim 2$ | $6$ | $\le 15$ | $\le n$ | 无 |
| $2$ | $3 \sim 4$ | $8$ | $\le 5\times 10^5$ | $\le n$ | A |
| $3$ | $5 \sim 8$ | $12$ | $\le 5\times 10^5$ | $\le 1$ | 无 |
| $4$ | $9 \sim 12$ | $14$ | $\le 3000$ | $\le n$ | 无 |
| $5$ | $13 \sim 16$ | $14$ | $\le 5\times 10^5$ | $\le n$ | B |
| $6$ | $17 \sim 20$ | $16$ | $\le 5\times 10^5$ | $\le n$ | C |
| $7$ | $21 \sim 30$ | $30$ | $\le 5\times 10^5$ | $\le n$ | 无 |

特殊性质 A：设 $w$ 为满足 $a_i>0$ 的位置数量，则 $w\le k$。

特殊性质 B：所有 $a_i>0$，并且 $i+a_i$ 随 $i$ 单调不降。

特殊性质 C：$k=1$。
