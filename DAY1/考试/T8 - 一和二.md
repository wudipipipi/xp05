# T1- 一和二

## 题目描述

小码君有一排写着数字的糖果，每颗糖果上只可能写着 $1$ 或 $2$。他想把这排糖果切成左右两段，再把两段糖果上的数字分别相乘。

给定一个长度为 $n$ 的序列 $a_1, a_2, \ldots, a_n$，其中每个元素均为 $1$ 或 $2$，表示这些糖果上的数字。

请判断是否存在一个整数 $k$，满足：

- $1 \leq k \leq n - 1$；
- $a_1 \times a_2 \times \cdots \times a_k = a_{k+1} \times a_{k+2} \times \cdots \times a_n$。

如果存在多个满足条件的 $k$，请输出最小的一个；如果不存在，请输出 $-1$。

## 输入格式

每个测试文件包含多组测试数据。

第一行包含一个整数 $t$，表示测试数据组数。

对于每组测试数据：

第一行包含一个整数 $n$，表示序列长度。

第二行包含 $n$ 个整数 $a_1, a_2, \ldots, a_n$，表示给定序列。

## 输出格式

对于每组测试数据，输出一行一个整数。

如果不存在满足条件的 $k$，输出 $-1$；否则输出最小的满足条件的 $k$。

## 样例

### 样例 1 输入

```text
3
6
2 2 1 2 1 2
3
1 2 1
4
1 1 1 1
```

### 样例 1 输出

```text
2
-1
1
```

### 【样例 1 解释】

对于第一组测试数据，当 $k = 2$ 时，有 $a_1 \times a_2 = a_3 \times a_4 \times a_5 \times a_6 = 4$，满足条件。虽然 $k = 3$ 也满足条件，但应输出最小的 $k$，因此答案为 $2$。

对于第二组测试数据，不存在满足条件的 $k$，因此答案为 $-1$。

对于第三组测试数据，$k = 1, 2, 3$ 均满足条件，因此答案为 $1$。

### 【数据范围】

对于所有测试数据，保证：

- $1 \leq t \leq 100$；
- $2 \leq n \leq 1000$；
- 对于所有 $1 \leq i \leq n$，均有 $1 \leq a_i \leq 2$。

| 测试点编号 | $n \leq$ | 特殊性质 |
| :--: | :--: | :--: |
| $1 \sim 2$ | $30$ | 无 |
| $3 \sim 4$ | $100$ | 无 |
| $5 \sim 6$ | $1000$ | A |
| $7 \sim 8$ | $1000$ | B |
| $9 \sim 20$ | $1000$ | 无 |

特殊性质 A：保证对于所有 $1 \leq i \leq n$，均有 $a_i = 1$。

特殊性质 B：保证对于所有 $1 \leq i \leq n$，均有 $a_i = 2$。
