# 建设工程

输入文件：train.in

输出文件：train.out

## 题目描述

某国有 $n$ 个火车站，编号为 $1 \sim n$。另外，该国有 $m$ 条双向铁路线，编号为 $1 \sim m$。第 $i$ 条铁路线连接了火车站 $a_i$ 和火车站 $b_i$，从一个站到另一个站需要花费 $c_i$ 分钟。

你是该国的交通部长，决定按照以下方式新建一条铁路线：

选择两个整数 $u,v$，满足 $1 \leq u < v \leq n$，在火车站 $u$ 和火车站 $v$ 之间建设一条双向铁路线，从一个站到另一个站需要花费 $l$ 分钟。注意，即使已经有一条连接火车站 $u$ 和火车站 $v$ 的铁路线，也可以继续建设这条新铁路线。

如果建设这条铁路线后，可以花费不超过 $k$ 分钟从火车站 $s$ 到火车站 $t$，国王就会高兴。我们不考虑换乘时间和等待时间。

你共有 $\frac{n(n-1)}{2}$ 种选择两个整数 $u,v$ 的方法。请你求出，其中有多少种方法会让国王高兴。

## 输入格式

输入的第一行包含两个整数 $n,m$。

输入的第二行包含四个整数 $s,t,l,k$。

接下来 $m$ 行，每行包含三个整数 $a_i,b_i,c_i$，表示第 $i$ 条双向铁路线。

## 输出格式

输出一行一个整数，表示让国王高兴的选择方法数量。

## 提示

### 【样例 1 解释】

在样例 $1$ 中，新建铁路的耗时为 $1$ 分钟，国王要求从火车站 $6$ 到火车站 $7$ 的总耗时不超过 $2$ 分钟。

共有以下 $4$ 种选择可以满足要求：

- 在火车站 $6$ 与火车站 $7$ 之间新建铁路；
- 在火车站 $3$ 与火车站 $6$ 之间新建铁路，此时可走 $6 \to 3 \to 7$；
- 在火车站 $1$ 与火车站 $7$ 之间新建铁路，此时可走 $6 \to 1 \to 7$；
- 在火车站 $5$ 与火车站 $7$ 之间新建铁路，此时可走 $6 \to 5 \to 7$。

因此答案为 $4$。

### 【数据范围】

本题采用测试点捆绑，同一行中的测试点作为一个测试点组计分。

对于所有测试数据，保证：

- $2 \leq n \leq 2 \times 10^5$；
- $1 \leq m \leq 2 \times 10^5$；
- $1 \leq s < t \leq n$；
- $1 \leq l \leq 10^9$；
- $1 \leq k \leq 10^{15}$；
- $1 \leq a_i < b_i \leq n$；
- 对于所有 $1 \leq i < j \leq m$，均有 $(a_i,b_i) \neq (a_j,b_j)$；
- $1 \leq c_i \leq 10^9$。

| 子任务 | 分值 | $n$ | $m$ | 特殊性质 |
| :--: | :--: | :--: | :--: | :--: |
| $1 \sim 4$ | $8$ | $2 \times 10^5$ | $2 \times 10^5$ | A |
| $5 \sim 12$ | $16$ | $50$ | $50$ | 无 |
| $13 \sim 26$ | $28$ | $3000$ | $3000$ | 无 |
| $27 \sim 50$ | $48$ | $2 \times 10^5$ | $2 \times 10^5$ | 无 |

特殊性质 A：$l=1$，$k=2$，且对于所有 $1 \leq i \leq m$，均有 $c_i=1$。
