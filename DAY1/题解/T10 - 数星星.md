# The Morning Star

## 题目描述

给定 $n$ 个点，第 $i$ 个点的坐标为 $x_i$，$y_i$。现需要将星星和指南针放在任意两个点上，使得星星在指南针的正北、正东、正西、正南、正东南、正东北、正西南或正西北方向，求一共几种放法。

## 输入格式

第一行输入一个$t$，表示数据组数。

在每组数据中，输入$n$，表示点的总数。

接下来 $n$ 行，第$i$行输入两个数 $x_i,y_i$，表示第$i$个点的坐标。

## 输出格式

输出共$t$行，每行一个整数，表示第$i$组数据的结果。

## 输入样例

```
5
3
0 0
-1 -1
1 1
4
4 5
5 7
6 9
10 13
3
-1000000000 1000000000
0 0
1000000000 -1000000000
5
0 0
2 2
-1 5
-1 10
2 11
3
0 0
-1 2
1 -2
```

## 输出样例

```
6
2
6
8
0
```

## 说明/提示

### 【样例1解释】

在第一组数据中：

指南针在 $(0,0)$，星星在 $(-1,-1)$，在指南针的正西南方向。

指南针在 $(0,0)$，星星在 $(1,1)$，在指南针的正东北方向。

指南针在 $(-1,-1)$，星星在 $(0,0)$，在指南针的正东北方向。

指南针在$(-1,-1)$，星星在$(1,1)$，在指南针的正东北方向。

指南针在$(1,1)$，星星在$(0,0)$，在指南针的正西南方向。

指南针在$(1,1)$，星星在$(-1,-1)$，在指南针的正西南方向。

所以答案为6。

在第二组数据中：

指南针在$(6,9)$，星星在$(10,13)$，在指南针的正东北方向。

指南针在$(10,13)$，星星在$(6,9)$，在指南针的正西南方向。

所以答案是 2。

### 数据范围

对于所有测试数据，都满足$2\leq n \leq 10^5$，$-10^9\leq x_i,y_i \leq 10^9$

所有测试用例中 $n$ 的总和不超过$2 \times 10^{5}$。

|   测试点   |          $n,k$          |                      $x_i,y_i$                      |
| :--------: | :---------------------: | :-------------------------------------------------: |
| $1\sim 1$  |   $1 \leq n \leq 300$   |       $ 0 \leq abs(x[i]), abs(y[i]) \leq 100$       |
| $2\sim 2$  | $ 1 \leq n \leq 10^{4}$ |     $ 0 \leq abs(x[i]), abs(y[i]) \leq 10^{4}$      |
| $3\sim 3$  | $ 1 \leq n \leq 10^{4}$ | $ 0 \leq abs(x[i]), abs(y[i]) \leq 2 \times 10^{4}$ |
| $4\sim 10$ | $ 1 \leq n \leq 10^{5}$ | $ 0 \leq abs(x[i]), abs(y[i]) \leq 2 \times 10^{9}$ |

## 题目大意

选择任意两个点，判断两个点是否在同一条直线上，只需要考虑直线的斜率为$-1$,$0$,$-1$,不存在的情况。记录对于每个点，只记录这四种情况，对于斜率为$-1$,$0$,$1$三种情况记录对应的k值，对于不存在的情况记录对应的x值，假设第i种情况有k个值，使用等差数列求值即可，共$k \times (k + 1) / 2$种。



## 思路分析

对于前$3$个测试点只需要使用特判+暴力求解即可。对于大数据的情况，考虑上、下、左、右、左上、左下、右上、右下$8$个位置的所对应的斜率，记录对应的$b$值或对应的$x$值，最后累加所有的情况即可。时间复杂度$O(nlogn)$。

```
#include <bits/stdc++.h>
#define startt ios_base::sync_with_stdio(false);cin.tie(0);
typedef long long  ll;
using namespace std;
#define vint vector<int>
#define all(v) v.begin(), v.end()
#define int long long
 
void solve()
{
    int n;
    cin >> n;
    map<int, int> up, side, diag1, diag2;
    int ans = 0;
    for(int i = 0; i < n; i++)
    {
        int x, y;
        cin >> x >> y;
        up[x]++;
        side[y]++;
        diag1[x-y]++;
        diag2[x+y]++;
    }
    for(auto x : up)
    {
        ans+=x.second*(x.second-1);
    }
    for(auto x : side)
    {
        ans+=x.second*(x.second-1);
    }
    for(auto x : diag1)
    {
        ans+=x.second*(x.second-1);
    }for(auto x : diag2)
    {
        ans+=x.second*(x.second-1);
    }
    cout << ans << endl;
}
 
int32_t main(){
    startt
    int t = 1;
    cin >> t;
    while (t--) {
        solve();
    }
}
```

时间复杂度$O(nlogn)$，对于所有的测试点都可以通过。
