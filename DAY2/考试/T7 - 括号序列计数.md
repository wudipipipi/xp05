# T1- 墨迹括号

## 题目描述

小 A 正在整理一份括号稿件。稿件中有些字符已经能看清是 $\texttt{(}$ 或 $\texttt{)}$，还有一些字符被墨水盖住了，只留下了 $\texttt{?}$。

现在需要把每个 $\texttt{?}$ 分别改成 $\texttt{(}$ 或 $\texttt{)}$。若字符串中一共有 $x$ 个 $\texttt{?}$，则共有 $2^x$ 种替换方式。

请你计算，有多少种替换方式可以使最终得到的字符串成为一个合法括号序列。答案需要对 $998244353$ 取模。

一个字符串是合法括号序列，当且仅当满足以下条件：

- 从左到右扫描字符串时，任意时刻已经出现的右括号数量都不超过左括号数量；
- 扫描完整个字符串后，左括号数量与右括号数量相等。

## 输入格式

输入一行一个非空字符串 $S$。

## 输出格式

输出一行一个整数，表示合法替换方式的数量对 $998244353$ 取模后的结果。

## 样例

### 样例 1 输入

```text
(???(?
```

### 样例 1 输出

```text
2
```

### 样例 2 输入

```text
??????????????(????????(??????)?????????(?(??)
```

### 样例 2 输出

```text
603032273
```

### 样例 3 输入

```text
)))))
```

### 样例 3 输出

```text
0
```

## 提示

### 【样例 1 解释】

对于样例 $1$，只有两种替换方式可以得到合法括号序列：

- $\texttt{()()()}$；
- $\texttt{(())()}$。

因此答案为 $2$。

### 【数据范围】

对于所有测试数据，保证：

- $1 \leq |S| \leq 3000$；
- 字符串 $S$ 仅由 $\texttt{(}$、$\texttt{)}$ 和 $\texttt{?}$ 组成。
