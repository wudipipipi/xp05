# 一和二

## 题目描述

你有一个序列 $a_1, a_2, \ldots, a_n$，每个元素是 $1$ 或 $2$。

判断是否存在一个整数 $k$ 满足以下条件：
- $1 \leq k \leq n-1$，
- $a_1 \cdot a_2 \cdot \ldots \cdot a_k = a_{k+1} \cdot a_{k+2} \cdot \ldots \cdot a_n$。

如果存在多个满足条件的 $k$，输出最小的那个。若不存在，输出 $-1$。

## 输入格式

每个测试包含多个测试用例。第一行包含测试用例的数量 $t$ （ $1 \le t \le 100$ ）。( $1 \le t \le 100$ ).测试用例说明如下。

每个测试用例的第一行包含一个整数 $n$ ( $2 \leq n \leq 1000$ )。

每个测试用例的第二行包含 $n$ 个整数 $a_1, a_2, \ldots, a_n$ ( $1 \leq a_i \leq 2$ )。

## 输出格式

对于每个测试用例，如果没有 $k$ ，则打印 $-1$ 。

否则，打印尽可能小的 $k$ 。


### 输入样例1

```
3
6
2 2 1 2 1 2
3
1 2 1
4
1 1 1 1
```

### 输出样例1

```
2
-1
1
```

## 说明/提示

#### 【样例 1 解释】

**注**

对于第一个测试用例， $k=2$ 满足条件，因为 $a_1 \cdot a_2 = a_3 \cdot a_4 \cdot a_5 \cdot a_6 = 4$ . $k=3$ 也满足给定条件，但应打印最小值。

对于第二个测试用例，没有 $k$ 满足 $a_1 \cdot a_2 \cdot \ldots \cdot a_k = a_{k+1} \cdot a_{k+2} \cdot \ldots \cdot a_n$ 的条件。

对于第三个测试用例， $k=1$ 、 $2$ 和 $3$ 满足给定条件，因此答案为 $1$ 。

对于所有测试点都满足 $1\leq t\leq 100$，$2\leq n \leq 1000$，$1\leq a_i\leq 2$。

|   测试点   |      $n$      |     特殊性质     |
| :--------: | :-----------: | :--------------: |
| $1\sim 2$  |  $n \leq 30$  |        无        |
| $3\sim 4$  | $n \leq 100$  |        无        |
| $5\sim 6$  | $n \leq 1000$ | $a_i$ 必定为 $1$ |
| $7\sim 8$  | $n \leq 1000$ | $a_i$ 必定为 $2$ |
| $9\sim 20$ | $n \leq 1000$ |        无        |


## 题目大意

给定一个只有 $1$ 和 $2$ 组成的数组，要求找到一个位置将数组分成两部分，使得两部分的乘积相等。存在多个位置时输出最小的。

## 思路分析

首先这个题可以暴力算一个前缀积和后缀积，之后比较前缀数组和后缀数组在哪一位上相等了。把这个代码里面等于二时的乘以二变成加一，也就是我们只考虑数字 $2$ 的个数的话，也可以得到一种解。

```c++
void solve() {
    int n;
    cin >> n;
    vector<long long> a(n + 1);
    for(int i = 1; i <= n; i++) cin >> a[i];
    vector<long long> pre(n + 2,1);
    vector<long long> suf(n + 2,1);
    for(int i = 1; i <= n; i++) {
        if(a[i] == 2){
            pre[i] = pre[i-1] * 2;
        }else{
            pre[i] = pre[i-1];
        }
    }
    for(int i = n; i >= 1; i--) {
        if(a[i] == 2){
            suf[i] = suf[i-1] * 2;
        }else{
            suf[i] = suf[i-1];
        }
    }
    for(int i = 1; i < n; i++) {
        if(pre[i] == suf[i + 1]) {
            cout << i << endl;
            return ;
        }
    }
    cout << -1 << endl;
}
```

下面是正解：

对于全是 $1$ 的情况，直接把第一个数字单独拿出来就是对的。我们实际上维护两边的乘积就是在维护两边 $2$ 的个数，所以只有 $2$ 的个数为偶数的时候是有解的。从前向后找到正好有 $2$ 的总个数一半的位置就是对的。

```c++
void solve() {
    int n;
    cin >> n;
    vector<int> vec(n+1);
    int cnt2 = 0, cnt1 = 0;
    for (int i = 1; i <= n; ++i) {
        cin >> vec[i];
        if(vec[i] == 1) cnt1++;
        else cnt2++;
    }
    if(cnt2 == 0){
        cout << 1 << endl;
    }else if(cnt2 % 2 == 1){
        cout << -1 << endl;
    }else{
        int tmp = cnt2 / 2;
        int cur = 0;
        for (int i = 1; i <= n; ++i) {
            if(vec[i] == 2){
                cur++;
            }
            if(cur == tmp){
                cout << i << endl;
                return;
            }
        }
    }
}
```
