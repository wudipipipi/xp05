## 友好区间的抉择 (sequence) 题解

这是一道极具启发性的位运算与双指针结合的题目。通过挖掘位运算的数学性质，我们可以将一个看似复杂的区间计数问题，转化为极其优雅的单调性问题。

---

### 核心性质推导

首先，对于任意非负整数，有一个恒等的数学关系：


$$\bigoplus_{i=L}^R a_i \le \sum_{i=L}^R a_i$$


即**按位异或和永远不可能大于算术和**。

题目要求的条件是异或和 $\ge$ 算术和，这意味着条件被强行锁死在**两者相等**的情况：


$$\bigoplus_{i=L}^R a_i = \sum_{i=L}^R a_i$$

两个非负整数相加等于它们的异或，当且仅当它们的二进制位**没有任何一位同时为 1**（即相加时完全不产生进位）。
推广到多个数，区间 $[L, R]$ 合法的充要条件是：**该区间内任意两个数在二进制下按位与（AND）的结果均为 0**。

因为题目限定 $a_i < 2^{20}$，每个数最多只有 20 个二进制位。由于每个二进制位在合法区间中最多只能出现 1 次，我们可以得到本题的黄金破绽：

> 💡 **任何一个友好区间内，严格大于 0 的非零元素个数绝对不能超过 20 个！**（`0` 的数量是任意的）。

---

### 算法设计：双指针 + 二分 + 前缀和

既然核心限制是非零点数量，我们很容易联想到**单调性**。

#### 1. 双指针预处理

我们定义 $f(R)$ 为：**以 $R$ 为右端点时，向左延伸能达到的最左（最小）合法左端点**。也就是说，只要 $L \in [f(R), R]$，子段 $[L, R]$ 就是合法的。

* **单调性证明**：如果区间 $[L, R]$ 是合法的，那么去掉左端点的元素，子段 $[L+1, R]$ 必然不可能发生新的位冲突，它也一定是合法的。因此，随着 $R$ 的右移，左边界 $f(R)$ 只可能向右移动或保持不变，具有非降的单调性。
* **具体实现**：我们可以用一个频次数组记录当前窗口内每个二进制位出现的次数。当右端点 $R$ 右移引入 $a_R$ 时，若某些二进制位出现次数超过 1，则不断右移左指针 $L$，直到消除位冲突。此时记录 $f(R) = L$。由于非零点极少，该步骤的时间复杂度为 $O(20N)$。

#### 2. 区间查询的数学拆分

对于一个询问 $[l, r]$，我们要统计有多少对 $(L, R)$ 满足 $l \le L \le R \le r$ 且 $f(R) \le L$。
这意味着对于区间内的每一个右端点 $R$，它能贡献的合法左端点数量为：


$$\text{贡献} = R - \max(l, f(R)) + 1$$

由于 $f(R)$ 是单调递增的，对于固定的询问左端点 $l$，整个查询区间 $[l, r]$ 会被 $f(R)$ 是否大于 $l$ **精准地切成左右两部分**：

1. **左半部分（$R \in [l, mid]$）**：此时 $f(R) \le l$，于是 $\max(l, f(R)) = l$。
* 每一个 $R$ 的贡献是 $R - l + 1$。
* 随着 $R$ 的增长，贡献依次为 $1, 2, 3, \dots, mid - l + 1$。这构成了一个**等差数列**！我们可以利用求和公式 $O(1)$ 直接算出这部分的贡献。


2. **右半部分（$R \in [mid+1, r]$）**：此时 $f(R) > l$，于是 $\max(l, f(R)) = f(R)$。
* 每一个 $R$ 的贡献是 $R - f(R) + 1$。
* 我们可以定义一个新数组 $g(i) = i - f(i) + 1$，并预处理出 $g$ 的**前缀和数组** $S_g$。那么这部分的贡献就是 $S_g[r] - S_g[mid]$，同样可以 $O(1)$ 求解。



#### 3. 寻找分界点 $mid$

由于 $f$ 数组单调递增，我们直接在 $f$ 数组的 $[l, r]$ 范围内通过 `std::upper_bound` **二分查找**最后一个满足 $f(R) \le l$ 的位置，该位置即为 $mid$。

---

### 复杂度分析

* **时间复杂度**：
* 预处理双指针和前缀和：$O(20N)$。
* 单次询问：二分查找需要 $O(\log n)$，计算公式 $O(1)$，总计 $O(q \log n)$。
* 总时间复杂度为 $O(20N + q \log n)$，面对 $2 \times 10^5$ 的数据规模可以在 0.1 秒内跑完，极其高效。


* **空间复杂度**：只需存储基础数组和前缀和，为 $O(n)$。

### C++ 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

typedef long long ll;

void solve() {
    int n, q;
    if (!(cin >> n >> q)) return;

    vector<int> a(n + 1);
    for (int i = 1; i <= n; i++) {
        cin >> a[i];
    }

    vector<int> f(n + 1);
    vector<int> bit_cnt(20, 0); // 记录每个二进制位在当前窗口中出现的次数
    
    // 1. 双指针预处理 f 数组
    int L = 1;
    for (int R = 1; R <= n; R++) {
        // 将 a[R] 的各个二进制位加入窗口
        for (int i = 0; i < 20; i++) {
            if ((a[R] >> i) & 1) {
                bit_cnt[i]++;
            }
        }
        
        // 如果发生位冲突（某一位出现次数 > 1），右移左指针 L
        while (true) {
            bool conflict = false;
            for (int i = 0; i < 20; i++) {
                if (bit_cnt[i] > 1) {
                    conflict = true;
                    break;
                }
            }
            if (!conflict) break;
            
            // 移出 a[L]
            for (int i = 0; i < 20; i++) {
                if ((a[L] >> i) & 1) {
                    bit_cnt[i]--;
                }
            }
            L++;
        }
        f[R] = L;
    }

    // 2. 预处理 g 数组及其前缀和
    vector<ll> g(n + 1), sum_g(n + 1, 0);
    for (int i = 1; i <= n; i++) {
        g[i] = i - f[i] + 1;
        sum_g[i] = sum_g[i - 1] + g[i];
    }

    // 3. 回答询问
    while (q--) {
        int l, r;
        cin >> l >> r;

        // 在 f 数组的 [l, r] 区间内二分查找最后一个满足 f[R] <= l 的位置
        // 也就是在 [l, r] 中找 upper_bound(l) - 1
        auto it = upper_bound(f.begin() + l, f.begin() + r + 1, l);
        int mid = (it - f.begin()) - 1;

        ll ans = 0;
        
        // 左半部分：f[R] <= l，贡献为等差数列
        if (mid >= l) {
            ll len = mid - l + 1;
            ans += len * (len + 1) / 2;
        }
        
        // 右半部分：f[R] > l，利用前缀和 O(1) 求解
        if (mid < r) {
            ans += sum_g[r] - sum_g[max(mid, l - 1)];
        }

        cout << ans << "\n";
    }
}

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    solve();
    return 0;
}

```
