# T118338.选举

## 题目描述

输入文件：`election.in`  
输出文件：`election.out`

JOI共和国由 `N` 个州组成，现在正在进行选举。理惠想要赢得选举，她决定通过发表演讲来提高支持率：

1. 在某州演讲时间达 `Ai` 可获一张选票；
2. 达 `Bi` 可获一名协作者（可与她同时在任何州演讲，多人同州演讲计多倍时间）；
3. 若 `Bi = -1` 表示无法获协作者，且保证 `Bi ≥ Ai`。

求理惠最快获得 `K` 张选票的最小演讲时间。

## 输入格式

第一行整数 `N`；

第二行整数 `K`；

接下来 `N` 行，每行两个整数 `Ai, Bi`。

## 输出格式

输出一个实数，表示所需最小演讲时间。答案与标准答案绝对误差不超过 `0.01`。

## 说明 / 提示

### 样例解释 #1

按照如下方案进行演讲，理惠将在 `5.5` 小时内赢得每个州的选票。

1. 在第 `2` 个州演讲 `2` 个小时，赢得一张选票。
2. 在第 `2` 个州再演讲 `1` 个小时，获得一个协作者。
3. 在第 `3` 个州与协作者一起演讲 `2` 个小时，赢得一张选票。
4. 在第 `1` 个州与协作者一起演讲 `0.5` 个小时，赢得一张选票。

这个样例满足子任务 `3,4,5,6,7` 的性质。

### 样例解释 #2

按照如下方案进行演讲，理惠将在 `32` 小时内赢得 `4` 张选票。

1. 在第 `1` 个州演讲 `4` 个小时，赢得一张选票。
2. 在第 `2` 个州演讲 `11` 个小时，赢得一张选票。
3. 在第 `3` 个州演讲 `6` 个小时，赢得一张选票。
4. 在第 `6` 个州演讲 `11` 个小时，赢得一张选票。

这个样例满足子任务 `1,2,3,4,5,7` 的限制。

### 样例解释 #3

按照如下方案进行演讲，理惠将在 `11.5` 小时内赢得 `3` 张选票。

1. 在第 `4` 个州演讲 `7` 个小时，赢得一张选票，并获得一个协作者。
2. 在第 `1` 个州与协作者一起演讲 `4` 个小时，赢得一张选票。与此同时，协作者在第 `2` 个州演讲 `4` 个小时。
3. 在第 `2` 个州与协作者一起演讲 `0.5` 个小时，赢得一张选票。

这个样例满足子任务 `2,3,4,5,7` 的限制。

## 数据范围

对于全部数据：

`1≤N≤500`

`1≤K≤N`

`1≤Ai≤10^3`

`Ai≤Bi≤10^3` 或者 `Bi=-1`

## 子任务

1. `1∼2`：`4` 分，`Bi=-1`
2. `3∼4`：`4` 分，`Bi=-1` 或者 `Bi=Ai`
3. `5∼10`：`12` 分，`N≤7`
4. `11∼16`：`12` 分，`N≤20`
5. `17∼33`：`34` 分，`N≤100`
6. `34∼38`：`10` 分，`K=N`
7. `39∼50`：`24` 分，无特殊限制
