# 括号序列计数

## 题目大意

给定一个只包含 `(`、`)` 和 `?` 的字符串。每个 `?` 可以替换成 `(` 或 `)`。

要求统计有多少种替换方式，可以使最终字符串成为合法括号序列，答案对 $998244353$ 取模。

## 思路分析

一个括号序列合法，当且仅当满足两个条件：

- 从左到右扫描时，任意前缀中左括号数量都不少于右括号数量；
- 扫描完整个字符串后，左括号数量等于右括号数量。

也就是说，我们只需要维护当前的“括号余额”：

$$
balance=\text{左括号数量}-\text{右括号数量}
$$

合法过程中，$balance$ 不能变成负数，最后必须回到 $0$。

设：

$$
dp[i][j]
$$

表示处理完前 $i$ 个字符后，当前括号余额为 $j$ 的方案数。

初始状态为：

$$
dp[0][0]=1
$$

对于第 $i$ 个字符：

- 如果它可以作为 `(`，那么余额可以从 $j$ 转移到 $j+1$；
- 如果它可以作为 `)`，那么要求当前 $j>0$，余额可以从 $j$ 转移到 $j-1$。

也就是说：

```c++
if (s[i] == '(' || s[i] == '?') dp[i + 1][j + 1] += dp[i][j];
if (s[i] == ')' || s[i] == '?') dp[i + 1][j - 1] += dp[i][j];
```

当然第二个转移必须保证 $j>0$。

最后答案就是：

$$
dp[n][0]
$$

如果字符串长度是奇数，也可以直接输出 $0$，因为合法括号序列长度一定是偶数。

下面是正解：

```c++
#include <bits/stdc++.h>
using namespace std;

const int MOD = 998244353;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string s;
    cin >> s;

    int n = (int)s.size();

    if (n % 2 == 1) {
        cout << 0 << '\n';
        return 0;
    }

    vector<vector<int>> dp(n + 1, vector<int>(n + 1, 0));
    dp[0][0] = 1;

    for (int i = 0; i < n; i++) {
        for (int bal = 0; bal <= n; bal++) {
            if (dp[i][bal] == 0) continue;

            if (s[i] == '(' || s[i] == '?') {
                if (bal + 1 <= n) {
                    dp[i + 1][bal + 1] += dp[i][bal];
                    if (dp[i + 1][bal + 1] >= MOD) {
                        dp[i + 1][bal + 1] -= MOD;
                    }
                }
            }

            if (s[i] == ')' || s[i] == '?') {
                if (bal > 0) {
                    dp[i + 1][bal - 1] += dp[i][bal];
                    if (dp[i + 1][bal - 1] >= MOD) {
                        dp[i + 1][bal - 1] -= MOD;
                    }
                }
            }
        }
    }

    cout << dp[n][0] << '\n';

    return 0;
}
```

状态数为 $O(n^2)$，每个状态转移常数次，所以时间复杂度为 $O(n^2)$。空间复杂度为 $O(n^2)$，在 $n\le 3000$ 的限制下可以通过。
