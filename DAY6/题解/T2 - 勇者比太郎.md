# 勇者比太郎 2

## 题目大意

有 $N$ 只怪物，第 $i$ 只怪物需要力量至少为 $A_i$ 才能击败，击败之后力量会增加 $B_i$。

比太郎可以选择一个起点 $j$，然后按照：

$$
j,j+1,\dots,N,1,2,\dots,j-1
$$

的顺序打完所有怪物。要求选择最优起点，使得初始力量 $x$ 尽可能小，输出这个最小值。

## 思路分析

首先考虑比较直接的做法。我们可以枚举起点 $j$，然后按照这个顺序模拟打怪，计算这个起点需要的最小初始力量。一次模拟需要 $O(N)$，枚举所有起点就是 $O(N^2)$。

这个做法可以通过 $N\le 2000$ 的子任务 $2$，拿到前两个子任务一共 $10+21=31$ 分。如果答案不超过 $10$，也可以二分或者枚举答案再检查，可以拿到对应的小答案子任务。

下面考虑正解。

环形顺序不太方便处理，我们先把数组复制一遍。这样从起点 $j$ 出发打完一圈，就等价于在线性数组中依次打：

$$
j,j+1,\dots,j+N-1
$$

设 $sum_i$ 表示复制后数组中 $B$ 的前缀和：

$$
sum_i=B_1+B_2+\cdots+B_i
$$

如果从起点 $j$ 开始，那么打到第 $i$ 只怪物时，前面已经获得的力量增加量是：

$$
sum_{i-1}-sum_{j-1}
$$

所以需要满足：

$$
x+sum_{i-1}-sum_{j-1}\ge A_i
$$

移项得到：

$$
x\ge A_i-sum_{i-1}+sum_{j-1}
$$

对于固定起点 $j$，要让整个区间 $[j,j+N-1]$ 内所有怪物都能被击败，就需要：

$$
x\ge sum_{j-1}+\max_{i=j}^{j+N-1}(A_i-sum_{i-1})
$$

因此我们只需要预处理：

$$
val_i=A_i-sum_{i-1}
$$

然后对于每个起点 $j$，查询区间 $[j,j+N-1]$ 中 $val_i$ 的最大值即可。

这就变成了一个静态区间最大值问题，可以用 ST 表处理。复制数组长度为 $2N$，建表复杂度 $O(N\log N)$，每个起点查询 $O(1)$。

下面是正解：

```c++
#include <bits/stdc++.h>
using namespace std;

using ll = long long;

struct SparseTable {
    int n, lg;
    vector<vector<ll>> st;

    void init(const vector<ll> &a) {
        n = (int)a.size() - 1;
        lg = __lg(n);
        st.assign(lg + 1, vector<ll>(n + 1));

        for (int i = 1; i <= n; i++) {
            st[0][i] = a[i];
        }

        for (int j = 1; j <= lg; j++) {
            for (int i = 1; i + (1 << j) - 1 <= n; i++) {
                st[j][i] = max(st[j - 1][i], st[j - 1][i + (1 << (j - 1))]);
            }
        }
    }

    ll query(int l, int r) {
        int k = __lg(r - l + 1);
        return max(st[k][l], st[k][r - (1 << k) + 1]);
    }
};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<ll> a(2 * n + 1), b(2 * n + 1);
    for (int i = 1; i <= n; i++) cin >> a[i];
    for (int i = 1; i <= n; i++) cin >> b[i];

    for (int i = n + 1; i <= 2 * n; i++) {
        a[i] = a[i - n];
        b[i] = b[i - n];
    }

    vector<ll> sum(2 * n + 1), val(2 * n + 1);
    for (int i = 1; i <= 2 * n; i++) {
        sum[i] = sum[i - 1] + b[i];
    }

    for (int i = 1; i <= 2 * n; i++) {
        val[i] = a[i] - sum[i - 1];
    }

    SparseTable st;
    st.init(val);

    ll ans = LLONG_MAX;
    for (int j = 1; j <= n; j++) {
        ll need = sum[j - 1] + st.query(j, j + n - 1);
        ans = min(ans, need);
    }

    cout << ans << '\n';

    return 0;
}
```

时间复杂度为 $O(N\log N)$，空间复杂度为 $O(N\log N)$。

其实这里的区间长度都是固定的 $N$，也可以用单调队列维护滑动窗口最大值，把复杂度优化到 $O(N)$。不过 ST 表写起来比较直接，也足以通过本题数据范围。
