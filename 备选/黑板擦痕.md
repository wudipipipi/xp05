# 黑板擦痕

## 题目描述

黑板上写着一个只含加号的算式：

$$
a_1+a_2+\cdots+a_n.
$$

现在可以选择若干个加号进行“半擦除”。每个被半擦除的加号会从 `+` 变成 `|`，其中 `|` 表示按位或运算。

例如，原式

$$
a_1+a_2+a_3+a_4
$$

可能变成

$$
a_1\,|\,a_2+a_3\,|\,a_4.
$$

在本题中，按位或运算 `|` 的优先级高于加法 `+`。因此上式的计算结果为

$$
(a_1\,|\,a_2)+(a_3\,|\,a_4).
$$

请你计算，有多少种不同的半擦除方案，使得新算式的计算结果与原算式

$$
a_1+a_2+\cdots+a_n
$$

的计算结果相同。

不擦除任何加号也算一种方案。

答案对 $998244353$ 取模。

## 输入格式

本题有多组测试数据。

第一行包含一个整数 $T$，表示测试数据组数。

对于每组测试数据：

第一行包含一个整数 $n$，表示数字个数。

第二行包含 $n$ 个整数 $a_1,a_2,\ldots,a_n$。

## 输出格式

对于每组测试数据，输出一行一个整数，表示合法半擦除方案数对 $998244353$ 取模后的结果。

## 样例

### 样例 1 输入

```text
4
3
1 2 3
3
0 0 5
3
1 1 0
4
1 2 4 8
```

### 样例 1 输出

```text
2
4
2
8
```

## 数据范围与约定

对于所有测试数据，保证：

- $1 \le T \le 100$；
- $1 \le n \le 2\times 10^5$；
- $0 \le a_i < 2^{31}$；
- 单个测试文件内所有测试数据的 $n$ 之和不超过 $2\times 10^5$。

本题采用测试点捆绑计分，同一子任务内的所有测试点均通过才能获得该子任务分数。

| 子任务 | 测试点编号 | 分值 | $n$ | $a_i$ | 特殊性质 |
| :--: | :--: | :--: | :--: | :--: | :--: |
| $1$ | $1 \sim 2$ | $8$ | $\le 20$ | $<2^{31}$ | 无 |
| $2$ | $3 \sim 4$ | $8$ | $\le 2\times 10^5$ | $=0$ | 无 |
| $3$ | $5 \sim 8$ | $14$ | $\le 2\times 10^5$ | $<2^{31}$ | A |
| $4$ | $9 \sim 12$ | $14$ | $\le 5000$ | $<2^{31}$ | 无 |
| $5$ | $13 \sim 18$ | $20$ | $\le 2\times 10^5$ | $<2^{12}$ | 无 |
| $6$ | $19 \sim 30$ | $36$ | $\le 2\times 10^5$ | $<2^{31}$ | 无 |

特殊性质 A：对于每个 $a_i$，要么 $a_i=0$，要么 $a_i$ 的二进制表示中恰好只有一个 $1$。
