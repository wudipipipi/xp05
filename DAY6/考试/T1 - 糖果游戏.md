# T118341.糖果游戏

## 题目描述

输入文件：`candy.in`  
输出文件：`candy.out`

在创世纪时期的古堡集市中，fool 经营着一家售卖魔法糖果的店铺。这里有 `N` 种不同的糖果，第 `i` 种糖果的售价为 `Ai` 枚魔法金币，同时每当购买第 `i` 种糖果后，会立即获得 `Bi` 枚金币的回馈。

Knight 怀揣着 `R` 枚金币来到集市，他渴望收集尽可能多的糖果。由于魔法糖果可以重复购买（只要金币足够），你需要帮助他计算最多能获得多少颗糖果。

## 输入格式

第一行输入一个整数 `T`，表示测试用例的数量（`1≤T≤10`）。

每个测试用例包含三行：

第一行两个整数 `N` 和 `R`，分别表示糖果种类数和 Knight 持有的金币数（`1≤N≤2×10^5`，`1≤R≤10^9`）。

第二行 `N` 个整数 `A1` 到 `AN`，表示每种糖果的售价（`1≤Bi<Ai≤10^9`）。

第三行 `N` 个整数 `B1` 到 `BN`，表示购买对应糖果后的回馈金币数。

## 输出格式

对于每个测试用例，输出一行一个整数，表示 Knight 能购买的糖果最大数量。

## 说明 / 提示

`1≤N≤2×10^5`

`1≤Bi < Ai ≤ 10^9`
