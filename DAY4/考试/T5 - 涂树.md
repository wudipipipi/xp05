# T118330.涂树

## 题目描述

Welcome24ever 有一棵 `n` 个结点的树，最初所有结点都是白色的。

现在在树上有两个棋子，分别叫做 `PA` 和 `PB`，一开始分别位于点 `a` 和点 `b` 上。Welcome24ever 需要按照顺序执行下列操作：

1. 将 `PA` 移动到相邻一个点，如果移动后的点是白色，那么就将其涂成红色。
2. 将 `PB` 移动到相邻一个点，如果移动后的点是红色，那么就将其涂成蓝色。

最开始，棋子 `Pa` 所在的位置会变成红色，如果 `Pb` 和 `Pa` 初始位于同一点，那么这个初始点就会变成蓝色。

注意，每一轮都需要先移动 `PA` 再移动 `PB`，两个棋子可以位于同一个顶点。

Welcome24ever 很想知道把所有顶点涂成蓝色所需要的操作轮数（`PA` 和 `PB` 都移动一次算一轮操作），请你帮帮他！

## 输入格式

第一行包含一个整数 `t`（`1≤t≤10^4`），表示测试用例的数量。接下来是 `t` 个测试用例。

每个测试用例的第一行包含一个整数 `n`（`1≤n≤2⋅10^5`）。

每个测试用例的第二行包含两个整数 `a` 和 `b`（`1≤a,b≤n`）。

接下来的 `n−1` 行，每一行包含两个整数 `xi, yi`（`1≤xi, yi≤n`），表示点 `xi` 和 `yi` 之间存在一条边，保证数据中这些边构成一棵树。

保证所有测试用例的 `n` 之和不超过 `2⋅10^5`。

## 输出格式

对于每个测试用例，输出将所有顶点涂成蓝色所需的最少操作轮数。

## 说明 / 提示

### 样例说明

在第一个测试用例中，Welcome24ever 能够把按照下列操作顺序把所有点都涂成蓝色：

最初，`PA` 位于点 `1`，并且 `PB` 位于点 `2`，点 `1` 被涂成红色，点 `2` 是白色。  
Welcome24ever 把 `PA` 移动到点 `2` 并且将其涂为红色。之后 Welcome24ever 把 `PB` 移动到点 `1` 并将其涂成蓝色。  
Welcome24ever 把 `PA` 移动到点 `1`。之后 Welcome24ever 把 `PB` 移动到点 `2` 并将其涂成蓝色。

## 测试点

1. `1∼1`：`n≤100`，对于所有边 `(ui, vi)`，都满足 `ui=i, yi=i+1`
2. `2∼3`：`n≤1000`，`a=b`
3. `4∼10`：`n≤2⋅10^5`，无特殊性质
