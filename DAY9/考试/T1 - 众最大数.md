# T112473.众最大数

## 题目描述

输入文件：`max.in`  
输出文件：`max.out`

小 L 有一个大小为 `n` 的可重集合 `S`，将其中元素从小到大排序后第 `i` 个元素为 `si`。

对于这个大小为 `n` 的集合存在 `2^n−1` 个不同非空子集 `T⊂S`（对于相同值的元素 `i≠j, si=sj` 在此处认为为不同的元素）。

现在将所有这些子集的最大值放入一个新的可重集合 `S'` 中，即

`S' = {max{T} | T⊂S}`

现在小 L 想知道 `S'` 这个集合的众数是多少，如果存在多个数出现次数都是最多那么输出其中最大的那个。

## 输入格式

第一行一个整数 `n`，表示可重集合 `S` 的大小。

接下来一行包含 `n` 个正整数，表示 `si`，保证对于 `i∈(1,n]`，`s(i−1)≤si`。

## 输出格式

共一行，包含一个整数，表示集合 `S'` 的最大众数。

## 输入输出样例

### 输入 #1

```text
4
1 2 2 3
```

### 输出 #1

```text
3
```

## 说明 / 提示

S={1,2,2,3}，可以得到 `S'={1,2,2,3,2,2,3,2,3,3,2,3,3,3,3}`，在一共 `15` 个数中 `1` 出现了 `1` 次，`2` 出现了 `6` 次，`3` 出现了 `8` 次，所以众数为 `3`。

## 数据范围

对于 `30%` 的数据保证 `1≤n≤10`；

对于 `60%` 的数据保证 `1≤n≤1000`；

对于 `100%` 的数据保证 `1≤n≤10^5, 1≤si≤10^9`。
