# 聚餐 1

## 题目大意

给定一个 $n\times m$ 的 $01$ 网格。需要统计有多少个矩形满足：

1. 矩形内任意两个相邻格子不能同时为 $1$；
2. 矩形内 $1$ 的数量大于矩形面积的一半。

## 思路分析

首先考虑暴力做法。枚举矩形的上下左右边界，然后检查这个矩形是否满足两个条件。这样复杂度至少是 $O(n^2m^2)$，如果再暴力检查矩形内部会更高，只能处理很小的数据。

下面考虑本题最关键的性质。

如果一个矩形中没有相邻的两个 $1$，那么这些 $1$ 在网格中相当于选了一个独立集。对于一个 $a\times b$ 的矩形，最多只能选：

$$
\left\lceil \frac{ab}{2}\right\rceil
$$

个互不相邻的格子。

题目要求 $1$ 的数量严格大于 $\frac{ab}{2}$，所以必须刚好达到：

$$
\left\lceil \frac{ab}{2}\right\rceil
$$

这说明矩形面积必须是奇数，也就是矩形的高和宽都必须是奇数。

并且这时 $1$ 必须全部落在棋盘染色中数量较多的那一类格子上。对于奇数高、奇数宽的矩形，数量较多的颜色就是左上角所在的颜色。所以合法矩形必须满足：

- 高度是奇数；
- 宽度是奇数；
- 左上角是 $1$；
- 整个矩形呈现完美棋盘格。

于是问题变成统计满足上述条件的棋盘格矩形。

为了判断棋盘格是否一致，我们定义每个格子的类型：

$$
type_{i,j}=a_{i,j}\oplus ((i+j)\bmod 2)
$$

如果一个区域是完美棋盘格，那么这个区域内所有格子的 $type$ 都相同。

接下来从上到下枚举矩形的上边界不太方便，代码中是从下往上扫描每一行，设当前行为矩形的上边界。对于每一列 $j$，维护：

$$
D_j
$$

表示从当前行开始向下，连续多少个格子的 $type$ 与当前格子相同。这样如果某段列的 $type$ 都相同，那么这段列内能向下延伸的最大高度就是这些 $D_j$ 的最小值。

对于当前行，我们先把它按 $type$ 相同切成若干连续段。只有在同一段里选择左右边界，才可能组成棋盘格矩形。

在某个连续段内，问题就变成：

> 对每个子区间 $[l,r]$，设 $mn=\min(D_l,D_{l+1},\dots,D_r)$，它能贡献多少个合法矩形？

因为高度必须是奇数，所以可选高度数量是：

$$
\left\lceil \frac{mn}{2}\right\rceil=\frac{mn+1}{2}
$$

宽度也必须是奇数，所以 $l,r$ 必须同奇偶。并且左上角必须是 $1$，所以还要求当前行第 $l$ 列为 $1$。

接下来用单调栈统计每个位置作为区间最小值的位置。

对于段内每个位置 $j$，用单调栈求出：

- $L_j$：左边第一个 $D$ 严格小于 $D_j$ 的位置；
- $R_j$：右边第一个 $D$ 小于等于 $D_j$ 的位置。

这样所有满足：

$$
L_j<l\le j\le r<R_j
$$

的区间，都会唯一地归到位置 $j$ 作为最小值位置，避免重复计算。

对于这些区间，我们只需要统计：

- 左端点 $l$ 在 $[L_j+1,j]$ 中，且当前行 $a_{i,l}=1$；
- 右端点 $r$ 在 $[j,R_j-1]$ 中；
- $l,r$ 同奇偶。

这可以用当前行的奇偶前缀和 $O(1)$ 计算。

下面是正解：

```c++
#include <bits/stdc++.h>
using namespace std;

void solve() {
    int n, m;
    cin >> n >> m;

    vector<string> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    long long ans = 0;

    vector<int> D(m + 1, 0);
    vector<int> cur_type(m + 1, 0), pre_type(m + 1, -1);
    vector<int> P_even_1(m + 1), P_odd_1(m + 1);
    vector<int> P_even_all(m + 1), P_odd_all(m + 1);
    vector<int> L(m + 1), R(m + 1);
    vector<int> st;

    for (int i = n - 1; i >= 0; i--) {
        for (int j = 1; j <= m; j++) {
            int type = (a[i][j - 1] - '0') ^ ((i + j) & 1);

            if (i == n - 1 || type != pre_type[j]) {
                D[j] = 1;
            } else {
                D[j]++;
            }

            cur_type[j] = type;
        }
        pre_type = cur_type;

        for (int j = 1; j <= m; j++) {
            P_even_1[j] = P_even_1[j - 1] + (j % 2 == 0 && a[i][j - 1] == '1');
            P_odd_1[j] = P_odd_1[j - 1] + (j % 2 == 1 && a[i][j - 1] == '1');
            P_even_all[j] = P_even_all[j - 1] + (j % 2 == 0);
            P_odd_all[j] = P_odd_all[j - 1] + (j % 2 == 1);
        }

        int start = 1;
        while (start <= m) {
            int end = start;
            while (end + 1 <= m && cur_type[end + 1] == cur_type[start]) {
                end++;
            }

            st.clear();
            for (int j = start; j <= end; j++) {
                while (!st.empty() && D[st.back()] >= D[j]) {
                    st.pop_back();
                }
                L[j] = st.empty() ? start - 1 : st.back();
                st.push_back(j);
            }

            st.clear();
            for (int j = end; j >= start; j--) {
                while (!st.empty() && D[st.back()] > D[j]) {
                    st.pop_back();
                }
                R[j] = st.empty() ? end + 1 : st.back();
                st.push_back(j);
            }

            for (int j = start; j <= end; j++) {
                int l1 = L[j] + 1, l2 = j;
                int r1 = j, r2 = R[j] - 1;

                long long left_even_1 = P_even_1[l2] - P_even_1[l1 - 1];
                long long right_even = P_even_all[r2] - P_even_all[r1 - 1];

                long long left_odd_1 = P_odd_1[l2] - P_odd_1[l1 - 1];
                long long right_odd = P_odd_all[r2] - P_odd_all[r1 - 1];

                long long width_choices = left_even_1 * right_even + left_odd_1 * right_odd;
                long long height_choices = (D[j] + 1) / 2;

                ans += width_choices * height_choices;
            }

            start = end + 1;
        }
    }

    cout << ans << '\n';
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int T;
    cin >> T;
    while (T--) {
        solve();
    }

    return 0;
}
```

每一行中，每个位置只会进出单调栈常数次，所以单组数据时间复杂度为 $O(nm)$。空间复杂度为 $O(m)$。
