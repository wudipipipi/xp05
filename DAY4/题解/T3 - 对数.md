# 对数

## 题目描述

给定一个包含 $n$ 个整数的数组 $a$。请找出所有满足以下条件的数对 $(i, j)$（$1 \le i < j \le n$）：  
数对的和 $a_i + a_j$ 大于等于 $l$ 且小于等于 $r$（即 $l \le a_i + a_j \le r$）。

例如，当 $n = 3$，$a = [5, 1, 2]$，$l = 4$，$r = 7$ 时，有以下两个符合条件的数对：
- $i=1$ 和 $j=2$（$4 \le 5 + 1 \le 7$）；
- $i=1$ 和 $j=3$（$4 \le 5 + 2 \le 7$）。

## 输入格式

第一行包含一个整数 $t$（$1 \le t \le 10^4$），表示测试用例的数量。接下来是 $t$ 个测试用例。

每个测试用例的第一行包含三个整数 $n$, $l$, $r$（$1 \le n \le 2 \cdot 10^5$, $1 \le l \le r \le 10^9$）—— 分别表示数组的长度以及数对和的上下限。

每个测试用例的第二行包含 $n$ 个整数 $a_1, a_2, \ldots, a_n$（$1 \le a_i \le 10^9$）。

保证所有测试用例的 $n$ 之和不超过 $2 \cdot 10^5$。

## 输出格式

对于每个测试用例，输出一个整数 —— 满足条件 $l \le a_i + a_j \le r$ 且 $i < j$ 的索引对 $(i, j)$ 的数量。


### 输入样例1

```
4
3 4 7
5 1 2
5 5 8
5 1 2 4 3
4 100 1000
1 1 1 1
5 9 13
2 5 5 1 1
```

### 输出样例1

```
2
7
0
1
```

## 说明/提示

#### 数据范围

对于所有测试数据，都满足 $1 \le t \le 10^4$，$1 \le n \le 2 \cdot 10^5$， $1 \le l \le r \le 10^9$，$1 \le a_i \le 10^9$，$\displaystyle\sum n \leq 2 \cdot 10^5$

|   测试点   |          $n$          |
| :--------: | :-------------------: |
| $1\sim 6$  |     $n \leq 1000$     |
| $7\sim 20$ | $n \leq 2 \cdot 10^5$ |

## 题目大意

给定一个包含 $n$ 个整数的数组 $a$。请找出所有满足以下条件的数对 $(i, j)$（$1 \le i < j \le n$）：  
数对的和 $a_i + a_j$ 大于等于 $l$ 且小于等于 $r$（即 $l \le a_i + a_j \le r$）。

## 思路分析

最暴力的解法显然是枚举所有数对 $(x,y)$，判断是否在区间内就可以了，这样的做法是 $O(n^2)$ 的。

```c++
void solve() {
    int n,l,r;
    cin >> n >> l >> r;
    vector<int> vec(n);
    for(int i =0;i<n;i++){
        cin >> vec[i];
    }
    int ans = 0;
    for (int i = 0; i < n; ++i) {
        for (int j = i + 1 ; j < n ; ++j) {
            if(l <= vec[i] + vec[j] && vec[i] + vec[j] <= r){
                ans++;
            }
        }
    }
    cout << ans << endl;
}
```



我们把 $l \le a_i + a_j \le r$ 变形成 $l - a_j \le a_i \le r - a_j$ 这样可以发现对于一个 $a_i$ 来说，其满足条件的数字值域上是连续的。而且这个题我们只关心数值不关心位置，所以可以直接把数组进行排序，之后遍历每一个 $a_i$，之后我们可以每次都用二分来找到前 $i$ 个数字里符合条件的区间范围，之后把区间长度累计到答案上 。对二分的范围做限制是为了防止重复计数。

```c++
void solve() {
    int n, l, r;
    cin >> n >> l >> r;
    vector<int> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    sort(a.begin(), a.end());
    long long ans = 0;
    for (int i = 0; i < n; i++) {
        int L = lower_bound(a.begin(), a.begin() + i, l - a[i]) - a.begin();
        int R = upper_bound(a.begin(), a.begin() + i, r - a[i]) - a.begin();
        ans += R - L;
    }
    cout << ans << endl;
}
```

之后再给出一种双指针的写法，我们考虑对于 $a_i$ 和 $a_{i+1}$ 其答案范围的变化，对于 $[L_i,R_i]$ 和 $[L_{i+1},R_{i+1}]$ 这两个区间来说，存在 $L_i \leq L_{i+1}$ 和 $R_i \leq R_{i+1}$ 这两个偏序关系，所以我们只需要排序之后维护一个总共的双指针，每次尝试向左移动两个指针就可以了。

```c++
void solve() {
   int n,l,r;
   cin >> n >> l >> r;
   vector<int> vec(n);
   for(auto &x : vec) cin >> x;
   std::sort(vec.begin(), vec.end());
   long long ans = 0;
   int L = n,R = n;
   for (int i = 0; i < n ; ++i) {
       while(L && vec[L-1] + vec[i] >= l) L--;
       while(R && vec[R-1] + vec[i] > r) R--;
       ans += min(i,R) - min(i,L);
   }
   cout << ans << endl;
}
```
