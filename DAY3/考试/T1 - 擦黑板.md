# T118319.擦黑板

## 题目描述

现在在黑板上有若干个数字 `1、2、3`。在一次运算中，可以进行如下操作：

选择两个不同的数字，将其从黑板上擦除，并且写上一个与擦掉的两个数字不同的那个数字。例如一开始的数字有 `1、1、1、2、2、3`，选择数字 `1` 和 `3`，并且擦掉，再写上一个 `2`，黑板上的数字就会变成 `1、1、2、2、2`。

经过任意次操作之后，黑板上能否只剩下一种数字，如果可以的话，可以是哪种数字？

## 输入格式

每个测试包含多个测试用例。第一行包含测试用例的数量 `t` (`1≤t≤10^5`)。测试用例说明如下：

每个测试用例一行，包含三个整数 `a, b, c` (`1≤a,b,c≤100`) —— 分别表示数字 `1` 的个数、数字 `2` 的个数和数字 `3` 的个数。

## 输出格式

针对每个测试用例，输出一行，需要包含三个数字。

如果经过某些操作后，黑板上可能只剩下数字 `1`，则第一个数应为 `1`，否则为 `0`。

同样，如果经过某些运算后，黑板上可能只剩下数字 `2`，则第二个数字应为 `1`，否则应为 `0`。

同样，如果经过某些运算后，黑板上可能只剩下数字 `3`，则第三个数字应为 `1`，否则应为 `0`。

## 输入输出样例

### 输入 #1

```text
3
1 1 1
2 3 2
82 47 59
```

### 输出 #1

```text
1 1 1
0 1 0
1 0 0
```

## 说明 / 提示

在第一个测试案例中，可以删除数字 `2` 和 `3` 并写入数字 `1`。之后，黑板上将有 `2` 个数字 `1`。她可以通过类似的操作使棋盘只剩下数字 `2` 或 `3`。

在第二个测试案例中，可以删除数字 `1` 和 `3` 并写入数字 `2`。在进行了 `2` 次操作后，黑板上将只剩下数字 `2`。可以证明，不可能只剩下数字 `1` 或只剩下数字 `3`。

在第三个测试案例中，有一个运算序列使得黑板上只剩下数字 `1`。我们可以通过某种方式证明不可能只剩下数字 `2` 或数字 `3`。

## 数据范围

对于所有测试数据，都满足 `1≤t≤10^5`，`0≤a,b,c≤100`。

测试点：

1. `1∼4`：`1≤t≤10^5`，`1≤a,b,c≤100`，`ai` 一定全是奇数
2. `5∼8`：`1≤t≤10^5`，`1≤a,b,c≤100`，`ai` 一定全是偶数
3. `9∼12`：`1≤t≤10^5`，`0≤a,b,c≤100`，`ai` 一定至少有一个为 `0`
4. `13∼20`：`1≤t≤10^5`，`1≤a,b,c≤100`
