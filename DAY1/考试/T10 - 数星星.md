# T3- 数星星

## 题目描述

夜空中有 $n$ 颗星星，每颗星星都可以看作平面上的一个点，第 $i$ 颗星星的坐标为 $(x_i, y_i)$，且任意两颗星星的位置不同。

现在需要选出两颗不同的星星，其中一颗放上指南针，另一颗作为被观察的星星。若被观察的星星位于指南针的正北、正南、正东、正西、东北、东南、西北或西南方向之一，则称这种选择是合法的。

请你求出一共有多少种合法选择。

换句话说，请求出有多少个有序数对 $(i, j)$，满足 $1 \leq i, j \leq n$ 且 $i \neq j$，并且点 $(x_i, y_i)$ 位于点 $(x_j, y_j)$ 的八个方向之一。

一个点 $(x_i, y_i)$ 位于点 $(x_j, y_j)$ 的八个方向之一，当且仅当满足以下条件之一：

- $x_i = x_j$；
- $y_i = y_j$；
- $x_i - y_i = x_j - y_j$；
- $x_i + y_i = x_j + y_j$。

## 输入格式

每个测试文件包含多组测试数据。

第一行包含一个整数 $t$，表示测试数据组数。

对于每组测试数据：

第一行包含一个整数 $n$，表示点的数量。

接下来 $n$ 行，第 $i$ 行包含两个整数 $x_i, y_i$，表示第 $i$ 个点的坐标。

保证任意两个点的坐标不同。

## 输出格式

对于每组测试数据，输出一行一个整数，表示合法有序数对 $(i, j)$ 的数量。

## 提示

### 【样例 1 解释】

对于第一组测试数据，合法放法如下：

- 指南针在 $(0, 0)$，星星在 $(-1, -1)$，星星在指南针的西南方向；
- 指南针在 $(0, 0)$，星星在 $(1, 1)$，星星在指南针的东北方向；
- 指南针在 $(-1, -1)$，星星在 $(0, 0)$，星星在指南针的东北方向；
- 指南针在 $(-1, -1)$，星星在 $(1, 1)$，星星在指南针的东北方向；
- 指南针在 $(1, 1)$，星星在 $(0, 0)$，星星在指南针的西南方向；
- 指南针在 $(1, 1)$，星星在 $(-1, -1)$，星星在指南针的西南方向。

因此答案为 $6$。

对于第二组测试数据，合法放法如下：

- 指南针在 $(6, 9)$，星星在 $(10, 13)$，星星在指南针的东北方向；
- 指南针在 $(10, 13)$，星星在 $(6, 9)$，星星在指南针的西南方向。

因此答案为 $2$。

### 【数据范围】

对于所有测试数据，保证：

- $1 \leq t$；
- $2 \leq n \leq 10^5$；
- $-10^9 \leq x_i, y_i \leq 10^9$；
- 任意两个点的坐标不同；
- 所有测试数据中 $n$ 的总和不超过 $2 \times 10^5$。
