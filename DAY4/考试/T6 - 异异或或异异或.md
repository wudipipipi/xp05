# 异异或或异异或

## 题目描述

给定一个数组 $a_1,a_2,\dots,a_n$。请计算满足以下条件的三元组 $(x,y,z)$ 的数量：

$1 \leq x \leq y \leq z \leq n$，

$f(x,y)\oplus f(y,z) > f(x,z)$。

其中，$f(l,r)=a_l\oplus a_{l+1}\oplus\dots\oplus a_r$，$\oplus$ 表示按位异或运算。

## 输入格式

每个测试包含多个测试用例。

输入的第一行包含一个整数 $t$，表示测试用例的数量。

对于每个测试用例，输入的第一行包含一个整数 $n$，表示数组的长度。

输入的第二行包含 $n$ 个整数 $a_1,a_2,\dots,a_n$，表示数组 $a$。

## 输出格式

对于每个测试用例，输出一行一个整数，表示符合题目要求的三元组数量。

## 样例

### 样例 1 输入

```text
3
3
6 2 4
1
3
5
7 3 7 2 1
```

### 样例 1 输出

```text
4
0
16
```

## 提示

### 【样例 1 解释】

在第一个测试用例中，数组为 $[6,2,4]$，共有 $4$ 个满足条件的三元组：

- $(1,2,2)$；
- $(1,1,3)$；
- $(1,2,3)$；
- $(1,3,3)$。

在第二个测试用例中，没有满足条件的三元组。

在第三个测试用例中，满足条件的三元组共有 $16$ 个。

### 【数据范围】

对于所有测试数据，保证：

- $1 \leq t \leq 10^4$；
- $1 \leq n \leq 10^5$；
- $1 \leq a_i \leq 10^9$；
- 所有测试用例的 $n$ 之和不超过 $10^5$。

| 测试点编号 | $n \leq$ |
| :--: | :--: |
| $1 \sim 4$ | $50$ |
| $5 \sim 10$ | $300$ |
| $11 \sim 20$ | $10^5$ |
